Floorcovering System Reports Overview

Several reports exist to help you manage your Floorcovering System. Below is a list of reports and an accompanying description:

  • TAG REPORTS
    • The Tag Assigned Report will display the tags assigned to a specific job by sale or customer order.
    • The Perpetual Inventory Report will allow you to view tags of all rolls currently in inventory. This report shows all the same information found on the Stock Tag of a product.
    • The Tag History Report provides you information on a parent roll and its children at a high level. By pressing CANCEL [F5], you can access more detailed information the Roll History Report for the parent.
    • The Movement Report shows a timeline of all transactions which occurred for a single Roll Tag (ie. sale, return, chargeback, ibar, po, inventory adjustment ...). The user will be able to browse through the dates on the screen or print a report. The user can also access this report from within Tag History by pressing Enter on a Tag Id and selecting View Tag Movement.
  • UNPAID WORK ORDER REPORT
    • Unpaid Work Order Report will show all the completed work orders for a given date range for which installers have not yet been paid. 
  • INSTALLER PERFORMANCE REPORT
    • Installer Performance Report will help you analyze the performance of your installers. This report lists all installers and the number of installs and re-works they have completed over the past year. This report can also be limited to display the performance of a single installer. 
  • OPEN WORK ORDERS REPORT
    • Open Work Orders Report will show all uninstalled work orders for a given date range by installer. 
  • JOB COST REPORT
    • The Job Cost Report will show a breakdown of price and cost per order, including expense items. It is designed for identifying weighted average cost issues.
