Floorcovering Scorecard Overview

The FC Scorecard summarizes store usage of key floorcovering application features focusing on process flow and transparency. Scorecard lines allow stores to quickly review incomplete order processes. Lines with high scores reveal a breach in procedural flow and the opportunity to improve facility process:

  • System Cleanup
  • Complete Open Flows
  • Streamline Process
  • Improve Process Transparency

The Scorecard will display the number of currently open work orders and the number of order lines open on these work orders along with detail on the status of these lines. When reviewing these numbers, the following points should be considered:

  • Work order lines all require reconciliation through Work Order Reconciliation.
  • Reconciled lines must be both processed (sale or expense) or disposed and paid to the installer to close and be removed from the Scorecard.
  • Lines do not clear when the order and fulfillment process is not correctly followed.
  • If a work order is stuck in the system, contact the Stores Helpdesk at 216-566-2740.
