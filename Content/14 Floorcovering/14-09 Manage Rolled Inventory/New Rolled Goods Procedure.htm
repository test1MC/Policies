New Rolled Goods Procedure

Step

Action

Receive Rolled Goods

1

From POS desktop, select Store -> Select Merchandise Management.

2

Enter your Employee Number and Password.

3

Select Receive/Verify from the Merchandise Management Menu.

4

Select a Receiving Option to search and select the Purchase Order.

5

Press the UP ARROW to move to the RCV-Q Field. 

6

To receive a roll into inventory, begin entering the RCV-Q Quantity to display the Tag ID Options Menu.

Note: For every line item, you will create a new Tag ID for each roll being received into inventory.

7

Enter the Roll#/Serial# assigned by the vendor in the Registration# Field.

Note: A Tag ID will be assigned to the roll. This unique ID will track the parent roll and child cuts throughout  MMS.

8

Enter the Dye Lot, Width, and Length.

Note: The color information will default to the ordered color and cannot be changed.

9

Press ACCEPT [F1] when complete. A Roll Tag will print for the received product. Attach this to material received or place it within the Stock Tag Book.

Note: If you have received the ordered quantity, the C(omplete) Column will display Y. Press Enter to continue; otherwise, you may make another selection from the Tag ID Options Menu, enter Y in the C(omplete) Column, or press CANCEL [F5] to return.

10

Do you need to make any corrections?

If Then
Yes

If errors are made in the receiving process, these may be corrected before the line is marked C(omplete) by navigating to the line and attempting to type within the RCV-Q Field and making one of the following selections from the Tag ID Options Menu:

  • Modify Existing Tag ID will allow you to change any of the information previously recorded during receiving for the roll. Note that color information cannot be changed.
  • Delete Tag will allow you to delete a recorded tag. Non-rolled goods may be received by typing the quantity received in the RCV-Q Field.

Press ACCEPT [F1] to continue.
No Press ACCEPT [F1].

11

Enter the Actual Receipt Date. 

12

Enter the actual freight in Freight Current Charges.

13

Select YES to accept all in-process receipts.

Notify a member of store management that you have received a purchase order, which is ready to be closed, or select YES to close the purchase order.

Print Labels Option

1

When receiving material on a special order purchase order created from a customer order, you will have the option to print Order ID Labels to help identify these goods. These tags will contain Customer Order information such as Order Number, Site Address and can be placed on rolled goods, boxes of tiles, andboxes of wood for easy identification at the store and by the installer.

2

Do you want to print Order ID Labels?

If Then
Yes Indicate how many labels to print (two tags will print on each label). Select YES to accept all in-process receipts. Notify a member of store management that you have received a purchase order, which is ready to be closed, or select YES to close the purchase order.
No Select YES to close the purchase order.
