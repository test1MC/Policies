Roll Tag Policy

  1. A current roll balance tag is required for all rolled goods products.
  2. Roll Balance Tags and Roll Cut Tags are to be used as follows:
    1. IN-STORE INVENTORY: Printing of these Roll Balance Tags and Cut Tags may be adjusted as required by store. See 14-02 Floorcovering Software for additional information on configuring these system settings.
    2. OFF-SITE INVENTORY: Required. The use of Roll Balance Tags and Roll Cut Tags is required by stores storing inventory offsite in either an SW location or a non-SW location. These tags should be retained in a Stock Tag Book and updated as cuts are made.
