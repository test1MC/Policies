Order ID Labels Procedure

Step

Action

1

From POS desktop, select Store -> Select the Floorcovering Menu.

2

Enter your Employee Number and Password.

3

Select Print Tag ID from the Floorcovering Menu.

4

Enter the Customer Order Control Number or the Purchase Order Control Number.

Note: Press LOOKUP [F3] in either field to look up the purchase order. Select the order and Press ACCEPT [F1] to continue.

5

Enter the Number of Labels you want to print.

Note: Two tags will print on each label.
