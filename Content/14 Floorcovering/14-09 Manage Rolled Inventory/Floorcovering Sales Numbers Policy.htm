Floorcovering Sales Numbers Policy

  1. Stores using floorcovering software are required to use sales numbers to enable appropriate inventory tracking, inventory visibility, and pricing via Price Record Cards.
    Note: Floorcovering Class Numbers have been removed from the manual. Facilities should order and sell all products using Sales Numbers.
  2. Rolled inventory is set up with separate sales numbers for rolls and cuts. The correct sales number is to be used at all times. 
    1. The “roll” Sales Number should be used for most product. 
    2. Only use the “cut” Sales Number when logging a special order product cut by the vendor. 
    3. Rolls and cuts are costed differently and recording inventory properly will maintain the correct costing in the system.
  3. The proper Material Type Code must be assigned when requesting a new product setup. The Material Type Code will determine how the product will be inventoried in POS and what additional information may be required in receiving or when the product is sold or entered into a customer order. Changes to a product’s Material Type Code may be requested by submitting a Product Add/Request Form via SWeep. 
    1. Roll goods are to be classified as follows. Product classified as a roll good will prompt for additional information in POS: color number, dye lot, width, length and the registration/roll number.
      1. RRRoll Good
      2. RCRoll Cut
      3. VRVinyl Roll
      4. VCVinyl Cut
    2. Non-roll goods are to be classified as follows. Product classified as a Non-Roll Good will only prompt for color number and name.
      1. BBCove Base Undefined
      2. BCCove Base Ceramic
      3. BRCove Base Rubber
      4. BVCove Base Vinyl
      5. CMCeramic
      6. LMLaminate
      7. VNVinyl VCT
