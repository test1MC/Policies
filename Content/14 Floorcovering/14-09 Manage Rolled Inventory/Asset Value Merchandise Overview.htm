Asset Value Merchandise Overview

Asset Value Merchandise is merchandise that has been expensed into a job (sale) to account for product purchased in excess of the job requirement. This merchandise (once expensed) no longer has inventory value but it does have asset value to the center because it can be sold at 100% gross profit.

Locations selling floorcovering are able to maintain rolled good Asset Value Merchandise [AVM] inventory. The software enables you to track rolled goods which have been marked as AVM. Lookup screens will show current inventory along with AVM roll goods. The Tag ID Lookup search screen enables you to filter the type of inventory: All, Obsolete, or asset value merchandise. Most AVM will be identified during Work Order Reconciliation. For more information, please see Work Order Reconciliation Overview for more information.

Please note that you are not able to mix product inventory types [Obsolete, Asset Value and Perpetual] on the same line in Customer Order Entry and Point-of-Sale because each inventory type uses a different sales number. Rolled AVM is sold under Sales Number 9999-13247 (AVM Roll) and non-rolled AVM is sold under Sales Number 9999-13254 (AVM- FC Other).
