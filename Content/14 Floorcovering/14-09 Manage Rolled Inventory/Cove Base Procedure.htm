Cove Base Procedure

Step

Action

Customer Order Details
1 From POS desktop, select Store -> Customer Order Entry -> Add Order ->

2

The carpet being used for cove base must be entered as a separate line item.

3

Enter Area Requirements for the cove base using Carpet as the Area Description and Covebase as the Cut Description.

4

Enter the Carpet Cove Base selling price. The selling price should include workroom charges

Work Order Creation
1 From POS desktop, select Store -> Floorcovering Menu -> Installer Work Order -> Add Work Order ->

2

When prompted for Special Instructions, include the following note: “00 x 00 piece of carpet sent to carpet workroom to be made into carpet cove base.”

3

Print the Picking Ticket to send with the carpet to the workroom.

4

Prepare the Work Order for installation after the workroom returns the completed carpet cove.

In Sales Transaction
1 From POS desktop, select Store -> Point of Sale -> Sale ->

2

Press COMMENT [F9] to add Carpet Cove Base to the cove base line.
