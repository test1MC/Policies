Set Up Floorcovering Bin Locations Procedure

Step

Action

1

From POS desktop, select Store -> Select Merchandise Management.

2

Enter your Employee Number and Password.

3

Select Inventory Control from the Merchandise Management Menu.

4

Select Bin Location Maintenance from the MMS Inventory Control Menu.

5

Enter the Sales Number or Press LOOKUP [F3] to search for the product.

6

Enter the Name of the Bin Location where the product is kept.

Note: You may name as many bins as you require for a product. There is a limit of six characters per name

7

Is this the primary location for this product?

If Then
Yes Type P.
No Press Enter to leave it blank.

8

Press ACCEPT [F1] to save.
