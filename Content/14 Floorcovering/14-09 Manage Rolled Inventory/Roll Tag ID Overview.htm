Roll Tag ID Overview

Part

Function

Tag Identification Number

Unique number assigned by the system that is formed by the receive date, purchase order control number, purchase order line number, the Roll/Cut [R/C], and the sequence number.

For example: YYMMDDPPPPPPPLLLR999.

Registration

A Roll #/Serial # assigned by the vendor.

Dye Lot

This number is provided by the manufacturer. It is the amount of carpet dyed at one time. This is important if the customer orders additional carpet and the color needs to be exact.

Width

The width of the carpet roll. Width sizes vary depending on the size of the loom that the carpet was made on.

Bin Location

The section where the carpet is located in the warehouse.

Note: This unique Tag ID will track the parent roll and child cuts through MMS.
