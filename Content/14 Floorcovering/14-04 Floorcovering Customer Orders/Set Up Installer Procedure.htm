Set Up Installer Procedure

  • Canadian Stores Only - Upon receipt of installation request from CanWest:
    • Verify available credit. 
    • Enter Customer Order per instructions outlined below.
    • Communicate approval or denial to CanWest. If approved, provide CanWest with SW Customer Order Number as PO#.

Is this an installation order?

If Then
Yes Select Installation from the Customer Service Options Menu.

Note: If you are creating an order that the customer will pick up, no selection is required from the Customer Service Options Menu.

Is this a contract job?

If Then
Yes

Select Contract Job.

The contract job should be used:

  • For installed orders when a contract customer requests that their invoice displays only one line item for all material and one line for all labor.
  • When there is a difference in billing and inventory.

Note: For more information on Contract Jobs, see Contract Jobs Overview.
No Select Non Contract job and proceed to Step 7.

Note: Installer Notes are meant to aid the installer and will not be seen by the customer. Sometimes it may be best to add installer notes in the work order; if you have two installers, one for carpet and one for tile, you are able to add work order specific installer notes for each installer this way.
