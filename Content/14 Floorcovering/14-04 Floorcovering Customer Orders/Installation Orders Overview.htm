Installation Orders Overview

For installed orders, Customer Order Entry will drive the Work Order and Purchase Order processes.  If any changes are required to the Work Order (products, cuts, pricing), they must be done in Customer Order Entry first. When canceling a customer order line that has been associated with a work order line or work order, the work order line is also canceled. When canceling a Customer Order associated with a Work Order, the Work Order will also be canceled and all cuts made staged for Store Expense. For Will Call orders, Customer Order Entry will help generate a pick ticket for the cutter to use in the Cut Zone.
