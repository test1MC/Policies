Re-Work Work Order Procedure

Step

Action

Re-work Order

1

From POS desktop, select Store -> Select Floorcovering Menu.

2

Enter your Employee Number and Password.

3

Select Installer Work Order from the Floorcovering Menu.

4

Select Add Work Order from the Work Order Menu.

5

Select Re-Work from the WO Type Menu.

Note: Select Re-Work for a previous installation requiring additional attention for repair, re-install, re-stretch, or any other instance in which the customer will NOT be charged. 

6

Enter the Work Order Number or press LOOKUP [F3] to search for the work order. Press ACCEPT [F1] to continue.

Note: When you create a Re-Work Work Order (customer has agreed to pay the original invoice), a Customer Order is NOT required.

7

Enter the Installer Account Number or press LOOKUP [F3] to search for the installer account.

8

Enter a Scheduled Date or press LOOKUP [F3] to access the Scheduler.  See Installer Scheduler Overview for additional information.

9

Identify the Order Lines to be selected:

  • All Available Lines.
  • Selected Lines.

Add Material and Labor to Work Order

1

Enter the Sales Number or press LOOKUP [F3] to search for the Sales Number.

2

Enter the past Color Number.

3

Enter the Dye Lot.

Note: The dye lot preference must be specified when adding rolled goods to a Re-Work Work Order. Choices are: only this dye lot, prefer this dye lot, and not this dye lot.

The system knows that material and labor added on the Re-Work Work Order do not exist on a customer order and sees them as expense items. These items will ’stage’ for the expense transaction.

  • Roll Goods  CY INV SHRINK 3710-125
  • Non-Roll Goods  SERV & SUPPLIES7402-099

4

Do you want to add labor to the re-work work order?

If yes, select Customer Satisfaction from the Customer Will Not Be Billed Menu.

5

Select a reason from the Customer Satisfaction Reason Screen.

6

Complete the installer cost. You will not be able to add customer pricing since the customer will not be charged. Press  ACCEPT [F1].

7

Process cuts.

8

You will have a final opportunity to add or update installer notes.

9

Release the Work Order and print the required installer documents.
