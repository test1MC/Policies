Clear Locked Work Order Procedure

Work Order Status can be cleared to unlock a Work Order that has become abnormally locked due to a system error.

Step Action

1

From POS desktop, select Store -> Select Help Desk.

2

Select Clear MMS/COE/WO Status from the Help Desk Menu.

3

Select Clear Work Order (WO) Update Status from the MMS/COE Maintenance Menu.

4

Enter your Employee Number and Password.

5

Select Yes to confirm clear.
