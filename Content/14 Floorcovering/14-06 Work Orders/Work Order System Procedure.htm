Work Order System Procedure

Step

Action

Process Work Order

1

From POS desktop, select Store -> Select Floorcovering Menu.

2

Enter your Employee Number and Password.

3

Select Installer Work Order from the Floorcovering Menu.

4

Select Add Work Order from the Work Order Menu.

5

Is this a new work order?

If Then
Yes Select New.
No Select Re-work.

6

 

Select the Customer Order from the All Order(s) Screen.

Note: Information from the Customer Order will populate the Work Order Header. Installer Account Number and Scheduled Date are not required when creating the Work Order but MUST be added BEFORE the Work Order may be approved in Reconciliation. 

7

Select the Install Type for the Work Order.

8

 

Enter the Installer Account Number or press LOOKUP [F3] to search for an installer.

Note: This information does not need to be entered when you first create the Work Order but will be required before it is released for installation.

9

Does your installer have multiple crews?

If Then
Yes Select the crew name.
No Proceed to Step 10.

Note: Installation crews will only appear if they have been set up in crew maintenance.

10

Enter a Scheduled Date or press LOOKUP [F3] access the Scheduler. See Installer Scheduler Overview for additional information.

11

Press ACCEPT [F1] to save the Work Order Header.

Work Order Line Details

1

Make the appropriate selection for the job from the Order Line Selection. For example, if both tile and carpet are being installed by different installers, choose Selected Lines and select which lines should be applied to the Work Order.

2

If you chose to select lines, select the lines for installation. Press ACCEPT [F1] to continue.

3

Expense items, for which the customer will NOT be billed, should be directly added to the work order.

Note: LOOKUP [F3] is disabled so the sales number must be entered.

4

 

Enter Quantity in the Assigned Column.

Note: The system knows that these items do not exist on the customer order and shows them as an expense item in the Line Item Information Window. These items will stage for a store expense under the G/L Account SERV & SUPPLIES 7402-099, Line 55 of the P&L. The Line Item Information Window will show that the line is an expense item. You will not be able to make changes to products, color, or required cut information. Changes must be made in Customer Order Entry first. This is to ensure that what is installed and what is sold is the same.  However, changes may be made to labor, and the changes will feed Customer Order Entry. If you need to make changes to your Installer Cost, move to that line and press VIEW [F7]. 

Process Cut

1

 

Use the UP ARROW key to navigate to the first product requiring a cut. Press ENTER to access the Area Requirements Window.

Note: For more information on making cuts including policies and roll balance threshold, stretch variance, and material split, please see Cut Zone Overview.

2

 

Area Requirements will show all cuts that must be made for the job for that product. Press ENTER to assign the cut for each required cut.

 

3

 

Area Requirements for the product to be cut will display in the yellow box. Available rolls for the style and color selected will display Available Length. This listing is sorted by Dye Lot.

4

 

Use your ARROW KEY to move to the line of the roll you would like to cut to fulfill the work order requirements. Then, type in the length you would like to have cut as the Assigned Length. Press ENTER.

5

Identify the roll option from the Roll Options Menu:

If

Then

A roll balance exists

Cut & Label  Roll

Entire roll to be used

Label Entire Roll

Cancel cut and no material reservation is made

Exit Without Action

Note: If a roll balance will exist after the cut is made, you will be required to scan or enter the Tag ID for the roll to be cut.

6

Select Cut Roll from the Cut Options.

Note: You will be prompted to print updated roll balance on stock tag after making all cuts. Printing of roll balance tags may be controlled in Work Order Maintenance. Inventory Stock Tags may be customized to automatically print, not print, or prompt to print roll balance tags while making cuts in the Work Order.  See 14-02 FC System for additional information on configuring these print options. If selected, the new Stock Tag will print for any remaining roll balance from the parent roll. This tag will show Begin Length and Current Length, if Roll Balance was selected.

7

A Cut Tag will print for the material cut.

8

The cut line now displays in green for easy identification. New roll tags will print for the cut required for the job and for any remnant. A new Tag ID will be assigned to each; the C in the Tag ID signifies that this is a cut from a parent roll. Press ACCEPT [F1].

Note: The Area Requirements List now displays YES for Assign Cut for the line, which was just cut. Continue to assign cuts or press ACCEPT [F1] to return to the Work Order Detail.

The Work Order Detail also shows the new cut as Assigned. Once all cuts have been assigned, press ACCEPT [F1].

Assign Pad

1

Enter the pad needed.

Note: You do not need to assign your pad to cut, but you do need to enter the quantity taken on the job in the Assigned Field. 

You are able to enter a different amount in the Assigned Field than what appears on the customer order for padding. Remember that the customer documentation will show the order amount.  Also, inventory will be relieved for the total amount on the order. You will need to complete an Inventory Adjustment to record the difference.

2

Do you want to add Special Instructions?

If Then
Yes Select Y to add additional information.
No Select N to continue to the next step.

3

Do you want to update installer notes?

If Then
Yes Select Y to make any changes.
No Select N to continue to the next step.

4

Do you want to print the work order?

If Then
Yes

Select from one of the following print options:

  • Work Order
  • Work Order Pick List
  • Confirmation of Installation
  • Installer Notes
No Select Do Not Print.

Note: When Confirmation of Installation is selected, an additional screen is displayed requesting an appropriate signature selection to be made. A copy of the Work Order and the Confirmation of Installation Document for the installer will print. 

5

The Confirmation of Installation must be signed and retained.
