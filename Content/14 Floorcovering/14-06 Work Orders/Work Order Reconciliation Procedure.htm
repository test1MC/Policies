Work Order Reconciliation Procedure

Step

Action

1

From POS desktop, select Store -> Select Floorcovering Menu.

2

Enter your Employee Number and Password.

3

Select Installer Work Order from the Floorcovering Menu.

4

Select Work Order Reconciliation from the Work Order Menu.

5

Enter the Work Order Number or press LOOKUP [F3] to search for the Work Order Number. Press [F1] Accept to continue.

6

Identify the type of Reconciliation:

If

Then

Adjust and approve labor and material line items

Reconcile all line items

Note: If you select Reconcile Labor Only or Approve Work Order as your Reconciliation Option and there are items or roll balances to be expensed, you will receive an error directing you to select Reconcile All Line Items.

Adjust and approve labor

Reconcile Labor only

Note: The material lines will show a status of C (closed) and you will only be required to approve labor lines. When approving labor, you will have a final chance to make adjustments to Labor Class Codes, Customer Price, and Installer Cost. Changes made in Reconciliation will back fill both the Work Order and Customer Order.

Approve all lines with no adjustments

Approve Work order

Note: A message will be received that all lines have been successfully approved. This option can ONLY be selected when the Work Order has NOT generated any expense lines.
