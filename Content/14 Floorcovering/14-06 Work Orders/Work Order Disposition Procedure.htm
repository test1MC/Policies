Work Order Disposition Procedure

Work Order Disposition enables you to tie pending work order labor lines to a previously processed point-of-sale transaction.

Step

Action

1

From POS desktop, select Store -> Select Floorcovering Menu.

2

Enter your Employee Number and Password.

3

Select Installer Work Order from the Floorcovering Menu.

4

Select Work Order Disposition from the Work Order Menu.

5

If

Then

Selecting paid labor lines and linking to a POS transaction and/or canceling the labor line

Tie Installer Payment to Previous POS Transaction

Selecting all staged pending expense merchandise

Cancel Staging to Store Expense

Note: Once staged pending expense material lines are canceled, materials will be returned to stock and a new stock tag will print.

6

Identify the Labor Line(s) to expense.

7

Select Process WO Line.

Note: For additional information about the work order, select view or print.

8

Do you want to link the work order line?

If Then
Yes Select Link Work Order Line to POS Transaction.
No

Select Cancel WO Line.

9

Enter the date range of the transaction and press [F1] Accept to continue.

10

Select the Installer Payment to link to the Work Order Line.
