Cancel Work Order Cut Procedure

If cuts have been assigned to an order, they must first be removed from the order followed by untagging the cut in the Work Order. Remember, inventory has not been relieved at this point; the cut has been committed to the order and is not available inventory.This process will return the material to available inventory.

Step

Action

1

From POS desktop, select Store -> Floorcovering Menu.

2

Enter your Employee Number and Password.

3

Select Installer Work Order from the Floorcovering Menu.

4

Select Update Work Order from the Work Order Menu.

5

Enter the Work Order Number or Press LOOKUP [F3] to search for the Work Order Number -> Press [F1] Accept to continue.

6

Using the arrow key, select the Item Detail Line you wish to cancel and view area requirements.

7

Press the tab key -> Press enter to access the Assigned Rolls Options.

8

Do you want to untag the selected roll?

If Then
Yes Select Untag Selected Roll.
No Select Exit without Change.

Note: Indicate whether or not you would like to print a Stock Tag for the cut being returned to inventory. If the cut was made from material received on a special purchase order, you may also need to un-reserve the cut in the Customer Order.

9

Carpet assigned, but not physically cut?

Remove the non-existent cut from inventory.

  • Perform a perpetual inventory adjustment to reduce the cut to zero.
  • Explain in the Memo Area that the carpet was not originally cut from the roll and you are moving the inventory amount back to the parent roll.

Correct inventory of parent roll.

  • Process another perpetual inventory adjustment to increase the size of the parent roll.
