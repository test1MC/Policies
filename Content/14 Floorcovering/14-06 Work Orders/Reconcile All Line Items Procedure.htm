Reconcile All Line Items Procedure

Step

Action

1

Is the installer who completed the work different from the original work order?

If Then
Yes Select Y to update the header information.
No Select N to continue.

Note: Changing the installer account number on the order header will backfill the work order and ensure the correct installer is paid through disbursement.

2

When selecting Reconcile All Line Items, items that have not been reconciled will display as N (new).

3

 

Typing A (approve) in the Status Column will allow you to reconcile the item.

4

 

Are you returning any material?

If Then
Yes Proceed to Step 6.
No Type 0.00 as the Length. Press enter to continue.

Note: To reconcile a roll good, you will note any material being returned from the job that will either be returned to inventory or expensed by typing the returned length in the Length Field.

5

 

Select Approved to complete the transaction and press [F1] Accept to return to the previous screen.

Note: Select No Action to cancel and return to the previous screen.

6

Enter the Return Length.

Note: Returning any amount in Reconciliation will require the same adjustment for that product through Customer Order Entry.

7

Identify the type of return:

If

Then

Expense Disposal

Amount disposed will be staged for Store Expense under CY INV SHRINK 3710-125. You must complete a Store Expense Transaction for the reconciled material.

Expense to Asset Value

You will be prompted to print a Stock Tag.

Return to Inventory

You will be prompted to a Stock Tag.

Note: If you select Return to Inventory or Approved, your product will now show a status of C (closed). Products selected for Expense Disposal will show a status of A (approved). Once the Store Expense Transaction has been completed, the status will automatically change to C (closed). When approving labor, you will have a final chance to make adjustments to Labor Class Codes, Customer Price, and Installer Cost. Changes made in Reconciliation will back fill both the Work Order and Customer Order. If your Return Quantity does not match your Reserved/Assigned Quantity, you will receive an error message asking to re-enter the length being returned.

8

Press [F1] Accept to complete the transaction and return to the previous screen.

9

Do you want to enter/change any special instructions?

If Then
Yes Enter special instructions and press [F1] Accept to continue.
No Select N and press [F1] Accept to continue.

Note: If any material has been returned through Reconciliation, the system will warn you that the reconciled quantity no longer matches the quantity in the customer order. You MUST make changes to the customer order BEFORE you can bill the customer for this product.  If line item will not be sold to the customer (entire amount returned), you can opt to perform Order Disposition instead of adjusting the order.
