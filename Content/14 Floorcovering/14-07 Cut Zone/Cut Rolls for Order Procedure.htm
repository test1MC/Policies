Cut Rolls for Order Procedure

Step

Action
1 From POS desktop, select Store -> Floorcovering Menu -> Cut Zone -> Cut Rolls for Order.

2

Select  how the orders should sort:

If

Then

Show rolls staged in work orders

Installation Orders

Show rolls on customer orders not for installation

Non Installation Orders

Note: The listing is sorted by sales number so you may see several cuts of the same material required for different orders. If the COE Required Date is used, this can be referenced to determine which cuts are required first. For each line you will see the order Area Requirements and any reserved (through COE) Tag ID in the lower window.

3

Press enter to access the lower half of the window.

Note: When the lower window is active, the order will be highlighted in black. Press CANCEL [F5] to return to the upper window.

4

 

If a Tag ID has been reserved for an area, you can view Tag ID Information from this window by pressing TAB. Using the UP ARROW and DOWN ARROW will navigate you through the Tags.

5

Select an area to prepare and press Enter to select a cut to prepare.

6

Enter the Length of the cut in the Assigned Length Field.

7

Select Cut & Label Roll from the Roll Options Menu.

8

Scan or enter the Tag ID of the roll to be cut.

9

Select Cut Roll from Cut Options.

10

Handle any roll balances or stretch variance amounts appropriately.

11

Attach the printed Roll Cut Tag to the cut and update the tag on any remaining inventory with the updated Roll Balance Tag.
