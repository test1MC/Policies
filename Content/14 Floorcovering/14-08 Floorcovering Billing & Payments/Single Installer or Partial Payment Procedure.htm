Single Installer or Partial Payment Procedure

Step

Action

1

From POS desktop, select Store -> Customer Order Entry -> Add Order -> Installation -> If the installer will be paid in installments throughout the installation rather than in a lump-sum at the end of installation, enter separate labor lines for each partial pay.

2

From POS desktop, select Store -> Floorcovering Menu -> Installer Work Order -> Work Order Reconciliation.

When completing Work Order Reconciliation, approve only the labor line that you choose to pay now.  Then, the next draw period, approve the next labor line and so forth.

3

From POS desktop, select Store -> Point of Sale -> Press Page Down to access Special Menu -> Disbursement -> Installer Payment by Store Draft.

When you pay the installer, you will see that the terminal/transaction number will be blank because the sale transaction has not been processed. Once the sale is processed the transactions will be tied together.
