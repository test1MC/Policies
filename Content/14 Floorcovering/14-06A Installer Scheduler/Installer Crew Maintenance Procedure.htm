Installer Crew Maintenance Procedure

Installer Crew Maintenance allows you to manager installers who have multiple installation crews. The installation crew for the job can be specified through:

  • Create work order
  • Update work order
  • Work order reconciliation
Step Action

1

From POS desktop, select Store -> Select the Floorcovering Menu.

2

Enter your Employee Number and Password.

3

Select Installer Work Order from the Floorcovering Menu.

4

Select Maintenance from the Work Order Menu.

5

Select Installer Crew Maintenance from Work Order Maintenance.

6

Enter the Installer# or press LOOKUP [F3] to search for the installer.

7

Enter the following information:

  • Crew name
  • Phone
  • # of crew members

8

Identify the types of installation that the crew can perform.

9

Enter the Install Type or press LOOKUP [F3] to search for the Install Type.

10

Press LOOKUP [F3] to search for a Description.

11

Enter Daily Capability by square footage.

12

Press ACCEPT [F1] to save your changes.
