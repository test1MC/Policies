Installer Scheduler Procedure

Step Action
Access Scheduler

1

From POS desktop, select Store -> Select the Floorcovering Menu.

2

Enter your Employee Number and Password.

3

Select the Scheduling Menu from the Floorcovering Menu.

4

Select the Scheduler from the Installer Schedule Menu.

5

 

When using the Scheduler, use the ARROW KEYS to navigate to the day on which you wish to schedule the installer for the job. Press ENTER to select that date.

Note: You will not be able to select dates, which have already passed. You may also us VIEW [F7] to access a calendar.  Using Shift + < and Shift + > will allow you to change the month you are viewing. Dates may be selected by using the ARROW KEY and pressing ENTER. 

6

Are you are accessing the Scheduler from within a Work Order?

If Then
Yes The detail information will populate for you and comments may be added.
No

Select a date to schedule for an installer. You will need to complete the Schedule Detail screen. Customer Number and Install Type are all that is required.
Print/View Schedule Entry

1

Select Print/View Schedule Entry from the Installer Scheduler Menu.

2

Enter Start and End Dates.

3

Enter the Installer Account Number of press LOOKUP [F3] to search for the Installer Account.

Note: The report can be displayed on the screen or sent to a printer. Your selection criteria can also be changed.
