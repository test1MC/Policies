Create Inter Store Request (ISR) Procedure

Step

Action

1

From POS desktop, select Browser -> Access Source homepage -> Use the Store Search to determine what locations have the stock available to fulfill the request.

2

Contact the store and agree on the terms of the purchase order, including cost.

3

From POS desktop, select Store -> Customer Order Entry -> Add Stock Transfer Order [ISR/IST].

4

Enter the Store Number.

Note: The store must already be set up in Transfer Store Maintenance (From POS desktop, select Store -> Customer Order Entry -> Maintenance -> Transfer Store Maintenance).

5

Press ENTER at the Customer Service Options Menu to continue.

5

Enter the product(s), quantity, and required date.

Note: If you need to have the servicing store tint the product,  include color information for the relevant products.

6

Enter Special Instructions if necessary. Select Print Review Options.

7

Select Transfer Order from the Order Entry Menu.

7

Narrow  search in the Order Lookup Window or press ACCEPT [F1].

8

Highlight the correct order and press Enter.

Note: The customer name will always be “**Stock Order **.”

9

Enter the store number of the transfer location. Press ACCEPT [F1] when complete.

10

Choose YES when prompted, “Transfer this order?”

Note: Once the transfer function has been completed, the originating location no longer has control of the order and changes cannot be made. Orders with the status “In Update” may be orders that have been transferred.

11

An ISR Purchase Order is automatically created through the transfer process. You will need to receive against this once the order is transferred back. Once an order has been transferred back to your location, the “Transferred In” number in the Status Window will increase.
