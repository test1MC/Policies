Credit Card Error Message Overview

If there was a system problem while processing a Point of Sale Transaction, you may see the following message (follow the instructions to correct the problem):

"The transaction just processed was abnormally terminated and the customer’s credit card as not charged. Please follow one of these procedures to process the transaction again:

  1. Tell the customer that the transaction was voided and did not go through. Process the transaction and swipe the customer’s credit card at form of payment.
  2. If your system is not operating, you are required to manually process the transaction on a Manual POST Form. Get a card imprint of the customer’s credit card and call for voice authorization. Enter the transaction in the system when it is working properly. Remember to select Enter and Validate Manual Form.
  3. If the customer has already left the store, call Store Computer Operations at 216-566-2740 for processing instructions."

The following message may print on the customer’s receipt if there is a system problem while processing the transaction:

"This transaction was voided and will be processed again. Please contact the store at the above phone number if you have questions."
