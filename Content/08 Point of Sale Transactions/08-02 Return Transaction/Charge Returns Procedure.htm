Charge Returns Procedure

Step Action

1

Obtain the Store Manager’s approval.
  From POS desktop, select Store -> Point of Sale -> Return -> Select Return Reason Code -> Enter customer information -> Select S-W Charge from Form of Refund Menu.

2

Returned merchandise must be credited against the customer’s S-W Account. Do not issue any refunds in currency or check.

3

After the employee and customer have signed the return, file the store copy in the computer system forms container.

4

If the return creates a credit balance for the account and the customer wants the amount refunded, then process a Payment Return.

Note: For more information, see Account Adjustment Overview.
