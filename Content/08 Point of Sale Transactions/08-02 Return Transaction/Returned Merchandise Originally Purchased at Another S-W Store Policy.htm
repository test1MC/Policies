Returned Merchandise Originally Purchased at Another S-W Store Policy

  1. Know in advance how your District Manager wants you to handle this type of return.
  2. Receipts should be required for this type of return.
  3. Call the selling store to verify the purchase if the customer does not have the receipt ,and you choose to accept the merchandise. Once verified, process a normal return transaction.
    1. From POS desktop, select Store -> Point of Sale -> Select Return from the Sale/Payment Menu.
    2. Press ACCEPT when prompted for the original terminal/transaction number (if you start to enter part of the information the computer system assumes that you know all the information).
    3. If the merchandise can be placed in your inventory process, the return will set at the lowest promotional price assuming the sale was a retail transaction.
    4. Press the COMMENTS key and explain the reason for the return.
  4. If appropriate, place the merchandise in inventory. If not, arrange an agreeable solution with the selling store through the direction of your District Manager.
  5. The visiting store should check with the original store to verify that the return can be made, vendor re-stocking charge, etc.
  6. Returned special orders need to be IST'd to the selling store so that the return to vendor can be processed.
  7. All other related return policy and procedures must be followed for this type of return.
