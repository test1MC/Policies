Special Order Return Policy

  1. Customized merchandise, such as tinted paint, made-to-measure draperies, and specially cut carpeting CANNOT be returned without Manager or District Manager approval. Normal Store Manager approval-level (up to $500) applies to the return of special order merchandise.
  2. If the merchandise was a special order item, such as wallpaper, charge the customer for the return or handling fee.

Note: Please see Return Transaction Procedure for steps on how to return special orders once District Manger approval is received.
