Automotive Cash Returns Over $100 Procedure

This section only applies to the Automotive Division.

Step Action

Processing Returns Over $100

1

Refer to Return Guidelines. From POS desktop, select Store -> Point of Sale -> Return -> Select Return Code.

Note: An item that was originally paid for by check cannot be returned until ten (10) business days after the sale.

2

Select Cash as the Form of Refund, but do not actually give any cash to the customer.

3

Both you and the customer must sign the return receipt. Make a note on the customer’s receipt that a check will be mailed to them.

Note: The customer does not need to remain in the store for the following steps. Be sure that you have the customer’s information on file.

Processing Miscellaneous Cash Receipts

4

From POS desktop, select Store -> Select Point of Sale from the Main Menu. Enter your Employee Number and press ENTER.

5

Press PAGE DOWN at the Sale/Payment Menu to access the Special Menu.

6

Select Misc Cash Receipt from the Special Menu.

7

Enter complete customer information when prompted, including name and address -> Press ACCEPT [F1].

8

Press ENTER to view the GL Account List -> Select A/R Suspense (Prime-Sub 0300-003). All check returns from the main office must be itemized under this entry.

9

Enter the total refund dollar amount that displayed in the Customer Return Transaction.

10

Press COMMENTS [F9] and type “Cash Refund Over $100—HQ Will Issue Check.”

11

Select Cash from the Form of Payment Menu.

12

Press TOTAL [F12] for the total amount.

Processing E-Fund Requests

13

From POS desktop, select Browser -> Access Source homepage -> Log in to E-Accounting and select the eFunds tab at the top of the screen.

14

Select New from the Menu -> Make sure that the Radio Button is On and select Submit. An eCheck number will automatically be assigned.

Note: The eCheck number is an eight-character system incremented number beginning with "CK" and ending with the year it originated.

15

Select Save.

16

Select Automotive in the Division Name dropdown box.

17

Select Automotive-Other in the Site Name/Num dropdown box and enter your branch information.

18

Use the description box to specify that the e-Fund request is for a “customer cash return > $100.”

19

Select No for 1099 Vendor & Law Firm/Attorney/Medical. Leave the SSN/Tax ID and eFCA/eExpense boxes blank.

20

Add the customer and all applicable customer information as a new vendor. Check must be payable to the same customer name as the original sales transaction (should not be an individual if the original was to a company).  

Note: Leave the Due Date box blank.

21

Under Request Details, enter the total amount of the cash refund in the Item Amount box. The check issue amount should match the total amount of the refund.

22

Enter C101 for Division. The Account should be the same prime-sub used to process the Miscellaneous Cash Receipt (Prime-Sub 0300-003) -> Enter 79XXXX for the Cost Center, with XXXX being your branch number.

Note: Canadian branches should use Division C220 and Cost Center 77XXXX.

23

Input 0000 for Future, Project Code, and Inter-Company.

24

Enter the total amount of the refund in Account Amt.

25

Select Item for Line Type.

26

Fill the In Payment box with the original sales terminal number, transaction number, and date. Also, note the return terminal number, transaction number, date, and return amount. If the customer did not have the original receipt, include the reason for the return.

27

Under Check Handling Instructions, select Mail direct to vendor and indicate that the check is to be mailed to the customer for a cash return.

28

Every e-Fund request needs at least two approvers. The originator counts as the first approver. Your ASM should be added as a second approver. Follow expense approval guidelines and add the proper approvers accordingly.

29

Fax a copy of the original sale, return transaction, and miscellaneous cash receipt to 216-586-1800.

30

At the bottom of the screen, you will see Save Request, Submit, or Cancel. Select Save if you need to come back to the request. If it is complete, select Submit. You will receive the following prompt: “Check will be issued for (total amount).” This gives the originator one last opportunity to change, correct, add to, etc. before releasing the eCheck. Validation occurs on certain fields when you select Submit. You will be notified if any required fields are missing or if the account code is invalid. If an error message is received, correct the error and select Submit. The eCheck is released to the next approver on your approved list. The next approver will receive an e-mail message that eCheck CKXXXXXX is awaiting approval. The brief description entered on the eCheck will also appear in the message of the e-mail. When all approvers have approved the request, you will receive a confirmation email.
