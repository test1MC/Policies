Wallcovering Return Procedure

This section does not apply to Automotive Branches.

The following Wallcovering Return Message will print on both sale and correction invoices whenever wallcovering is sold:

"Unopened double rolls of wallcovering are returnable within 30 days of the date that you are notified that your order is in stock. Returns are subject to 25% restocking fee or return freight charge."

Note: Does not apply to cut rolls.

When processing a wallcovering return or correction transaction, determine if the wallcovering will be returned to the vendor. After entering the Book/Pattern number, quantity, and price, you will be prompted to enter the last 2 digits (gross profit) of the sales number.

If Then

The return is going to a vendor

Process the return/correction transaction. After entering the book/pattern number, quantity, and selling price for the returned wallcovering, press DISC [F11] and enter the 25% restocking fee. This process is necessary to correctly report the inventory dollars for the returned merchandise. Do not change the gross profit percentage. Press ENTER when prompted to enter the last 2 digits of the Sales Number.

The return is not going to a vendor

You may process a correction or return transaction to correct the billing for the customer. If the wallcovering will not be returned to the vendor, be sure to change the gross profit to reflect the same gross profit captured on the original sales transaction. This change is necessary to report the correct margin—excluding the 25% restocking fee—because the merchandise will not be returned to the vendor.
