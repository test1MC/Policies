Store Draft for Refund Procedure

This section does NOT apply to Duron, MAB Paints, Columbia Paint or General Paint.

Step Action
1 From POS desktop, select Store -> Point of Sale -> Select Return from the Sale/Payment Menu -> Select Return Code.

2

Enter the transaction as normal return. If available, provide complete original transaction information. Enter complete customer information, including: name, address, city, state, and phone number.

3

After entering the returned product information, select Store Draft from the Form of Refund Menu. If not already provided, a prompt will appear for complete customer information.

4

The System Manager Password is required before refunding by Store Draft. Enter the Draft number to process the Store Draft.

5

Have an authorized signer (either Store or Assistant Manager) complete the draft.

6

Attach the store copy of the draft to the Store Draft Register that prints at the end of the day.
