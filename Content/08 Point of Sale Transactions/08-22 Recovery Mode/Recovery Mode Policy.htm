Recovery Mode Policy

In the event of a serious system failure, re-entry of all lost transactions based on the Journal Tape or transaction copies will be entered at the store.

  1. Recovery Mode should only be used if instructed by Store Computer Operations.
  2. Enter only transactions that have previously been run through the Point of Sale when instructed by Store Computer Operations.
  3. Employees must use the Journal Tape to enter all transactions.
    1. If a Journal Tape is not available, use the store copy of the transaction for entry.
  4. Any transaction entered in Recovery must mirror the original entry.
  5. Any transaction originally entered into the point of sale must be entered during Recovery Mode.
  6. Enter the employee number for the person running the original transaction.
  7. All pricing must be entered manually. Pricing will not automatically return to the screen.
  8. While running a Close Transaction in Recovery mode, enter the total drawer count. For more information, see 08-06 Close.
  9. Batch recovery transactions with daily transactions according to batching instructions.

Note: The District Manager will be notified of any recovered transaction not reported to Store Computer Operations.
