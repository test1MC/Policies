Recovery Mode Procedure

After the store computer system is operating properly, follow the Recovery procedures listed below to make sure that no transaction information is lost.

Note: The transactions will be flagged as having been created in Recovery Mode. All transactions entered in Recovery Mode will be polled by the central computer at Headquarters. If an error is made during Recovery, contact Store Computer Operations immediately at 216-566-2740.

Items needed to enter transaction information in the Recovery Mode:

Item Description
Journal Tape Covering All Transactions

If the Journal Tape is not available, use the original pink store copies of the transactions during Recovery Mode.
Transactions Paid By Check

Enter as Cash.
Return Transactions

Necessary to capture the customer information not on the Journal Tape.
Highlighting Marker

Check off each transaction entered during Recovery Mode.
Original Pink Store Copies

Must be available for referencing if needed.

Note: Enter transactions one day at a time (a complete Open to a complete Close) on the point of sale.

Step Action

1

Store Computer Operations will give instructions as to the transactions that should be entered during Recovery Mode. Enter only those transactions.

Note: If the computer system is only down for part of the day, Store Computer Operations will instruct you to enter only those transactions that were lost in memory.

2

While running a Close transaction in Recovery Mode, enter the total drawer count as it appears on the Close transaction (from the Journal Tape).

3

While entering transactions during Recovery Mode, do not forget to check off each transaction on the Journal Tape after re-entering. This procedure will help to eliminate errors in re-entering transaction information.

4

If the Recovery Close does not match the original close (Over/Short is not the same), check the transaction copies with the Journal Tape to find the error.

Note: Once the error is located, call Store Computer Operations (216-566-2740) for further instructions.
