Cancel Store Expense Procedure

Step Action

1

From POS desktop, select Store -> Point of Sale -> Access Sale/Payment Menu with Employee Number -> Press Page Down for the Special Menu -> Select Store Expense Cancel.

2

Enter:

  • Original Terminal Number
  • Original Transaction Number
  • Original Date
  • Original G/L Account

3

Press COMMENT [F9] to enter the explanation for running the cancel transaction.

4

After entering all of the original information, the computer will prompt to enter the billing information.

5

Enter all product information for cancellation and total the transaction.
