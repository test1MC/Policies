Return Disposition Mode Policy

  1. The Store Manager must review the printed store copy for customer and employee signatures and initial the box in the upper left corner.
    1. The store copy of the return must be retained because it contains the customer and processing employee signature.
  2. Related transactions (i.e. Chargeback, Store Expense) to the return must be processed within 48 hours of the return.
  3. Return Dispositions must be resolved within 48 hours of the original return transaction unless the Store Manager has been scheduled to be out of the store during the 48 hour period.
  4. The Store Manager may delegate the return review process in Return Disposition Mode to the Assistant Manager or Operations Manager with written approval from the District Manager.
    1. Once District Manager approval is on file, the designated employee will be able to perform the Return Disposition process.
