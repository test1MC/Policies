Return Disposition Mode Procedure

Step Action

1

From POS desktop, select Store -> Point of Sale.  Enter Employee Number -> Select Enter Return Disposition Mode from Sale/Payment Menu.

2

Enter your password.

3

Select the return you wish to process.  Press Enter.

4

Select Process Transaction to indicate the mode of disposition.

5

Indicate how the return goods were disposed.

Option Description

Merchandise Returned to Stock

No additional transactions are required or process the proper transaction to finalize the customer return.

Link to Chargeback

Merchandise is being returned to the vendor. Select the vendor chargeback from the listing. The chargeback must already have been created to select this option.

Link to IBAR

Merchandise is being returned to the DSC. Search for and select the appropriate IBAR on which the product is being returned. The IBAR must already have been created to select this option.
IST

Merchandise is being sent to another store. Process an IST directly from this menu or link to an existing IST.

Store Expense

Merchandise is being expensed. Process a Store Expense directly from this menu or link to an existing Store Expense.

6

After completing the necessary transaction, select Choose Another Return/Correction Transaction or Exit Return Disposition Mode.

7

When prompted, indicate that you wish to close the return disposition.

Note:  The system will automatically cross-reference all transactions.
