PQR Details on SWeep Procedure

Once directed to SWeep via the browser, follow these steps to enter the PQR details.

Step Action
Complete PQR

1

Select a Failure Reason Code from the drop down menu.

Do you need additional help?

If Then
Yes Select Show Reason Description for definitions and pictures if available.
No Proceed to Step 2.

Note: Only select Other if there is no reason code listed that describes the issue. Choosing this option will result in additional research and delay the response time to process your request.

2

Answer the questions about the complaint.

Note: The questions asked are based on the Failure Reason Code chosen.

3

Identify the Batch Date:

Can you locate and/or read the batch date?

If Then
Yes Enter the first two letters of the Plant Id, plus the Julian Date.
No Select No Batch Date Reason and choose a Reason from the drop down box.

For additional information, please see:

  • Batch Date Overview

Note: If no batch number is entered, it will delay the response time.

4

Enter the Quantity.

5

Enter the Sequence Numbers.

6

Enter the Estimated Liability for the Product and any associated Labor Costs.

7

Enter the Job Information.

8

Select the Type of Help you need.

9

Complete the Comments Field: Enter any additional detail or relevant information.

10

Select Exit to return to POS.

Note:  The store will receive a Test Report letter containing the specs for the product, where the batch was passed at, and where the retain results were sent.  You will receive this letter automatically via e-mail.
Additional Steps

1

Do you need to submit a paint sample?

If Then
Yes Instructions will be displayed and mailed to you on how to send the sample. You must request a PQR Shipper kit from LabelMaster and complete the shipping information in the PQR located under the Technical Review Tab.
No The PQR is completed.

Note: If the field sample request has not been received after 10 days, an automated reminder will be sent to the store.

For additional information, please see:

  • LabelMaster PQR Shipping Procedure
  • PQR Type of Help Overview
