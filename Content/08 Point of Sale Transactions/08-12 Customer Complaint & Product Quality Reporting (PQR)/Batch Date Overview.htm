Batch Date Overview

The following chart explains how to read/interpret a Batch Date:

Part Function

A or AA

First two Alpha Characters

The first line of the code contains the Batch Number (identifies the plant where the paint was manufactured).

Four digit numeric

Identifies the Julian date (the date the product was made).

Note: The first three numbers identify the day of the year, ranging from 1-365, that the product was made. The last number is the calendar year.

A..Z

Two digit Alpha Suffix

Identifies the Batch Number, where A represents the number 1 in the alphabet and Z represents the number 26. If a second letter is present, it indicates there were more than 26 batches made that day.

Color Sequence Number

The numbers on the second line of the container code under the Batch Number represent the sequence container number. Example: 05478 is the 5,478th can from the batch.

Batch Maker Code

The letters on the second line of the container code under the Batch Number represent the batch maker’s initials. For example: SM MAW
