Automotive Store Expense Overview

Applies to Automotive Branches Only

Cost and extended cost is available on the following screens:

  • Store Expense
  • Store Expense Cancel
  • Inquiry [F4] – Invoices Section: Inquiry Menu, By Transaction Number

An additional prompt will appear giving you the option to view cost.

Note: The Manager Password is required to see cost and extended cost.
