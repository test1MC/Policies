Email Receipts Procedure

Send an electronic receipt to a customer's email address from within the Inquiry Menu or a POS transaction.

This section does not apply to the Caribbean and Quebec.

Step Action

1

Note: This procedure is only available for customers that have a PC or Employee account and an email address on file.

If Then

Issuing (or re-issuing) electronic receipts

 From POS desktop, select Store -> Press Inquiry[F4] -> Select Inquiry Menu -> Select POS—Point of Sale -> Enter the appropriate parameters to search for the transaction -> Highlight the transaction and press Enter.
Processing a transaction From POS desktop, select Store -> Select Point of Sale -> Sale -> Enter customer information and items for purchase.
 
If Then
Issuing (or re-issuing) electronic receipts Select Email Receipt to Customer.
Processing a transaction

After entering and collecting the customer’s form of payment, the Receipt Options screen will appear. According to the customer’s preference, select:

  • Print Only (Default)
  • Email Only (a physical barcode will still print and must be given to the customer)
  • Print & Email

Note: To amend receipt selection, prompt Email Options from the Form of Payment menu.

6

The system will return to the email address on file.

If Then

The address is correct

Press Accept [F1].

The customer wants to change the email address for the current transaction only

Enter the new address -> Press Accept [F1].

The customer wants to permanently change the email address on file

Enter the new address -> Press Accept [F1] -> Update the Customer Account Information on SOURCE (see )

Note: Changes made to the customer’s email address from the Form of Payment Menu will only affect the current transaction. Changes will not be saved for future transactions unless the employee updates the Customer’s Account Information on SOURCE. If the customer’s email does not exist, you will receive an email notifying you that the receipt could not be sent.

7

Only if Issuing (or re-issuing) electronic receipts: Press Enter to send the receipt.
