Gift Card Return Procedure

Process a Return Transaction for a Gift Card or a Cash Refund for the Balance on a Gift Card.

Step Action
1
If Then
Cash Refund (Select States Only) Proceed to Step 2.
Gift Card Refund

If the customer only has the original receipt, send an email to FSS Bookkeeping at bank.questions@sherwin.com. Please include the following information:

  • Terminal and Transaction Number
  • Transaction Date
  • Gift Card Number
  • Customer Name, City, State, Zip Code, and Phone Number
  • Proposed Gift Card Balance
  • 4-Digit Store Number
  • Store Contact

If a replacement is approved by FSS Bookkeeping, deactivate the original card and issue a replacement using the procedure below.
2

From POS desktop, select Store -> Point of Sale -> Select Return from Sale/Payment Menu -> Select Reason.

3

If available, scan the barcode at the bottom of the original receipt.
4
If Then
Cash Refund (Only Select States)
If Then
Current store is not the original selling store

Enter the original terminal/transaction number and date.

The customer only has the Gift Card

Press ENTER three times. Do not enter anything in the terminal/transaction number and date fields.
Gift Card Refund
If Then
Current store is not the original selling store Enter the original terminal/transaction number and date.

The customer only has the original damaged card

Press ENTER three times. Do not enter anything in the terminal/transaction number and date fields.

5
If Then

Current store is not the original selling store

Enter the original terminal/transaction number and date.

The customer only has the original damaged card

Press ENTER three times. Do not enter anything in the terminal/transaction number and date fields.

6

Complete customer information is required, including: customer name, address, city, state, zip code, and phone number.

7

Add the Gift Card line item to the return:

If Then

Able to scan the original receipt

Press LOOKUP [F3] to select the original line entry.

Receipt information is unavailable or entered manually

Press ENTER when prompted to enter the Sales/Product Number. Select Gift Card from the Non-Merchandise Menu.

8

Swipe the Gift Card when prompted. The computer system will automatically retrieve the available balance on the Gift Card and return it to the screen.

9
If Then
Cash Refund (Select States Only) Press TOTAL [F12] and select Cash from the Form of Refund Menu.
Gift Card Refund

Press TOTAL [F12] and select Gift Card from the Form of Refund Menu.

Activate a new Gift Card using the Credit Card Authorization Reader. Swipe the new card when prompted. The new card will be activated for the same amount processed on the Return Transaction.
