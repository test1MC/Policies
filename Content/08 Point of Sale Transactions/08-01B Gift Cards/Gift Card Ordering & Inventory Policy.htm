Gift Card Ordering & Inventory Policy

  1. Gift Cards are expense merchandise and are not inventoried. Gift Cards do not need to be included in cycle counts or physical inventory counts.
  2. Gift Cards may be transferred to another Sherwin-Williams location using Asset Retirement/Transfer of Fixed Assets Form in the Forms Repository.
    1. Find the Asset Retirement/Transfer of Fixed Assets in the Forms Repository. Search form name or keyword “fixed” in the Form Name field. All available forms containing your search word will appear below in the form description field.
  3. Gift Cards can be ordered from Order Express (access Order Express: from POS desktop, select Browser -> Access Source homepage -> Information Directory -> Forms Repository). Gift Cards are packaged twenty-five (25) to a pack, so ordering a quantity of ‘1’ will supply with twenty-five (25) Gift Cards. Gift Cards should be ordered using the following sales number: 400-0335.
    1. The Gift Card merchandiser display can be ordered from Order Express as well. Only Sherwin-Williams displays are available. Order using sales number 700-1410.
