Configure Media Check – C711 PostScript Print Driver Procedure

Disable Media Check in the PostScript driver for proper operation when printing labels out of the HALS system. If Media Check is not configured correctly, the printer will prompt to install paper in a tray with every print and will not print until a button is pressed on its front panel.

Step Action

1

Select the Start Button on your computer.

2

Select  Devices and Printers from the Start Window.

3

Select the Printer and then Printing Preferences.

4

Select on the Advanced Button.

5

Under Printer Features, select  Off for Media Check.

6

Select Ok to save changes.

7

Select Ok to exit Printing Preferences.
