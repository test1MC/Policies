HALS 2000 Overview

This applies to Automotive Branches Only

HALS 2000 is the HEARS Automated Label System. The purpose of the HALS software is to provide container labels that display specific, current, and accurate regulatory information. Select the link below to access the HALS Getting Started: HALS 2000

Find the HALS Automotive Label Order form in the Forms Repository. Search form name or keyword “hals” in the Form Name field. All available forms containing your search word(s) will appear below in the form description field.
