Zebra Printer Setup Procedure

Set the Zebra Printer as the default printer and configure the Zebra printer.

Step

Action

1

Select the Microsoft Start button in the lower left corner of the screen.

2

Select Devices and Printers dialog shown on the menu to the right.

3

Select the Zebra Printer shown.

4

Select Set as Default Printer from the pull down menu (a check should appear on the printer).

5

Select the Zebra printer shown again.

6

Select the Printer Properties tab and then Preferences at the bottom of the window.

7

Select the Option Tab.

Option

Setting

Darkness

10

Paper Format

Landscape

Size

Width 2.25, Height 4.00

8

Select Apply and then OK to return to the Printer Properties window.

9

Select the Advanced Tab, and then select Printing Defaults.

10

Set the exact same settings for darkness, paper format and size.

11

Select Apply and then OK. Your Zebra printer settings are now correct and complete.
