Zebra Label Printer Setup Procedure

Step Action

1

Go to Devices and Printers from the Start Menu.

2

Locate the Zebra USB Printer or Zebra Printer (depends on how your printer is connected). Select View Queue which will open the Queue Window.

3

Select Printer in the upper left corner and choose Open As Administrator.

4

Select Printer in the upper left corner and select Properties.

5

This will give you the Zebra Printer Properties Window. Settings will need to be changed in two places.

6

The first change is Printing Preferences on the General tab. The General tab should be the default tab when the window opens. Select the Printing Preferences button near the lower center. The Printing Preferences window should open.

7

Make sure the Paper Format is set to landscape and change the Width size to 2.25 and the Height size to 4.00 as shown in Step 6. Then select Apply.

8

The second change is the Printing Defaults on the Advanced tab. Select the Advanced tab. Select the Printing Defaults button near the lower left. Change these settings in the same way – landscape, 2.25 and 4.00 as was done in Step 7. Then select Apply again. Select OK and that should close the Printing Defaults Window. You should be back on the Printer Properties window.

9

Select Apply and then select OK. That will close the Printing Preferences window.

10

Are there any print jobs in the queue?

If Then
Yes They must be deleted and restarted for these changes to apply.
No

Close all remaining windows and verify the changes by printing a label.
