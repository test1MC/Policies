iFex Driver Installation Procedure

This applies to Automotive Branches Only.

When you receive a new PC or if the iFEX® driver is uninstalled for any reason, you must first install the driver that allows iFEX® to communicate with your PC.

Step Action

1

Login to FormulaExpress® using your custom login routine and credentials.

2

Once in Preferences, select the General Tab.

3

Once in General Preferences, select the Yes Button next to Activate iFEX® Spectro.

4

Select the Save button at the bottom of the page and return to the FormulaExpress Main® Menu.

5

The iFEX® icon is now available under Color Search. Select the iFEX® icon.
