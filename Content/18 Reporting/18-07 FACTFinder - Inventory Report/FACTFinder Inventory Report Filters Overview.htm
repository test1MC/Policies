FACTFinder Inventory Report Filters Overview

The Inventory Report has several options for how to sort the product information presented:

  • All Products
  • By Sales #
  • By Vendor
  • By Product Family

Users can filter the products by department or type as well:

Filters

Description

No Sales—12 Months

This filter displays products that have been sold in the last 365 days and can be used to identify slow moving products. To reduce slow moving/excess inventory:

  • Search for stores that may be moving the product by selecting on the Sales # (Stores highlighted in green have two months or less supply)
  • Contact customers who have previously purchased the products.
  • Use the Authorized Return Process (located in the Forms Repository).

Note:  All Stores within the District plus 30 miles of the border are included.

Month Supply > 10

This filter displays products with a 10 month supply or more and can be used to identify excess inventory.

Non Plan-O-Gram

This filter dispays products not included on the current Plan-O-Gram (store/sku specific).

Plan-O-Gram

This filter dispays products included on the current Plan-O-Gram (store/sku specific).

Discontinued

This filter displays information for products that have been discontinued (at the store level not necessarily DSC).

Negative Quantity

This filter displays any products that have a negative perpetual inventory quantity.

Written Off

This filter displays vendor direct products that have been written off through the 4-Month write-off process (data is back loaded to Nov. 2014). With this filter, stores can see the Write Off $ column, which shows the dollar amount last written off for the specific product or the Write Off Period column, which shows the period when the last write off occurred for a specific product.

Note: Once the report is generated, the filter builder can be used to sort the information in the report. Users can select the down arrow next to the column and then make the appropriate selections.
