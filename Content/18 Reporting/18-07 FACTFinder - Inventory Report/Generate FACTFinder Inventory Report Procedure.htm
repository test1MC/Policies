Generate FACTFinder Inventory Report Procedure

Step Action

1

From POS desktop, select Browser -> Access Source homepage -> Select FACTFinder from Store Reports-> Reports Tab.

2

Select Inventory.

3

At the store, the Detail defaults to your store number. 

Note: DAD is disabled as an option based on security level.

4

Select the Report Level.

Note: The report defaults to a summary view.

5

Select the appropriate filter.

6

Select Run Report.
