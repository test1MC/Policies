FACTFinder Inventory Report Full Overview

The Inventory Report (Full) provides more detailed information regarding a store’s inventory and all individual products. 

Full Report Column Headings

Description

Unit Cost

This column represents the unit cost/value of each item in perpetual inventory as the end of the previous day.

Note: This should represent the WAC or standard cost for the product.

Est Date

This column represent the date a product was created in the Master Product File.

Last Sold

This column indicates the date the product was last sold.

Note: An asterisk* indicates zero sales in the past five years.

Plan

This column indicates yes (Y) or no (N) if an item is on the current Plan-O-Gram (store/sku specific).

Min/Max

This column indicates the minimum and maximum quantities set in MMS for the specific sku.

Disc Date

This column indicates the date the product was discontinued.

Note: An asterik * indicates the product is not discontinued.

Months Supply

Calculates how many months of inventory the store has on hand based on the prior 12 months of selling activity.

Note: For all new products, this show as null (*).

Note: Every blue, underlined number or word can be clicked on for more specific information and every product has a Notes icon to ask a question or to make a comment. Export the report to a PDF or Excel to print it.
