FACTFinder A/R Drilldown Function Procedure

Journal Entry Detail for an A/R Prime-Sub will allow you to drill into the ARJV detail and view basic customer detail from the Store P&L, P&L, and Balance Sheet reports.

Step Action
1

From POS desktop, select Browser -> Access Source homepage -> Select FACTFinder from Store Reports.

2

Select on the Description hyperlink of the transaction line for which you wish to view customer detail.

3

Select on the Customer account number.

4

Customer Detail will display.
