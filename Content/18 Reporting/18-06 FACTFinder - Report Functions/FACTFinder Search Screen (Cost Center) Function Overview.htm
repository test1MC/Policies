FACTFinder Search Screen (Cost Center) Function Overview

The Search function for Cost Center can be used in several of the FACTFinder reports when either the Cost Center or the correct format to use is not known. 

Step

Action
1

From POS desktop, select Browser -> Access Source homepage -> Select FACTFinder from Store Reports.

2

Select on the Search Function (magnifying glass icon) to the right of the cost center input field.

3

Select your search option:

  • Business Unit/Division
  • Name/Number

4

Did you select Business Unit/Division?

If Then
Yes Proceed to Business Unit/Division Search.
No Proceed to Name/Number Search.

Business Unit/Division Search

1

Select the desired Business Unit.

2

Select the desired Division.

3

Select the cost center you wish to view in the results screen.

4

Select Run Report.

Name/Number Search

1

Select to search by Name or Number.

2

Enter in the known information.

3

Select Get Matches.

Note: If more than twenty-five (25) results are returned, you may wish to expand the numbers of results displayed.

4

Select the cost center you wish to view in the results screen.

5

Select Run Report.
