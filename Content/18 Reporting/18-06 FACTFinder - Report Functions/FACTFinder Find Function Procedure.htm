FACTFinder Find Function Procedure

The Find function can be used in the Balance Sheet, P&L, and Store P&L reports in lieu of drilling down into the report to find a specific Prime-Sub. 

Step

Action
1

From POS desktop, select Browser -> Access Source homepage -> Select FACTFinder from Store Reports.

2

Select the Find function on the toolbar for the report.

3

Enter the Prime-Sub for which you want to search. Select Ok.

4

The report focus will move in on where the Prime-Sub is located and highlight the Prime-Sub.
