FACTFinder Search Screen (Division) Function Overview

The Search function for division can be used in the P&L Report, Balance Sheet, and Fixed Assets reports when either the information or the correct format to use is not known. 

Note: Division searches can only be used by those with access to view division information.

Step

Action
1

From POS desktop, select Browser -> Access Source homepage -> Select FACTFinder from Store Reports.

2

Select on the Search Function (magnifying glass icon) to the right of the division input field.

3

Identify a Business Unit.

4

Select a Division.

4

Select Run Report.
