MicroStrategy Web Procedure

MicroStrategy Web provides a highly interactive and easy to use interface, which allows users to view and analyze key performance data, including sales, gallons, and margins.

Step Action

Access MicroStrategy Web:

1

From POS desktop, select Browser -> Access SOURCE homepage -> Stores Reports -> MicroStrategy Reporting System.

Note: Users can also gain access via SherLink by selecting MicroStrategy Reporting System or via MySherwin -> MY APPS -> MicroStrategy.

2

Select Access MicroStrategy. Enter MySherwin/GEMS username and password to gain access:

Note: MicroStrategy Support contact information is included here.

3

Select the available project(s).

Note: After selecting the available project, the Dashboard landing page will launch.

Navigate MicroStrategy:

1

From the Dashboard – Store landing page, select the buttons on the left to view the available reports in each section:

Part Function

Dashboard

Sales and Call performance for Opportunity, New and Existing Accounts

Note: Store Key Account information coming in 2017.

Overview

Overall Store and District performance

Scorecards

Flash and Month End Store performance with Top/Bottom Accounts, Product Groups, and Market Segment data

Customer

Top Customer performance

Product

Overall Product and Product Grouping performance

2

Select the report name to run the available reports in each section:

Note: Select the printer or excel file icon to run report automatically as a PDF or to export an excel file. When prompted, open with Open Office.

Contact TAG Reporting at stores.reporting@sherwin.com or 216-566-2727 with any questions or concerns.
