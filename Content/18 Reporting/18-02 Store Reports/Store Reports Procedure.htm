Store Reports Procedure

Step

Action

1

From POS desktop, select Browser.

2

Access Source homepage -> If store reports are available to you, Store Reports will appear.

3

Select the Month and Year of the report you wish to review. Select Submit.

4

Select the report you want to review. If necessary, specify the time frame for the selected report.

Note: Only reports available for the selected time frame will display. If no report was generated for that month, it will not appear in the listing. If you want to print the report, select the Shrink oversized pages to paper size option in print settings.
