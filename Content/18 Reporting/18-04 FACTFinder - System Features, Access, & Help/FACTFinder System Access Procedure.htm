FACTFinder System Access Procedure

The following users have access to FACTFinder:

  • Division Presidents
  • Area Vice Presidents
  • District Managers / City Managers
  • Store Managers / Assistant Managers / Operations Managers
  • Store Auditors
  • Financial / Credit Personnel (TAG / Division / District)
