Request Correction on Invoice Procedure

Step

Action

1

From POS desktop, select Browser -> Access Source homepage -> Select FACTFinder from Store Reports -> Reports Tab -> Store Purchases Report -> Select the Notes icon for the line in question.

Note: Be mindful of the Purchase Type: External or Internal.

2

Once the dialogue box appears, select Send Question.

3

Select the appropriate group to contact based on the error type:

Source

PurchaseType

General Mailbox

SWP, External Invoices without a PO#

External

AP Expense

MMS, External Invoices with a PO#

External

AP Merchandise

01, 02, 03, 04, 05, 06, 07, 08, 09, 10, 11, 12, VI

Internal

Store Billing

4

In the text box, ask for the account to be reclassed and provide the following information:

  • Reason
  • New cost center
  • Prime Sub
  • Amount

Note: A picture or document may be included. Select Browse and then select the file to attach it.

5

Select Send Question/Request.
