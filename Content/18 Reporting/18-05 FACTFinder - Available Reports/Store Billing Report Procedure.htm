Store Billing Report Procedure

Information on internal store purchases previously found in the Store Billing Report such as Stock Orders (STAR Orders), Interstore Transfers (ISTs), Store Expenses, and IBARs, can now be viewed in the Stores Purchases Report.

If

Then
Access Store Billing Report

Stock Orders and Store Expenses

Step

Action
1 From POS desktop, select Browser -> Access Source homepage -> Select FACTFinder from Store Reports -> Reports Tab.
2 Select Store Billing from the Report Dropdown -> Select the (Accounting) Period for which you wish to view the report.

3

Select Invoice # Summary from Detail List.

4

Select the Filter button in the Source Column.

5

Check 01-Receipt.

Note: To review Store Expenses, check Store Expense Procedure.

6

Select OK.

Prime-sub Details

Step

Action
1 From POS desktop, select Browser -> Access Source homepage -> Select FACTFinder from Store Reports -> Reports Tab.
2 Select Store Billing from the Report Dropdown -> Select the (Accounting) Period for which you wish to view the report.

3

Select Account Summary from Detail List.

4

Select Run Report.

Note: Drill down to the necessary Prime-sub.

Invoice #, PO #,  and Product Details

Step

Action
1 From POS desktop, select Browser -> Access Source homepage -> Select FACTFinder from Store Reports -> Reports Tab.
2 Select Store Billing from the Report Dropdown -> Select the (Accounting) Period for which you wish to view the report.

3

Select Internal Details from Detail List.

4

Select Product, Invoice, or PO.

5

Select Run Report.

Generate Store Billing Report

1

From POS desktop, select Browser -> Access Source homepage -> Select FACTFinder from Store Reports -> Reports Tab.

2

Select Store Billing from the Report Dropdown.

3

Select the (Accounting) Period for which you wish to view the report.

4

Select the type of Detail you wish to review.

5

Did you select either Product Details or Prime-sub Details as your Detail selection?

If Then
Yes Identify the Product or Prime-sub for which you wish to view information.
No Proceed to Step 6.

6

Enter the Cost Center of the store for which you wish to query or use the search function to search for a store by name.

7

Select Run Report.

Note: The following FACTFinder report functions may be used on the Invoice Search Report: Group (filter) data, Export (pdf or xls), Find, Add Notes, Send Questions.
