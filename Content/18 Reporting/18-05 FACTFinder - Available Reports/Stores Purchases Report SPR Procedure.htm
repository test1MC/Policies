Stores Purchases Report (SPR) Procedure

The Stores Purchases Report lists all merchandise and expense invoices paid by Stores Group Accounts Payable. Receiving against a purchase order on the system is your approval for Headquarters to pay the vendor invoice for merchandise.

Step

Action

1

From POS desktop, select Browser -> Access Source homepage -> Select FACTFinder from Store Reports -> Reports Tab.

Note: FACTFinder can be accessed from mySherwin as well.

3

Select Store Purchases Report.

4

Select Detail type:

5

Select Run Report.

6

Review the report and select the Invoice # to view more information.

Note: While reviewing the report, employees can track their progress by selecting the Reviewed column where a green check will appear.

7

After reviewing the report, select Sign Off at the top of the report.

Note: A text box will appear where a member of store management can sign off. Select Add Sign Off.

8

Type sign off message in the text box and select Add Sign Off -> Select OK.

Note: After rerunning the report, the sign off will be displayed.
