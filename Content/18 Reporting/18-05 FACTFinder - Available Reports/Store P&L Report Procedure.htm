Store P&L Report Procedure

The Store P&L Report allows you to drill into the details of the Stores Profit and Loss Report. The Profit and Loss Report provides a rolling picture of the financial health of your location from month to month and year to year, along with budget and plan information. Focus on reviewing sales, expenses, and inventory margins for accuracy.

Step

Action

Generate Store P & L Report

1

From POS desktop, select Browser -> Access Source homepage -> Select FACTFinder from Store Reports -> Reports Tab.

2

Select Store P&L from the Report Dropdown.

3

Select the (Accounting) Period for which you wish to view the report.

Note: The system will indicate whether or not the accounting period is open or closed. FACTFinder data draws from the Corporate FACTS financial data, so the open status is based on Corporate, which typically closes a day or so after TAG closing.

4

At the store, the Detail defaults to your store number for you. 

Note: DAD is disabled as an option based on security level.

5

Select the Format in which you wish to view the report results.

  • Tree – Summary report with ability to drill down for more detail
  • Grid – Summary report with all detail displayed

6

Select Run Report.

Navigate Report

1

Did you select the Tree View?

If yes, select on the Plus Sign (+) to expand a line for additional information or select on the Minus Sign (-) to return to a higher level view.

2

 

Line and GL Account names are hyperlinks. Select on any of these to view more detail:

  • 2 Year History
  • Journal Entry Detail (available on GL Account line only)

3

Have you selected to view detail from a hyperlink?

If yes, you may drill into the Journal Entry detail for the selected prime-sub. The existing P&L Report will shrink to an upper frame while the new detail report displays in a lower frame. 

Note:  The following FACTFinder report functions may be used on the Store P&L Report: 

  • Tree View (high level): Export (pdf or xls), Tree Node Control (expand all / collapse all), Column (display / hide), Add Notes and Find.
  • Tree View (detail): Export (pdf or xls).
  • Grid View (high level): Export (pdf or xls), Add Notes, Column (display / hide) and Find.
  • Grid View (detail): Export (pdf or xls).
