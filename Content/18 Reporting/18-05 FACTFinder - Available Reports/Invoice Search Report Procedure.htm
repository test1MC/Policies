Invoice Search Report Procedure

The Invoice Search Report presents detail on external store purchases, including invoice details and images.

Step

Action

Generate Invoice Search Report

1

From POS desktop, select Browser -> Access Source homepage -> Select FACTFinder from Store Reports -> Reports Tab.

2

Select Invoice Search from the Report Dropdown.

3

Select the (Accounting) Period for which you wish to view the report.

4

Select your search by criteria:

If

Then

Vendor Number / Name

A search option is available. Be as specific as possible and be sure to select the correct vendor.

Vendor Invoice Number

Enter Invoice Number.

Cost Center

Enter Division and Cost Center. A search option is available for both fields.

5

Did you select Vendor or Vendor Invoice Number as your search criteria?

If Then
Yes Choose to filter by Cost Center.
No Proceed to Step 6.

6

Select Run Report.

View Invoice Details

1

Select on the Invoice Number hyperlink to view invoice details from the report results.

Note: If check data exists for your invoice, check clearing information will be available.

2

Perform additional invoice functions:

If

Then

Save Invoice

Select the File Format desired. Select Save.

View Image

If an image exists for an invoice, select on the hyperlink Display Invoice Image.

Note: The following FACTFinder report functions may be used on the Invoice Search Report: Group (filter) data, Export (pdf or xls), Find, Add Notes, Send Questions.
