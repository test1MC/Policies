Fixed Assets Report Procedure

The Fixed Assets Report lists all fixed assets within the cost center queried and includes details for each asset such as description, service date, original cost, net book value, eFCA#, and service life. Total store valuation at purchase and current book value is also calculated and displayed.

Step Action

1

From POS desktop, select Browser -> Access Source homepage -> Select FACTFinder from Store Reports on the SOURCE Homepage -> Reports Tab.

2

Select Fixed Assets from the Report Dropdown.

3

Enter the Division and Cost Center for which you wish to query or use the search function.

Note: Fixed Asset reports may only be run by Cost Center; the ability to run reports by FCA will be a future enhancement.

4

Select Run Report.

5

Export the file by selecting Export, then select To XLS.

6

Select Open with: LibreOffice, then select OK.

7

Select Edit Document in the upper right-hand corner of the screen.

Note: To retire, enter a R in the Retirement Column. To transfer, enter the Cost Center in the Transfer Column. For partial retirement or transfer, include the quantity.

8

Select File and Save as a Copy.

9

Type in your Store Number as the new name -> Select Save.

10

Send an email to the District Coordinator with the attached file.

Fixed Assets Report

Note: The following FACTFinder report functions may be used on the Fixed Assets Report: Group (filter) data, Export (pdf or xls), Find, Add Notes, Send Questions.
