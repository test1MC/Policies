Count & Record Inventory Procedure

Inventory Check List (Stores) 

Step

Action

Before Counting

1

Merchandise physically received in the store must be received in MMS against the appropriate PO before you begin your counts.

Note: The status box on your POS screen must not have any Purchase Orders in an In-Process status.

2

Review the center section status box under Ecommerce/EDI/COE.

If

Then

Orders appearing as Ready/ Del/Instl that have been delivered

These orders must have a sales transaction processed prior to closing on the day of the inventory.

Orders appearing as Ready/ Del/Instl but have not been delivered.

You must include the product in the physical inventory count prior to closing on day of the inventory.

3

Any unresolved Return Disposition Transactions need to be completed prior to the physical inventory date. Review the status box screen to ensure this is complete.

4

Review to ensure that any Chargebacks in a New status are complete and released.

Note: There should not be any Chargebacks that need to be released.

Clear Inventory

1

Manually clear inventory before beginning the physical counts.

Note: Headquarters will automatically clear the inventory count approximately one week prior to the physical count.

2

From the POS Desktop -> Main Menu -> Merchandise Management -> Enter Employee Number and Password -> Inventory Control -> Inventory -> Clear Inventory.

3

Press ACCEPT [F1] to continue or press any key to exit.

4

Type ‘Y’ when prompted: Are you sure you want to clear inventory?

Note: The system default is set to ‘N.’

Count & Record Inventory

1

Ensure you have completed the steps above and then select inventory method:

If

Then

Using Hand-Held Scanner (PDT)

From Merchandise Management -> Record Inventory.

Note: If the Hand-Held Scanner is inactive for too long, it might lock you out of an inventory location. Before continuing, ensure no one is currently recording inventory on another terminal for the locked location. Then, go to a POS machine and select Help Desk -> Reset Inventory Locations. Carefully select the locked location and press the Clear [F8] key to reset. Press Accept [F1] to save and exit. This process will clear the lock but will not affect your recorded inventory. This should take care of the problem. If not, please contact Store Computer Operations at 216-566-2740.

POS

From the POS Desktop -> Main Menu -> Merchandise Management -> Inventory Control -> Inventory -> Record Inventory.

Note: The prompts for recording inventory are the same whether you are using the Hand-Held Scanner (PDT), or any store computer system terminal.

2

Enter the correct location based on the Store/Branch Layout you created. Your location choices are:

  • Front Room : F1-F8
  • Back Room : B1-B8
  • Other Room/Location : X1-X8

3

Use one the following Inventory recording options:

  • Scan the UPC
  • Manually key the UPC
  • Enter the sales number
  • Enter the product/REX NUMBER and size code

4

Verify that the Product Description and Size Code are correct. Enter the Quantity at the QTY prompt.

Note: The cursor will blink at the last digit of the quantity that was entered.

5

Ensure you have recorded the entire location before proceeding.

6

Change locations by pressing CANCEL [F8] until you are prompted to [ENTER LOCATION].

Note: Pressing the Cancel key will not delete your previous counts.

Correct Error Recording a Count
1 If an error is discovered while recording a count, do not clear inventory.

1

Enter the Location where the recording error occurred.

2

Enter the Sales Number.

3

The following message will appear:

You have already recorded a count of XX in this location (XX). Press ENTER to continue.

4

Press ENTER.

5

Enter the correct Quantity for the product.

Note: If the store receives a STAR Order shipment the day of inventory, follow the normal receiving procedures and do not remove product from the received order unless absolutely necessary. Record inventory from the received shipment in a separate inventory location [ex X8].
