Floorcovering Inventory Policy

This section applies only to stores using the Floorcovering Installation Software

  1. Stores using SOURCE floorcovering software are not required to roll out and measure rolls or cuts for inventory.
  2. Floorcovering inventory is to be complete as follows:
    1. Stores using the SOURCE floorcovering software and roll tags are to use the floorcovering option for inventory in the system and scan their roll tags for each roll in inventory.
    2. Stores not using the SOURCE floorcovering software are required to use write-in pages. The policies in this section do not apply to these locations.
  3. Rolled good inventory that is currently out with an installer cannot be counted - these are orders that you have indicated are Ready for Installation in the Work Order. Tagged rolled inventory in this status will be automatically counted and added into your inventory for you.  You must still count any non-roll inventory out with an installer.
  4. Stores using the SOURCE floorcovering software should be aware that any current rolled good Asset Value Merchandise [AVM] inventory that has been properly entered into the system must be counted during inventory using the AVM Sales Number.  Make sure that you inventory all asset value merchandise that has been entered into your system. If you do not inventory the Asset Value Merchandise during inventory, the AVM inventory information will be removed from the system. Scan the tag barcode and verify the information for all AVM inventory during year-end inventory.
  5. Current Floorcovering includes:
    1. Floorcovering purchased within the past three years from December 31st of the current year.  EXAMPLE: If this is the October 2010 inventory, materialpurchased on or after 1/1/08 is current.
  6. Obsolete Floorcovering includes:
    1. Floorcovering merchandise purchased prior to three years from December 31st of the current year.  EXAMPLE: If this is the October 2010 inventory, material purchased on or before 12/31/07 may be considered obsolete.
    2. Damaged Floorcovering merchandise not in salable condition.
    3. Special order merchandise not returnable to the vendor for credit.
    4. Remnants - There are two distinct types of remnants:
    1. Carpet remnants represent end cuts/leftovers of sold orders and are pieces of carpet that are 90 square feet or less. These remnants are to be inventoried as Obsolete.
    2. Resilient remnants that are less than 36 square feet are to be inventoried as Obsolete.
