Record Rental Inventory Procedure

Step

Action

1

From the POS Desktop -> Main Menu -> Merchandise Management -> Inventory Control -> Inventory -> Record Write-Ins -> Rental Equipment.

2

Press ENTER to record additional rental equipment.

3

Select Rental Equipment Type:

  • Spray Equipment
  • Pressure Washer Equipment

4

The following information is required:

  • Description
  • Serial Number
  • MFG Part Number
  • Original Date of Purchase
  • Original Landed Unit Cost

Note: Make sure original date of purchase and landed unit costs are accurate. The system will calculate the Current Net Value based on this information.
