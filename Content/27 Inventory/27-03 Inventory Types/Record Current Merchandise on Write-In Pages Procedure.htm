Record Current Merchandise on Write-In Pages Procedure

This section also applies to Obsolete and Ready-To-Go Merchandise.

Step

Action

1

From POS Desktop -> Main Menu -> Merchandise Management -> Inventory Control -> Inventory -> Record Write-Ins -> Current Merchandise.

2

Press ENTER to continue.

3

Enter Class Number.

Note: You can only record one of the allowable class numbers. Do not enter a GP with the class number. Sales numbers cannot be used on the write-in pages.

4

Enter Manufacturers Part Number (optional).

Note: If you don’t know the Manufacturers Part Number, leave the field blank.

5

Enter Description.

Note: If you do not enter a description, the description for the class number will be used.

6

Enter Size Code or press LOOKUP [F3] to find the correct size code.

7

Enter Quantity.

8

Enter Landed Unit Cost.

Record Rental Equipment

1

From POS Desktop -> Main Menu -> Merchandise Management -> Inventory Control -> Inventory -> Record Write-Ins -> Rental Equipment.

2

Press ENTER to record additional rental equipment

3

Select Rental Equipment Type:

  • Spray Equipment
  • Pressure Washer Equipment

4

The following information is required:

  • Description
  • Serial Number
  • MFG Number
  • Original Date of Purchase
  • Original Landed Unit Cost

Note: Ensure the original date of purchase and landed unit costs are accurate. The system will calculate your Current Net Value based on this information.
