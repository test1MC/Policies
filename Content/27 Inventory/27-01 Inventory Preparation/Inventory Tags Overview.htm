Inventory Tags Overview

Inventory Tags should be used to manually record inventory that will be entered later, and should be used where inventory control stickers or the product list are not viable methods (i.e., the bar codes or SMIS numbers are not displayed, or products of similar categories or vendors are not located together.)

  • Inventory tags are to be issued in a series of 50. As the tags are issued, the inventory tag control log must be filled, indicating the tags issued, to whom, and the location.
  • Prior planning of locations is advised.
  • Employees are responsible for the tags issued to them.
  • An inventory tag must be completed with the specified information for Manufactured Paint products.
  • Only one REX NUMBER and size code can be recorded per tag. The same REX NUMBER and size code may have two inventory tags if they are in different locations.
  • Attach the tag to the product.
  • After all Manufactured Paint products in a location are counted, all unused and voided tags must be returned to the person maintaining the control log
  • Do not throw away or destroy any tags assigned to you.
  • All tags issued must be returned to the person maintaining the log as either used, unused, or voided.
  • The tags voided or unused will be listed on the control log.
  • Do not return unused tags to Headquarters.
