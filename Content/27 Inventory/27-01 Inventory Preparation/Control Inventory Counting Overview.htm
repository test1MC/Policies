Control Inventory Counting Overview

There are several methods available to control inventory counting. Any or all of these methods can be used. When choosing a method, consider the different types of inventory you have and how well the items are organized, labeled (or bar coded), and stored.

All Sales Number inventory (including book wallcovering awaiting customer pickup) must be counted using one of the following methods:

  • Inventory Control Stickers
  • Printed Product List
  • Inventory Tags

Once the product has been counted it must be recorded by using one of the following items:

  • Handheld Scanner (PDT)
  • Point of Sale Terminal
  • Workstation Terminal

Click here to learn more about recording inventory using the Record Inventory on Hand-held Procedure. See UUInventory TroubleshootingUU for more help.

You have the ability to record inventory on all three devices at the same time. However, you cannot record inventory counts in the same location on more than one device at the same time.
