Mandatory Review

Step

Action

1

Ensure filter is set to “Mandatory Items”, “Store Pending”, and “ALL” and hit the Search button.

2

If items are displayed, you must take action on them.  Select the Update tab to take action. 

Note: Not all stores will have >3K items.

3

Ensure the quantities (current and/or obsolete) and values are accurate.  The “obs qty” is only used if the product is or should be counted as obsolete.  Make changes if necessary.  If no change is required, check the Review Flag button for each SKU.

4

Select the Save Your Updates button at the top left to apply changes and acknowledge review.

Adding Missed Counts

Step

Action

1

Select the Add tab.

2

Enter required info: Sales number, Current or Obsolete Quantity, and Cost. You can add up to 5 at time.

3

Select Submit. A message will pop up indicating if they were added successfully.  If you already counted a SKU, you must locate the SKU and make necessary changes in the Update tab.  (See Updates/Changes).  You must re-submit “adds” if one or more did not add successfully. 

Updates/
Changes – Existing Counts

Step

Action

1

Set filter to “All”, “All”, and “All” and hit the Search button.  If looking for a particular SKU, enter the sales number before hitting search.

2

Update necessary quantities or costs.  Quantities should represent actual on hand during the inventory. “Obs Qty” is for product that should be counted as obsolete.

3

Select the Save Your Updates button at the bottom left to apply changes each time a change is made.

Review All Changes/
Modifications

Step

Action

1

Set the filter to “All”, “Store Pending”, and “MODIFIED” and hit the Search button.  Ensure the SKU field is blank so all modifications are returned.

2

To view an audit trail of changes for a particular SKU, click on the sales number while in the view tab.

3

After validating all of your modifications, select Click to Complete Review to submit changes.  If the Click to Complete button is not active, make sure you reviewed and checked off all >3K items.

Note: If adjustments need to be made to the quantity/ unity cost of a SKU or a product was counted as current when it was obsolete, go to the Update view and locate the product by searching by sales number in the search tool.  Be sure the views are set to “All” in the drop-down-boxes at the top. In the update screen, you can change the quantity or unit cost. Or, change the current quantity and enter the quantity in the “Obs Qty” (Obsolete Quantity) column.  Select Save Your Updates after you make your change.

Note: You must also perform a perpetual inventory adjustment in your POS for any quantity adjustments made in SWeep. Ensure you enter “Y” in the DM Adjustment box.
