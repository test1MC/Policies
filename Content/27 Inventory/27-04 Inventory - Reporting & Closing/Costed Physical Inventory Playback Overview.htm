Costed Physical Inventory Playback Overview

This report will show you a playback of your "costed inventory" - Store Merchandise Inventory and is sorted by department. Within the department it is sorted by vendor and then part number.

Note: Pages run sequentially for departments 11, 18 and 21 and then start over for the remaining departments. Refer to the example of this report for an explanation of the fields. The key fields to be reviewed for accuracy are:

  • Sales Number
  • Units
  • Unit Cost

If a sales number was entered at inventory that was invalid on the Headquarters product database on the physical inventory date, it will appear as all ones (1111-11111) at the beginning of each category on the Costed Inventory Playback. Since it is invalid, it is “costed” at a penny. The unit counts came from your physical inventory counts on the physical inventory date. The unit cost was determined based on FIFO costing.

If the [sales number], [units] or [unit cost] are incorrect as reported, the page/line number assigned to the sales number is required for any corrections.

Part

Function

Sales Number

Nine-digit number identifying a product.

PR TP

This number corresponds to the last two numbers in the P & L Analysis report prime-sub account numbers identifying the product type.

DP

Department the sales number is assigned to.

Part No

This is the manufacturer’s part number. On paint the last two digits are the size code.

Manufacturers Name

Manufacturer associated to the product.

Description

Product description.

Size

Size description.

Units

Quantity counted on the physical inventory date.

Unit Cost

The unit cost used to value your inventory.

Page Line

Required to make adjustments to "costed inventory".

Gross Value

This is the extended value of the inventory.

Obsolete Value

The extended value of the number of units for a sales number that were flagged as obsolete.
