Review Internal Closing Reports Procedure

Step

Action

1

From SOURCE homepage -> Operations -> Processes -> SWeep ->  Inventory Adjustments -> Review Closing Reports.

2

Select the appropriate report:

If

Then

Billed Not Received Internal (“A1”)

Select Billed Not Received - Internal and proceed to Step 3.

Note: Closing Report A2: Billed Not Received ISR is Add Only and not generated automatically. Refer to the Add OnlyClosing Reports procedure below, if applicable.

Received Not Billed – Internal (“C1”)

Select Received Not Billed - Internal from DSC and proceed to Step 3.

Note: Closing Report C2: Received Not Billed ISR is Add Only and not generated automatically. Refer to the Add Only Closing Reports procedure below, if applicable.

Uncleared IBARs (“D1”)

Select Uncleared IBARs and proceed to Step 3.

Note: Closing Report D2: ISTs and Store Expenses is Add Only and not generated automatically. Refer to the Add Only Closing Reports procedure below, if applicable.

Note: Hover over icons to display hints or additional information.

3

Click on the Purchase Order or IBAR.

4

Review the Purchase Order or IBAR for accuracy:

If

Then

An item is inaccurate or needs to be adjusted

Select the item and make the appropriate changes in the editable fields. Click the check mark to indicate the line as reviewed.

Note:  To view more information about specific product lines, click on the line to view its details.

An item has an error and it should not be used

Click on the circle with a slash through it on the far right-hand side to indicate the closing report should not be used.

 

 

All the items are accurate

Stores have the option to individually select each item or to bulk review all items on the PO or IBAR.

Note: Stores also have the option to select “do not use” for any or all of the items.

Note: If necessary, click Add Product to include any additional items not on the report. Stores have the option to search for existing products on the Purchase Order or to include a product not already on the PO.

5

After reviewing the Closing Report, click Add Purchase Order or Add IBAR to search for already existing POs or IBARs if you have not found the PO or IBAR needed.

6

Click Save Report.

7

Check the Review Status to confirm that all the necessary information has been reviewed.

Note: The Review Status will only turn green after all lines are reviewed and saved for that particular PO. All POs and IBARs must have the green indicator before the Approve Report button is enabled.

8

Click Approve Report.

Note: This must be clicked in order to complete the review.
