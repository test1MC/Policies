Review Add Only Closing Reports Procedure

Step

Action

1

From SOURCE homepage -> Operations -> Processes -> SWeep -> Inventory Adjustments -> Review Closing Reports.

2

Select the appropriate report:

If

Then

Billed Not Received ISR (“A2”)

Select Billed Not Received ISR and proceed to Step 3.

Received Not Billed ISR (“C2”)

Select Received Not Billed ISR and proceed to Step 3.

Uncleared ISTs and Store Expenses (“D2”)

Select Uncleared ISTs and Store Expenses and proceed to Step 3.

Uncleared Chargebacks (“E”)

Select Uncleared Chargeback and proceed to Step 3.

Billed Not Received – External (“F”)

Select Billed Not Received - External and proceed to Step 3.

Product Recodings (“R”)

Select Product Recodings and proceed to Step 3.

3

Click Add a Purchase Order or Add Chargeback.

Note: For Product Recodings, select Add Purchase Order for Recoding.

4

Enter the last few characters of the P.O. Number.

5

Review the Purchase Order for accuracy:

If

Then

An item is inaccurate

Select the item and make the appropriate changes in the editable fields. Click the check mark to indicate the line as reviewed.

Note:  To view more information about specific product lines, click on the line to view its details.

An item has an error and should not be used

Click on the circle with a slash through it on the far right-hand side to indicate the closing report should not be used.

All the items on the Purchase Order are accurate

Stores have the option to individually select each item or to bulk review all items on the PO.

Note: Stores also have the option to select “do not use” for any or all of the items.

Note: Click Add Product to include items on the report. Stores have the option to search for existing products or include a product not already on the PO.

6

Click Save Report.

7

Check the Review Status to confirm that all the necessary information has been reviewed.

Note: The Review Status will only turn green after all lines are reviewed and saved for that particular PO. All POs and IBARs must have the green indicator before the Approve Report button is enabled.

8

Click Approve Report.

Note: This must be clicked in order to complete the review.
