Review External Closing Reports Procedure

Step

Action

1

From SOURCE homepage -> Operations -> Processes -> SWeep -> Inventory Adjustments -> Review Closing Reports.

2

Select the appropriate report:

If

Then

Received Not Billed – External (“B”)

Select Received Not Billed – External and proceed to Step 3.

Received Not Billed – External Other (“B Other”)

Select Received Not Billed – External Other and proceed to Step 3.

Uncleared Chargebacks (“E”)

This report is Add Only and not generated automatically. Refer to the Add Only Closing Reports procedure below, if applicable.

Billed Not Received – External (“F”)

This report is Add Only and not generated automatically.  Refer to the Add Only Closing Reports procedure below, if applicable.

Product Recodings (“R”)

This report is Add Only and not generated automatically. Refer to the Add Only Closing Reports procedure below, if applicable.

Note: Hover over icons to display hints or additional information.

3

Click on the Purchase Order.

4

Review the Purchase Order for accuracy:

If

Then

An item is inaccurate

Select the item and make the appropriate changes in the editable fields. Click the check mark to indicate the line as reviewed.

Note:  To view more information about specific product lines, click on the line to view its details.

The following information will appear on the side panel.

Note: The report amount is the Closing Report amount.

An item has an error and should not be used

Click on the circle with a slash through it on the far right-hand side to indicate the closing report should not be used.

All the items on the Purchase Order are accurate

Stores have the option to individually select each item or to bulk review all items on the PO.

Note: Stores also have the option to select “do not use” for any or all of the items.

Note: Click Add Product to include new items on the report. Stores have the option to search for existing products or include a product not already on the PO.

6

After reviewing the Closing Report, click Add Purchase Order. If you have not found the Purchase Order needed, the screen below will appear and you can search for other existing Purchases Orders.

7

Click Save Report.

 

8

Check the Review Status to confirm that all the necessary information has been reviewed.

Note: The Review Status will only turn green after all lines are reviewed and saved for that particular PO. All POs and CBs must have the green indicator before the Approve Report button is enabled.

9

Click Approve Report.

Note: This must be clicked in order to complete the review.
