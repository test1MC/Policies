Promotional Package Procedure

This section does not apply to Automotive Branches

Step

Action

1

Upon receipt, open the POP Package to ensure the kit has all of the components listed.  If something is missing, contact the vendor listed at the bottom of the instruction sheet.

2

Set the package aside in a designated area until the day prior to the start of a sale

3

Read the instructions included with each kit prior to installing the materials. The elements in the package may change for different promotions.

4

Remove all expired sale tags/signs from the sales floor and discarded. 

5

Install all new POP the evening prior to the start of a sale.

6

When the sale is over, be sure to remove all signs and sale tags
