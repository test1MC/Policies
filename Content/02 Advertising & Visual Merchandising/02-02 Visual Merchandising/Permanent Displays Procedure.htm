Permanent Displays Procedure

This section does not apply to Automotive Branches

Step

Action

1

Upon receipt of new displays, check for shipping damage prior to signing for receipt. Contact the Visual Merchandising Department at 216-566-2342 for replacement if necessary.

Note: For more information see Vendor Loss or Damage Claim Policy.

2

Review the instruction sheet from the carton.

3

Read and carefully check contents list against the actual contents. If any components are missing, contact  the vendor listed on the instruction sheet.

4

Assemble and install display using instructions on the carton and direction from management.

5

From replacing parts to permanent displays, check the First Impressions In-Store Merchandising Catalog on Source: POS desktop -> Browser -> Source homepage -> Advertising -> In Store Merchandising for items you need.

For additional assistance, refer to Quick Reference Guide
