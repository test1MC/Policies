Plan-o-Gram Procedure

This section does not apply to Automotive Branches

Step

Action

1

Review new Plan-o-Grams with all store personnel, noting the addition of new products and the deletion of any existing products.

2

Review the procedure The FloorPlan and Plan-o-Gram in SWeep: POS desktop -> Browser -> Source homepage -> Information Library -> Plan-o-Gram Instructions -> Set each section according to the Plan-o-Gram.

3

If any updates to POP, fixtures, or supplies are needed, refer to the First Impressions In Store Merchandising catalog: POS desktop -> Browser -> Source homepage -> Advertising -> In Store Advertising
