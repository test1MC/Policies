Preferred Customer Redemption Procedure

The Preferred Customer Program is only available to TAG locations in the US and Canada.

It is critical to run every sale transaction using the customer’s account:

  • Base preferred customer discounts are granted based on the account CAC Code.
  • History is tracked by account.

To avoid creation of duplicate accounts, complete both steps when processing sales for preferred customers:

Step Action

1

When prompted to enter the Account Number, press LOOKUP [F3] and search for the customer by name. 

2

Select the customer from the query results.

Note: If the customer’s account is not found, press LOOKUP [F3] again to search the HQ database. You will be prompted to enter additional parameters to search the HQ database, including: home store, phone number, and/or zip code. If no customer account can be found, follow the Preferred Customer Program Enrollment Overview
