Vendor Merchandise Return Overview

This section applies only to External Vendors.

A return to vendor involves the return of merchandise to a supplier other than a Sherwin-Williams Distribution Service Center. Merchandise may be returned to a vendor as a result of:

  • damage
  • overstock
  • customer dissatisfaction
  • shipping errors
  • defective
