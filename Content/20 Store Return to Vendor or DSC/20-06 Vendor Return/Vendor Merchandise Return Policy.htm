Vendor Merchandise Return Policy

This section applies only to External Vendors.

  1. The vendor must be contacted and an authorization obtained prior to the merchandise being returned to the vendor. All allowable returns must be returned to the vendor promptly.
  2. Chargebacks for quantity and pricing errors are automatically processed by Headquarters based on the comparison between the Purchase Order, receiving record, and invoice.
  3. Chargebacks must be accurately completed on the store computer system before they will be processed by Headquarters Accounts Payable. Retain a copy of the chargeback with the call tag and return information.
  4. Chargebacks must be released in order to be processed.
    1. Running Chargebacks are allowed, but must be released at least once a monthly.
  5. Defective merchandise must be packaged properly when returned to the vendor.
    For more information see Vendor Defective Merchandise Policy
  6. The appropriate return or handling fee must be added if the return involves a customer’s special order merchandise. A 25% restocking fee is to be charged to the customer for special order returned wallcovering.

Note: The customer is charged a 25% restocking fee and Sherwin-Williams is charged a 20% restocking fee by the wallcovering vendor. When writing the Chargeback, deduct 20%, if applicable.
