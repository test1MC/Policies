Vendor Special Order Return Procedure

This section applies only to External Vendors.

Special order merchandise is typically not returnable because it was made specifically for the customer.

Step Action

1

Contact the vendor to see if the material could be returned.

If Then

The vendor will accept the merchandise.
  • Please see Vendor Merchandise Return Procedure

The vendor will not accept the merchandise.
  • Set the merchandise out on the sales floor after marking the merchandise for quick sale.
  • Do not enter a store expense.
  • Label the merchandise with the appropriate sales number and retail selling price. Include the "uick sale selling price.
  • When the merchandise is sold, enter the retail selling price. Press the DISC [F11] key to dollar discount.

2

Nonreturnable material may be in the store during inventory.

    If
    Then

The merchandise is on the sales floor during an inventory.
  • First time the item is inventoried - full value (only if returnable merchandise).

The merchandise is not returnable (ex. incorrectly measured blinds).
  • The merchandise is to be inventoried as obsolete on the first inventory.
  • Second time inventoried - write off as obsolete.

For more information see Vendor Merchandise Return Overview
