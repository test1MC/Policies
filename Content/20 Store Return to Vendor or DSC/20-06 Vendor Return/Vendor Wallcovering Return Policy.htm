Vendor Wallcovering Return Policy

This section applies only to External Vendors.

  1. A Chargeback must be prepared once a month for wallcovering returns to each participating vendor. The completed Chargeback must include the total cost of the returned merchandise less the return freight quoted by the vendor.
  2. All allowable returns must be returned to the vendor within 60 days from the date the merchandise is received. The customer is required to pay for return freight to the vendor. Use the Wallcovering Division Freight Chart to calculate the appropriate charges.
    For more information see Wallcovering Freight.
  3. Store employees are encouraged to make one return shipment per vendor before the 25th of each month. The store employee must prepare the merchandise for return shipment and await receipt of either an ARS return label or a Call Tag pick-up.
  4. The vendor will issue a full merchandise credit after receiving the returned products.
