Vendor Wallcovering Return Procedure

This section applies only to External Vendors.

Step Action
Return of Customer Merchandise

1

Inform the customer that they are required to pay the return freight charges. They are not required to pay the restocking fee.

Note: The standard restocking fee may be less than the return freight charge for small quantity returns.

2

Process a CORRECTION transaction for the returned wallcovering.

3

On the negative side of the correction transaction, enter the original Book/Pattern Number, Quantity Returned, and the Selling Price. If necessary, press the DISCOUNT [F11] key, select [Other]. Enter [WC SALE-RETURN] as the reason for the discount.

4

On the positive side of the correction transaction, enter the freight class number (774000). Refer to the Wallcovering Division Freight Chart to find the proper freight charges to be charged to the customer.
Return Merchandise to Vendor

1

Issue the customer a credit for the full amount of the returned merchandise minus the return shipping and handling charge by processing a Correction Transaction. Use the Division Shipping and Handling Chart to calculate the appropriate return charge.

2

Include multiple returns on one Chargeback per month to each vendor. Make sure that you include:

  • Original purchase order information
  • Name of the person who authorized the return
  • ARS (Authorize Return Shipment) label or Call Tag.

3

Use a rubber band to attach a copy of the Chargeback to the wallcovering return.

4

Segregate all returned wallcovering by vendor. Before the 25th of each month, pack the returned merchandise for UPS pickup. Combine each vendor’s wallcovering returns in one (1) package for each vendor.

5

Please be careful to return the wallcovering to the correct vendor. Select vendors will allow multiple returns to be handled under one call tag or ARS in order to minimize the cost of returned freight to Sherwin-Williams. Please process all other vendor returns as soon as possible. Only wallcovering returns for the Vendors listed should be held at the store and returned once a month.
