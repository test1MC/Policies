Vendor Special Order Return Overview

This section applies only to External Vendors.

A special order is an order placed by a customer for merchandise not regularly stocked in the store. In most cases, a customer is charged freight (cutting charges, if applicable) for these items. The Point of Sale deposit procedure is used for handling deposits made on Special Order for Wallcovering, Floorcovering, Window Treatments, etc. For more information, see Vendor Direct Shipments Policy.
