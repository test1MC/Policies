Drum Return & Disposal Procedure

Some Sherwin-Williams' stores buy drums filled with lacquer thinner or other solvents. These drums can be re-used or returned to the vendor after they are emptied.

If

Then

Vendors Require Deposit

Drums should be returned to the vendor after they are emptied. Use 734-1274 to capture the drum deposit on the P.O.

Customer Drum Deposit

Add the Drum Deposit to the transaction using the following sales numbers:

  • 734-1274 - Drum Deposit
  • 421-6602 - Drum / EPA Charge

Return Empty

Complete a Chargeback form to the vendor using the above sales numbers. List the number of drums being returned and the amount of deposit paid for each drum.

Disposing of Drums
  • Locate a local buyer of used salvageable drums and record the name and address of the buyer while processing the Miscellaneous Cash Receipt transaction.
  • Add "sale of X drums at $ per drum" in the Comments section while running the cash receipt.
  • Process a Miscellaneous Cash Receipt transaction coded to 7599-099. Attach a note to a copy of the transaction stating that you want to have the transaction recoded to 0406-802.
  • Mail the transaction copy and note to your Division Controller.
