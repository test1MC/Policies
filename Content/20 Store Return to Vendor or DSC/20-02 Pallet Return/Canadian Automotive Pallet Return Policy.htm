Canadian Automotive Pallet Return Policy

Canadian Automotive Branches ONLY.

  1. All pallets used in the shipment of GOM and AP merchandise from the DSC or plant are the property and asset of The Sherwin-Williams Company.
  2. Automotive Division operates an approved pallet recycling program and the Canadian Hub Branches should arrange for pallet returns to the Pallet Return Site.
  3. Empty pallets should not be allowed to accumulate or be disposed of locally in any way.
  4. Empty pallets should be returned in a minimum quantity of 15 pallets and a maximum quantity of 45 pallets per pallet return to the Hub Branch locations. Pallets must be stretch wrapped or banded for return shipment.
  5. Canadian Branches will IST empty pallets to Hub Branches for consolidated returns to the Pallet Recycling Center.
  6. The Hub Branch locations will return empty pallets to the Pallet Recycle Center via an IBAR.
  7. If you have any questions or problems regarding pallet returns, please contact the DSC Freight Coordinator at (859) 623-3200 x40.
