Step

Action

IBAR Processing

1

From POS desktop, select Store -> Merchandise Management -> IBAR -> Add New IBAR -> Returns.

2

Select Stop Sale – To DSC as the Reason Code.

3

Enter the IBAR using the following information:

  • Return Authorization Number: PALLETS99
  • Stop Sale Number: PALLETS99
  • DSC Number: 492
  • Destination: Richmond DSC
  • SCAC: Press CLEAR [F8] to remove Conway and use LOOKUP [F3] to search for and select the carrier shipping your pallets to the recycling center.
  • Division Number: D24
  • Cost Center: 9900
  • G/L Account: 7599-102

Note: Only include pallets when processing this IBAR. Use Sales Number 9999-11035 and enter the Quantity of pallets being returned.

7

Release the IBAR.

8

Prepare a Manual Bill-of-Lading Automotive U.S. (located in the Forms Repository).

9

An e-trac will be created once the IBAR is released. Place the store copy of the IBAR in the Awaiting SPR folder. Retain the driver-signed store copy of the Bill-of-Lading.

Packaging & Shipping

1

Pallets are to be returned with the comment, “Free Return Requested,” via the carrier that delivers the Branch’s weekly STAR Order.

2

Contact your delivering carrier terminal and arrange for a Free Pallet Return pick up and let the carrier know how many pallets you have to be returned and when the pallet return will be available for pickup.

3

Only SW 30”x42” pallets are to be returned. Accumulate a minimum of 15 pallets and a maximum of 45 pallets for each return.

4

Pallets must be stretch wrapped or banded for return shipment with a shipping address label.

5

Pallets are to be returned to the following Pallet Recycle Centers as noted below (with the exception of FedEx Freight, SAIA, Dayton, and Estes as noted).

Process Manual Bill of Lading

1

The Manual Bill-of-Lading for the pallet return should be completed as follows:

  • Invoice/Transaction Number: “IBAR9……..”
  • To: Consignee: Pallet return site as noted above.
  • From: Shipper: Sherwin-Williams #9…., Street, City, State and Zip Code
  • BOL Bill-to location: "Sherwin-Williams c/o DDI Inc. PO Box 818019 Cleveland OH 44181-8019"
  • BOL Ship-to location:  The closest pallet depot to your location.
  • Use the shipping description line for pallets (BOL: 2848) and enter the number of pallets in the No Pkgs column and the total weight (no. of pallets x 35lbs) in the Weight column.
  • Enter the totals in the last shipping description line for Total Number of Packages and Total Weight.
  • Return amount: 15 - 20 pallets, one stack, shrink wrapped.
  • For SEFL and AACT, designate freight charges as Prepaid. Otherwise, DO NOT select either Prepaid or Collect in the COD or the Freight Charges sections of the Bill-of-Lading. Leave all terms blank.
  • Contact delivering terminal to arrange pickup.
  • Special Instructions: Note carrier specific instructions as listed below.
