ISR & IST for Pallet Return Procedure

Step

Action

ISR Processing (HUB Branch Locations Only)

1

From POS desktop, select Store -> Merchandise Managment -> Purchase Order -> Add New Purchase Order -> Inter Store Request (ISR).

Note: Do NOT use the Add Stock Transfer Order Process in Customer Order Entry. This option is only for obtaining product for the fulfillment of customer orders. For more information on creating ISR Purchase Orders in MMS, please see Internal Purchase Orders Overview.

2

Include only pallets when processing this ISR. Use Sales Number 9999-11035 and enter the quantity of pallets being returned.

3

Fax a paper copy of the ISR Purchase Order to the shipping branch so that they may complete the IST.

4

Once the pallets have physically arrived at your location, receive against the ISR Purchase Order.

ISR Processing (Shipping Branch Locations Only)

1

Upon receipt of the faxed ISR Purchase Order, review the ISR Purchase Order -> Complete your IST for the pallet return. 

For more information on processing an IST, please see Create IST Procedure.

2

Create a Manual Bill-of-Lading for the shipment.

Packaging & Shipping (Shipping Branch to Hub Branch)

1

Pallets are to be shipped freight prepaid on a Manual Bill-of-Lading Automotive (English) or Manual Bill-of-Lading Automotive (French) to the Hub Branch locations via the carriers noted below:

Western Region Branches
  • Canadian Freightways

2

Only SW 30”x42” pallets are to be returned. Accumulate a minimum of 15 pallets and a maximum of 45 pallets for each return to the Hub Branch locations.

3

Pallets must be stretch wrapped or banded for return shipment with a shipping address label to the Hub Branch locations.

4

The Manual Bill-of-Lading (English) or Manual Bill-of-Lading (French) for the pallet return should be completed as follows:

  • Invoice/Transaction Number: “IST……..”
  • To: Consignee: Hub Branch location as noted above.
  • From: Shipper: Sherwin-Williams #9…., Street, City, State and Zip Code
  • BOL Bill-to location: "Sherwin-Williams c/o DDI Inc. PO Box 818019 Cleveland OH 44181-8019"
  • BOL Ship-to location:  The closest pallet depot to your location.
  • Use the shipping description line for pallets (BOL: 2848) and enter the number of pallets in the No Pkgs column and the total weight (no. of pallets x 35lbs) in the Weight column.
  • Enter the totals in the last shipping description line for Total Number of Packages and Total Weight.
  • Return amount: 15 - 20 pallets, one stack, shrink wrapped.
  • BOL Freight Terms:  "Prepaid or PPD"
  • Contact delivering terminal to arrange pickup.

IBAR Processing (HUB Branch Locations Only to Pallet Recycle Centre)

1

From POS desktop, select Store -> Merchandise Management -> IBAR -> Add New IBAR -> Returns.

2

Select Reason Code: Stop Sale – To DSC.

3

Enter the IBAR using the following information:

  • Return Authorization Number: PALLETS99
  • Stop Sale Number: PALLETS99
  • SCAC: Press DELETE [F8] to remove Conway and use LOOKUP [F3] to search for and select the carrier shipping your pallets to the recycling center.
  • DSC Number: 492
  • Destination: Richmond DSC
  • Division Number: D24
  • Cost Center: 9900
  • G/L Account: 7599-102

6

Include only pallets when processing this IBAR. Use Sales Number 9999-11035 and enter the quantity of pallets being returned.

7

Release IBAR.

8

Prepare a Manual Bill-of-Lading.

9

Once the IBAR is released, an e-trac will be created. Place the store copy of the IBAR in the Awaiting SPR folder. Retain the driver-signed store copy of the Bill-of-Lading.
