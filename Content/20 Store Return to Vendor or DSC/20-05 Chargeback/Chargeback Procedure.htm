Chargeback Procedure

Step Action

1

Obtain the vendor's approval to return the merchandise. Note the name of the person giving the approval. Request a call tag number from the vendor for wallcovering and other returns.

2

From POS desktop, select Store -> Merchandise Management.

3

Enter your Employee Number and Password.

4

Select Chargebacks from the Merchandise Management Menu.

5

Select Add New Chargeback.

6

Select the Type of Chargeback:

  • Receive credit from a vendor
  • Give credit to a vendor

7

Press LOOKUP [F3] to search for the vendor. 

Note:  If you selected receive credit from a vendor, you will have the option to create a spray equipment warranty claim.

8

Enter the Sales Number for the returned merchandise.

9

Enter the original Purchase Order Number or press LOOKUP [F3] to search for the PO number.

Note: If there is no purchase order associated with the chargeback line, press ENTER.

10

Press the COMMENT [F9] key to include any specifics about the merchandise.

11

Enter a Chargeback Reason.

12

Are there additional freight or labor charges?

If Then
Yes Press the ENTER key to access the Non Merchandise Codes.
No Proceed to Step 13.

13

Is there an additional re-stocking fee?

If Then
Yes Use the following SKU: 559-922.
No

Press the DISC [F11] key to deduct any restocking fee. Press ACCEPT [F1] to continue.

14

Enter the following information:

  • Vendor contract
  • Vendor return authorization code
  • Vendor authorization date
  • Bill of lading number
  • Carrier
  • Ship date

15

Include the printed copy of the chargeback with the returned vendor merchandise. Affix the call tag to a copy of the chargeback to be retained at the store along with the return information.

16

Multiple returns may be entered on the same chargeback throughout the month for the same vendor.

Note: Chargebacks must be released at least once per month.
