Wet Sample Shipments Procedure

This section applies to Automotive Finishes Only.

Step

Action

Bill-of-Lading

1

Is the product in POS?

If Then
Yes

From POS desktop, select Store -> Customer Order Entry -> Delivery Menu -> Open BOLDOC -> Generate a BOL electronically through HEARS.

No

Create a Manual BOL found in the Forms Repository on SOURCE.

2

Confirm BOL for accuracy:

  • Shipper/Consignee
  • Confirm number of units
  • Confirm the weight
  • Sign the document
  • Ensure the driver signs the BOL

Note: For help, please see 11-03 Bill of Lading Application.

Assemble Package

1

Is this wet paint sample in a glass vile?

If Then
Yes

Wrap the sample in an absorbent packing material. Put the sample and packaging into a quart or pint can, whichever is size appropriate.
No

Put the sample into a metal quart or pint can.

2

Properly prepare and close the package by sealing the can lid with clips or a ring.

 

Package Marking

1

Fill out an HMIS Label and adhere to the can.

Note: Mark the package with the appropriate hazard label.

2

Add a label with the product part number to the can.

3

Pack the can into an appropriately sized carton for shipping.

4

Mark the package with Limited Quantity (LQ) exception marking, if applicable.

 

Note: LQ label must be 4” per side if the carton size permits, otherwise it can be 2”.

5

The Orientation Arrows Symbol must appear:

  • Twice on opposing sides of the carton
  • Exactly as pictured

Note: Single arrow or double arrows without the horizontal line under them may not be used.

Seal Carton

1

Close the packaging according to manufacturer instructions.
