Recall Previously Tinted Order Procedure

This section does not apply to Automotive Branches or General Polymers

Press Recall [F10] to modify a previously tinted pending order and adjust a color for a customer before the sales transaction is processed.

Step

Action

1

Press Recall [F10] and the most recent order will display at the top of the list. Locate the correct order and press Enter.

2

Select the correct line and press Color [F6].

3

Select Correct to adjust a tinted container before processing the Point-of-Sale transaction.

Note: The previous formula displays on the screen and you are able to add colorant manually or by a percent.

4

Select Clear to return to the previous screen.

Note: Tag the container as a Mistint so that the manager will be able to identify the Mistint while processing a Store Expense transaction. For more information, see 09-07 Mistints.

Note: If the customer brings back a gallon of paint and would like you to adjust the color, users can recall previous order information after being sold.
