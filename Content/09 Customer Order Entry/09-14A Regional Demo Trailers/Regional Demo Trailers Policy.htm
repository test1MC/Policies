Regional Demo Trailers Policy

  1. A specific Demo Trailer cost center must be established by PCG Financial Accounting. (No Demo Trailer inventory or expense can be charged to a Region cost center).

  2. Regional Demo Trailers must remain fully stocked and maintained properly on a regular basis (such as running the trailer in order to turn the product daily to prevent it from going bad). Failure to do so could result in branches incurring a larger than normal expense all at once.
  3. An eExpense request must be submitted through eAccounting in order to establish the initial inventory that will be located in the trailer at any given time.
  4. A subsequent eExpense request should be submitted if any changes (>$500) are made to the trailer’s inventory. For example, if a new product is being introduced or the amount of inventory is increasing/decreasing, a new eExpense should be submitted for approval.
  5. If existing materials’ quantities are increasing or new materials are added to the trailer’s inventory and an eExpense has been fully approved, the branch that is supplying the merchandise is responsible for the following actions:
    1. Notifying PCG Financial Accounting that an expense to prime sub 7599211 has been processed for the new material
    2. Faxing the expense transaction to (216)586-1800
    3. Writing the Regional Cost Center to expense at the top of the store expense transaction before faxing.  Accounting will manually move the expense to the appropriate cost center. This is the only time the regional demo trailer cost center will incur an expense for material
  6. Spray guns and equipment are not to be expensed to the regional demo trailer cost center. If a new spray gun/equipment is required, the expense will be incurred by the Tech Rep using the gun/equipment. It will then be the responsibility of the Tech Rep to properly care for this spray gun/equipment.
