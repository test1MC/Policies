Update Quick Order Procedure

The Quick Order will not price the order or allow you to commit inventory. All information can be updated later when you have more time. Once you complete the QUICK ORDER, press Accept [F1].

Step

Action

1

From the POS desktop, select Store -> Customer Order Entry - > Enter Employee Number and Password ->Update/Review Orders.

2

Select Yes to Review All Quick Orders. Now you can see Purchase Order Number, Account Number, Job Number, Sales Number, Status, Product Description and Quantity.

Note: The Order Entry Lookup screen now displays more information.

3

Select the correct Quick Order and modify it to include the customer Purchase Order Number, Installation Information, and any other required information to complete the order.

Note: The updated Customer Order will retain the original QUICK ORDER number. This will enable you to review the order origination. The Quick Order application can remain open all day to allow “quick” order processing. If you choose to leave Quick Order open all day remember to CANCEL out of Quick Order before performing the Close Transaction and End of Day.
