Configure Color Station Procedure

This section does not apply to Automotive Branches or General Polymers

The store employee must configure each Source workstation specifically for the attached color equipment to successfully implement Sher-Color. Color equipment includes the Macbeth XTS Sher-Color Color eyes, Tinters, or Monarch label printers.

All Source workstations initially default to a “standalone” configuration that has Quick Order and Color Lookup functionality only. If your store has no color equipment, do not continue this configuration procedure.

Note: Color Station Information is used to assist Store Computer Operations with troubleshooting.

Step

Action

1

Locate the models and serial numbers for all color equipment attached to a Source workstation and record them BEFORE executing the configuration procedure. Check the back of the Tinter and Color Eye and the bottom of the Monarch Label Printer for identification stickers or plates.

2

Install the appropriate cabling for your color device. If you are converting your existing color equipment this cabling is already in place. If you are adding a new Tinter, color eye, or label printer follow the installation instructions provided with the equipment.

3

Exit all color applications and Quick Order on any Source workstation.

Note: For your initial Sher-Color installation, configure all workstations with Tinters and/or label printers first and your color eye second. All non-color Source workstations should be configured last.

4

Repeat this process for each Source workstation in the store.

 

From the POS desktop, select Store -> Help Desk -> Configure Color Station.

Note: When Sher-Color is installed the store employees will need to configure each color station.

5

Enter your employee number at the prompt in the Employee Number field and press enter to move to the next field.

Note: The store number, station number, “MAC” address, and PC information are automatically retrieved from your current Source configuration. Sher-Color provides a default workstation number.

6

Enter a Tinter choice to install a Tinter on Sher-Color:

Step

Action

1

Choose the specific tinter model operated in your store and press Enter to select your choice and move to the next field.

Note: If a tinter is already attached to the workstation the model may be pre-selected in the list.

2

Enter the tinter serial number accurately. Press Enter.

Note: If no serial number plate or tag is found please enter ‘99999’.

Note: If this workstation is not a Tinter workstation, use the default “standalone” choice; press Enter to move to the next field.

 

If this workstation IS a color matching workstation, press the DOWN ARROW key to get a drop down list of all the color eye models:

Step

Action

1

Choose the specific color eye model that this workstation operates and press Enter to move to the next field.

Note: The XTS is currently the only valid Sher-Color color eye.

2

Enter the color eye serial number and press enter to move to the next field. The unique part of the serial number will be electronically validated to the unit.

Note: If you cannot complete successful serial number entry in this field you should call the Stores Computer Operations at 216-566-2740.

Note: It is highly recommended that all stores have a Monarch label printer installed on SOURCE for label printing. If you do not have a Monarch label printer, call the Stores Computer Operations at 216-566-2740 for directions on procuring a label printer for your store.
