Color Training Overview

This section does not apply to Automotive Branches or General Polymers

Each employee should be given the opportunity to better understand Sher-Color by accessing Color Training from the Color Menu.

  • Color Training does not create tint orders or dispense tint. You will be able practice on the Color Eye if it is attached to the workstation where you are accessing training.
  • You will be able to practice the functions of Sher-Color to help you better understand each phase of the new tinting process by stepping through each process without dispensing tint, expensing orders, and processing point of sale transactions.
  • Practice using each selection from the Color Types menu.
