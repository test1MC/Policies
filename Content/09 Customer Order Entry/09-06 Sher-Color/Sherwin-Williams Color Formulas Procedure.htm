Sherwin-Williams Color Formulas Procedure

This section does not apply to Automotive Branches or General Polymers

Sherwin-Williams Color enables you to access all Sherwin-Williams color formulas.

Step

Action

1

Select Sherwin-Williams Color and enter the Color Number.

Note: If you don't know the color number, press Lookup [F3] to enter a portion of the color name

2

Enter the Sales Number, Quality, Composite, Finish, Rex, Size or press Lookup [F3] to view all bases.

3

Select from the following options:

  • Sher-Color Formula
  • Formula Book
  • Percent of Formula Book

4

Select from the following options:

  • Print
  • Dispense
  • Next
  • Customize
  • Hand Dispense
  • Clear

Note: The default selection is Dispense.

5

Enter the Number of One Gallon Containers to dispense.

6

Scan the UPC barcode on the paint can or manually enter the Product Sales Number.

7

You will be prompted to Position the Container and Press Any Key to Begin Dispensing.

8

Press Next to continue with the order or select from the following options:

Option

Description

Print

Print an additional formula label for the top of the can.

Dispense

Tint another container using the previously entered information. A Formula Label will print once the colorant is dispensed.

Next

Return to the detail area of the order.

Correct

If you select Correct you will be able to adjust the previously tinted product by choosing from the following options:

Option

Description

Manual Addition

Change the formula manually. The original formula remains as a reference on the screen. Press Lookup [F3] to view the Colorants selection list.

Percent Addition

Calculate a percentage of the current formula.

Tag as Mistint

Tag as Mistint so that the manager can process the Store Expense transaction.

Clear

Clear the correction cycle and start over.

After selecting the correct product, quality, and base, select from the following options:

Option

Description

Print

This selection will prompt you to enter the number of required labels.

Dispense

Tint another container using the previously entered information

Next

Tint a different product or continue with the order

Customize

You will be able to Modify or Adjust the formula by percent or size

Manual Dispense

Marks the order that you will handle the dispense of, manually.

Clear

Clear the correction cycle and start over

Tag as Mistint

Tag as Mistint so that someone in store management can process in Order Disposition

Hand Dispense

Marks the order that you will dispense manually.

9

Press Proceed [F12] when finished.
