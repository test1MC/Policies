Custom Color Procedure

This section does not apply to Automotive Branches or General Polymers

Custom Color enables users to tint a gallon of paint manually. If employees want to customize the color by adding colorant, they should make this selection.

Step

Action

1

Press Lookup [F3] or manually enter the colorant and quantities.

2

Select from the following options:

Option

Description

Print

Print an extra formula label for the top of the can.

Dispense

Tint another container using the previously entered information.

Next

Tint a different product or continue with the order.

Modify

Modify the custom formula before dispensing tint.

Adjust by %

Adjust by a percentage of the current formula.

Adjust by Size

Adjust the formula by container size.

Save Color

Once saved, lookup through Recall Custom Palette, which enables users to retrieve any previously saved Tint-a-Color formulas.

Manual Dispense

Marks the order that you will dispense manually. End a tint line item when you choose to use manual tint canisters.

Clear

Clear the screen and start over.

Note: The default option is Dispense.

Note: When you select Custom Color from a Color Eye terminal you will be prompted to use the color eye or proceed manually. If the Color Eye is not attached you will not see the prompts for the Color Eye option.
