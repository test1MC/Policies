Attach Label Printer Procedure

This section does not apply to Automotive Branches or General Polymers

The label printer field requires you to identify how you will implement your label printer on this workstation. If you do not have a label printer at this workstation, choose None and press Enter to continue to the next field.

Step

Action

1

Choose how you will implement label printing on this workstation from the drop down list and press enter to continue. The choices are:

Label Printer Choice

Description

Label Printer This Station

Print labels only from this SOURCE workstation. Select this choice when the label printer is attached to the Source workstation.

Label Printer Elsewhere in The Store

Allows any SOURCE workstation to print labels at a Monarch label printer located on another Source workstation. Select this choice when there is no label printer attached to the Source workstation.

Store Laser Printer

Print labels on your SOURCE laser printer. We recommend this option only as a temporary measure until you procure a Monarch label printer.

None

No label printing from this SOURCE workstation.

2

Select the default Yes when prompted: Will other stations print here too?

3

Enter a serial number in the Printer Serial Number field and press Enter to continue.

4

When all data has been entered, pressing the ENTER key will automatically accept changes and return you to the Help Desk menu. You may also press Accept [F1] to accept all data changes entered.
