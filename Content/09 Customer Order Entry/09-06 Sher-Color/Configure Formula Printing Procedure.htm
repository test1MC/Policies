Configure Formula Printing Procedure

This section does not apply to Automotive Branches or General Polymers

Color Numbers, Color Names and Color Formulas will automatically print on the customer's receipt even if the formula does not print. The store manager is able to change the setting for formula printing for all customer transactions by following these procedures.

Step

Action

1

From POS desktop, select Store -> Manager Functions -> Enter Manager Password -> System Control -> Configure Formula Printing.

2

Select Customer Receipt to adjust the formula printing on all customer receipts.

3

Select Picking Ticket to add or remove formula printing from all Picking Tickets and Customer Orders to add or remove formula printing from all Customer Orders.

4

Press Enter to change the formula print option from YES to NO.  Press Accept [F1] when complete.

Note:  Before completing a point of sale transaction users have the option to adjust the way formulas print on the customer's transaction. Select F. Formula? from the Form of Payment menu and Enter to change the YES to NO. Press Accept [F1] when complete. Making a change in the formula printing will only affect the transaction being processed. Color Numbers and Color Names will automatically print on customer transactions even if the formula does not print.
