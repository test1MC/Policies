Color Menu Overview

This section does not apply to Automotive Branches or General Polymers

At any workstation, you can look up the correct formula and base for a particular color by pressing Inquiry [F4] and selecting Color Menu.

Function

Description

Color Base Checker

This function allows users to research the correct base for any Sherwin-Williams color by entering a partial/complete color number or name.

Color Formula

This function allows users to look up a formula for quick review.

Tint Queue

This function allows the person tinting the paint to instantly see the next tint request since it refreshes every 15 seconds with the most recent request displaying first. A normal Order Flow using the Tint Queue works like this:

  • Order is taken
  • Employee enters the color number and product to be tinted, which will be labeled “Hold for Later Dispense.”
  • Employee tinting the paint selects the request.
  • The sales transaction can be processed.

Color Eye Utilities

This function enables users to troubleshoot and calibrate the Color Eye.

Option

Description

Closest Fan Deck

This option enables users to measure a sample with the Color Eye and find the closest Sherwin-Williams color or a competitor’s color.

  • Place the sample in the holder and press Enter. Press Cancel when finished.
  • The Delta-E is a measure of how close the sample is to the original color. In most cases, the human eye cannot detect a Delta-E less than .50, and a Delta-E greater than 1.00 can be detected.

Calibrate

This option provides a method to recalibrate the Color Eye without having to wait for the automatic prompt to calibrate, which occurs every 4 hours. Calibration involves taking No Reflectance Measurement, a measurement of white and a test measurement of a known sample.

  • Use an open port for the No Reflectance Measurement, use the white calibration tile for the white measurement and use the green tile for the test measurement.
  • Hold the Sample Holder Arm down and keep everything away from the opening to prepare for the Open Port Measurement. Do not look directly into the Color Eye measuring apparatus. Press Enter.
  • Ensure the White Calibration tile is clean and place the white calibration tile in the Sample Holder Arm. The round indent on the back of the calibration tile should fit the round backing of the Sample Holder Arm. When the tile is in place press Enter to take the measurement.
  • Ensure the Green Tile is clean and place the green tile in the holder. The round indent on the back of the tile should fit the round backing of the Sample Holder. When the tile is in place, press Enter.

Note: If you receive a Delta-E warning message as a result of the Test Check Tile step, call your Store Computer Operations Help Desk Technician at 216-566-2740.

Test Check Tile

This option is used to determine if the Color Eye is reading accurately. It will compare the measurement of the Green Tile against the first time you measured it when the Color Eye was installed.

  • Ensure the Green Tile is clean and place the green tile in the holder. The round indent on the back of the tile should fit the round backing of the Sample Holder. When the tile is in place press Enter.

Note: If the Color Eye is still reading within tolerance you will receive [Green Tile Test Successful] message. If you receive a Delta-E warning message call your Store Computer Operations Help Desk Technician at 216-566-2740.

Advanced Color Utilities

This feature will only be used if instructed by the Help Desk.

Tinter Utilities

This function displays all possible Tinter commands for all Tinter models. You will receive a message indicating if a particular function is not available for a Tinter model:

Operation

Description

Purge All Colorants

Move fresh colorant through the dispenser head. Requires purge container under dispenser head. Normal daily operation requires the employee to execute this operation each morning. Purge can also be done as necessary during the day. Not available for Shertinters.

Purge One  Colorant

Move fresh colorant through the dispenser head. Requires purge container under dispenser head. Use this function to resolve colorant problem with one specific canister. Available Corobs only.

Dispense Colorants

A quick manual dispense program used for Tinter problem solving and special or unusual customer needs.

Colorant Levels

Maintain colorant level tracking as you fill Tinter canisters with colorant. Colorant Levels must be accurately reset when canisters are filled to avoid mistinted gallons.

Initialize Tinter

Starts Tinter normal processing cycle. Used as basic Tinter operational status check during problem resolution by Stores Computer Operations.

Agitate All Colorants

Stir colorant in all Tinter canisters.

Recirculate All Colorants

Moves fresh colorant through all dispense lines and canisters. This operation does not dispense colorant. This operation on Shertinters sets calibration data for the dispenser.

Recirculate One Colorant

Moves fresh colorant through one dispense canister and its lines. This operation does not dispense colorant.

Clean Lines

Dispenses clear colorant through dispense head and lines on the Shertinters only. Requires purge container under dispenser head.

Advanced Utilities

Special functions used by the Stores Computer Help Desk to troubleshoot Tinter problems. Used only when directed by Store Computer Operations staff.

Note: The Advanced Utilities choice will display another menu of advanced Tinter diagnostics and utilities. This menu and its choices must only be used with direction by the Stores Computer Operations technicians.
