Copy Color Overview

This section does not apply to Automotive Branches or General Polymers

Copy Color should be used when a customer returns to the store for more paint from a previous order. Users can copy a color entry from any previous order to the current order (it does not have to be for the same customer). To easily identify a previous order to Copy From, narrow the search in the Order Lookup menu by entering a specific customer, sales number, color, etc. Each tinted line of an order displays on the screen. Locate the correct item and press Enter.

Note: Light Source will only be available when the product is tinted using Sher-Color Formula.
