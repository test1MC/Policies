Wallcovering

Wallcovering Orders must be processed using the Wallcovering Order Memo through Customer Order Entry.  The manual Wallcovering Order Memo [655-2970] should only be used for downtime purposes. Line Items for deposits will not print on the Wallcovering Order Memo.

If

Then

Wallcovering Order is placed and the deposit line on the Customer Order has not been processed

This message will print on the bottom of the Wallcovering Order Memo:  “A deposit of $000.00 is required.”

Deposits are processed through the Sale Transaction for a Wallcovering Order

The Wallcovering Order Memo will automatically be printed.

The deposit is processed without linking to the Wallcovering Order and Order Disposition is later used to link the Sale Transaction

The user will have the option to print the Wallcovering Order Memo with deposit amount at that time.

Once all your information is entered, you must print the order in order to obtain the customer signature on the order. Select Print/View Wallcovering Order Memo. 

By selecting Print Wallcovering Order Memo, you will able to print out a copy for you and the customer. Have the customer sign the store copy.

The Terms and Conditions contained on the manual Wallcovering Order Memo will automatically print on the system generated Wallcovering Order Memo.

Note: For more information on Wallcovering purchase orders, see 15-01 Wallcovering Special Order.
