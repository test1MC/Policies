Cancel or Delete COE Order Procedure

Step

Action

1

From POS desktop, select Store -> Customer Order Entry -> Enter Employee Number and Password -> Update/Review Orders.

2

Select the correct order.

3
If Then
Canceling a COE Order

Select Cancel Order -> Procedure is complete.

Note: Only New or In Process orders with no sales can be canceled. If merchandise has been sold from the original, delete each remaining line. If you cancel an order and later realize that you need to complete the order, you will be required to start over with a new order.

Deleting a COE Order Review the Line Status on each entry. Any merchandise that has been processed on a point of sale transaction will have a Line Status of SOLD.

4

Press the DELETE key while the cursor is on the item quantity for the merchandise that the customer will not be picking up.

5

Answer Yes to the question DELETE ITEM - ARE YOU SURE? The balance of the order line will be canceled. You are able to review canceled order lines in PRINT/VIEW ORDER.

6

If necessary, press ENTER or the END key to move to a blank line. Press Total [F12] when finished. The completed Customer Order wills no longer display while processing a Point Of Sale transaction once all items have either been SOLD or DELETED.
