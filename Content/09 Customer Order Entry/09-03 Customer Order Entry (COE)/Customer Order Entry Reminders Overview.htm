Customer Order Entry Reminders Overview

Several features are available to remind you of incoming orders and store processed orders:

Reminder

Description

Status Box

Contains an "ECOMMERCE/EDI" reminder area. Review this area throughout the day to identify incoming orders that require immediate action.

Suggested STAR Order

If you do not currently have enough product at your location, the system will automatically place DSC merchandise on your next Suggested STAR Order. If you will not receive your next STAR order before the required date the information will print on the ITEMS TO BE ORDERED report. You will have to manually add the items to the STAR order if the Suggested STAR Order was generated earlier in the day.

Items to be Ordered

This report automatically prints when you exit customer order entry. The report lists items that must be ordered to complete the customer request.

Note: To print this report on demand, select Print/View Order from the Order Entry menu and search for the order. Then, select Print Order Reminder.

Customer Order Entry Items Received

When you receive merchandise against a purchase order that is needed to fill an order, a report titled CUSTOMER ORDER ENTRY ITEMS RECEIVED will automatically print. Use this report to identify and fill outstanding orders.

Clear MMS/COE (Customer Order Entry) Status

A selection is available from the Help Desk menu that will enable you to clear a purchase order or customer order when you have processing problems.

Note: Please do not use this function without first calling the Store Computer Operations Help Desk at 216-566-2740.
