Add COE Order Procedure

Step

Action

1

From POS desktop, select Store -> Select Customer Order Entry -> Enter Employee Number and Password -> Add Order.

2

Enter the customer’s purchase order number (up to twenty alpha/numeric characters).

Note: If the customer does not use a purchase order, press Enter.

3

Enter the customer's Job Number/Lot Number or press Enter to continue.

  • Control number (Ctl #): a system generated tracking number for each generated order.
  • Type: location of entered order, Store Entered, EDI and Internet.
  • Order: current order (0E00000034A1035)
  • (0E): Order Entry-Store generated, AT-SWAT, SW-E-Commerce, ED-Electronic Data Interchange
  • 8 digits: numeric order number
  • (A): alpha/numeric field that ties orders together
  • (1035): store number

4

Make a selection from the Customer Service Options menu or press Enter to select Press Enter to Continue to move on without selecting a Service Option.

5

If Delivery or Installation is chosen, a dialog box is displayed:

Choose from the following options:

If

Then

None

You will return to the previous screen to select a different option.

Some

An extra field at the end of the line will display that enables you to select whether the line item is to be delivered or installed.

All

All entered line items are defaulted to Deliver or Install.

6

Enter the customer's account number or press Lookup [F3] to enter a partial or full customer name. For more information, see 09-01C Point of Sale.

7

If available, contact and phone number will return from the price record card. If unavailable, enter the contact and phone number.

8

If the [Bill To] and the [Ship To] addresses are the same, complete the information on the screen (excluding the [Ship To] information), then press Accept [F1]. Answer Yes when prompted to [Use BILL-TO Address as SHIP-TO Address?], press Accept [F1] to continue. The [Ship To] information will automatically fill in on the screen. If you select NO you will be given an opportunity to enter the [Ship To] information. Enter only a Lot Number or Job Location on the first address line of the SHIP TO information if a complete ship to address is not available. Press Accept.

9

Enter the product sales number, or press Lookup [F4] to locate the product number. After entering the sales number, you will see additional inventory information.

Note: Add comments to a line by pressing the Comment key. Continue entering the product sales numbers. When complete, press Total [F12] or Accept [F1]. After entering Special Instructions, press Accept [F1]. Print a Picking Ticket to reference while preparing the order or print a copy of the customer order to give to the customer.

10

Review the Inventory section in the top right of the screen to review inventory activity for the product.

  • Individual line items can be selected for Deliver or Install if the appropriate option was chosen in the Header section of the customer order.
  • For a delivery order, the heading for the last column is "D"
  • For an Install order, the heading for the last column is "I"
  • If not a Delivery/Install order, the column heading is blank.
  • Press the SPACE BAR to select Deliver/Install.
  • If a line item is "deleted" [by pressing the DELETE key], the line status is changed to X=Canceled, and the line is removed from the display.

11

After completing the order, print and/or update the order if necessary.
