Customer Order Entry Inventory Status Overview

Status

Description

Non Committed

Quantity available at your location (does not include the committed amount)

Committed

Quantity committed on other orders but not yet sold

On Order

Amount currently on order but not received

Line Status

Condition for current product:

Line Status

Description

F

Committed

Stock was available and inventory was committed for the order quantity

H

On Hold

Item is on hold waiting for action

L

Sold/Canceled

Items on order sold; the balance of the order was canceled

N

No Action

Item is not committed and not ordered (mainly happens with EDI orders. You are required to take action - either commit the inventory or place an order – then the item moves to the next status)

O

On Order

External item placed on Order Reminder Report; Internal item placed on Memo Demand

P

Partially Committed

Inventory was committed for part of the order quantity and the balance was placed ON ORDER

S

Sold

Point of sale transaction processed for this product

X

Canceled

Order canceled

Available

When product should be available

Commit

Amount committed from this order

Order

If not committed, internal products will automatically generate a Memo Demand and be added to your next Suggested STAR Order if the next delivery date is before the Required date
