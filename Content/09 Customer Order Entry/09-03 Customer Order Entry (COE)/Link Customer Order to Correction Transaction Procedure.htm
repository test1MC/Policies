Link Customer Order to Correction Transaction Procedure

Customer Orders can be linked to Correction Transactions.

You will be prompted to select from the All Orders list when you start a Correction Transaction. The selected order will be applied to the positive side of the Correction Transaction. This process will be corrected in an upcoming release of software so that you are not prompted to select from the All Orders list until you are in the positive side of the Correction Transaction.

Step

Action

1

From POS desktop, select Store -> Point of Sale -> Press END key -> Select Correction from the Special Menu.

2

Enter the original Terminal/Transaction Number and account information for the original point-of-sale transaction.

3

Enter the original transaction information on the negative side of the Correction Transaction. Press Accept [F1] when complete.

4

Answer Yes when prompted to Review Orders. Press Accept [F1] to access one order or press Enter to select several orders and press Accept [F1] when complete.
