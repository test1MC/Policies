Link Customer Orders to One Transaction Procedure

You are able to link several customer orders to one point-of-sale transaction. Customer Orders MUST contain the same Customer Purchase Order Number in order to process the orders on the same point-of-sale transaction.

Step

Action

1

From the POS desktop, select Store -> Point of Sale -> Enter Employee Number and Password -> Sale.

2

Enter the customer's account number and press Enter.

3

Answer Yes when prompted, "Review Orders?"

4

Select the appropriate order.

Note: The X in the far right corner shows that the order is selected and will be brought into the point-of-sale transaction.

5

Use the down arrow key to move to the next order. Press Enter to select and Accept [F1] when complete.

6

Select All Unsold Lines.

Note: Customer order numbers tied to the point-of-sale transaction will print at the bottom of the customer's receipt.  If you find that you need to add another order after selecting All Unsold Lines, press Void and start over.
