Customer Order Entry Status Box Procedure

The STATUS BOX contains an "ECOMMERCE / EDI" reminder area. Review this area throughout the day to identify incoming orders that require immediate action.

Step Action

1

From POS desktop, select Store -> Customer Order Entry -> Enter Employee Number and Password -> Update/Review Orders.

2

Narrow your search by selecting Order Type and NEW status orders or Press Lookup [F3] to view the types of orders. Press Accept.

3

Select the appropriate order from the list: Suggested STAR Order, Items to be Ordered, Customer Order Entry Items Received, Clear MMS/COE (Customer Order Entry) Status.
