Note: Seven options are available after narrowing the search to one order. Make the proper selection and follow the system prompts in the lower left corner of the screen. To view another order, select View Another Order Record from the Options menu. Press Cancel [F5] to return to the previous menu.
Printing E-Commerce Order Select Update/Review Orders -> Select E-Commerce from the Order Type menu -> Select New from the Customer Order Status menu -> Press Enter to select All Service Options. Press Accept [F1] twice -> Proceed to Step 3.

3

Press Enter to select Sort List of Selected Customer Order by Required Date.

Note: A list of E-Commerce orders will display on the screen.

4

Highlight the appropriate order and Select Print Order -> Select View Another Order Record -> Repeat this process to print each order -> Press Cancel [F5] when complete.

5

Select Store Entered from the Customer Order Types Menu -> Select Ready for Delivery and Delivery Only -> After selecting the order from the Order Entry Lookup menu, select Reprint Confirmation of Delivery Ticket.

Note: If material is going to be dropped off at a jobsite where no one is present to sign for the receipt of the material, the Confirmation of Delivery must show who ordered the material, and a signed Authorization and Release Form must be on file at the store stating that they will accept responsibility for the material without a signature. If this is not possible the material is to be returned to the store for later delivery.
