Process COE Order Procedure

Step

Action

1

Follow your normal procedures for processing a point of sale transaction.

Note: All Point Of Sale functions are available while processing the transaction.

2

After entering the customer account number or selecting their name from the LOOKUP menu choose if you would like to process a normal transaction or finalize an order.

If

Then

Finalize an order

Process all unsold lines or select the lines that you want to sell.

Add another order

After selecting All Unsold Lines, press Void and start over.

3

Any item on the original order that has not been sold [back order or customer does not need yet] will print on the second page of the sale transaction. The unsold items will be identified [ITEMS AVAILABLE, NOT PICKED UP].

4

Once the order is complete the selected products will be marked as SOLD.

If

Then

No

Process a normal transaction for the customer.

Yes

Finalize a previously entered order. If the customer has several orders the ORDERS FOR CUSTOMER menu will return a list of orders. After selecting YES, press LOOKUP to view ORDER LINES or press ACCEPT to select ORDER. You will be able to quickly review the products for each customer order.

5

Enter all unsold lines or select lines that you would like to sell.

If

Then

Selected Lines

Select only those items you choose to sell on the transaction by pressing ENTER on each line (pressing ENTER on each line will place an X in the selected (S) column). Then press ACCEPT when complete. While processing the sales transaction press VIEW (F7), when on a blank line, to view all order entry lines for this order. If you choose to add another line, press ENTER to Select, then press ACCEPT to transfer the selected item to the point of sale transaction.

All Unsold Lines

The on-order or committed lines with prices will automatically return to the screen.

Note: Order entry line item with a price of zero (0.00) will not be brought into point-of-sale. Also, you will not see zero (0.00) priced lines when you press View [F7].
