Print Picking Ticket Procedure

Permanently change the default print number for Picking Tickets or adjust the number of Picking Ticket copies each time you print a Picking Ticket.

Step

Action

1

From the POS desktop, select Store -> Manager Functions -> Enter Employee Number and Password -> System Control.

Note: Only users with the system management password will be able to access.

2

Select COE Picking Ticket Copies from the System Control menu.

3

Enter a new default print number for Picking Tickets.
