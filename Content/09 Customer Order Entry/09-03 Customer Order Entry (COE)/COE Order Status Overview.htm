COE Order Status Overview

Status Description

N

New

Awaiting action to be taken at the store

H

Hold

Order is holding for further action

I

In Process

The order is in the process of being filled at the store

D

Ready for Delivery

Order has been finalized for Delivery. No changes (except deleting line items) can be made to the order

P

Ready for Install

Order has been finalized for Installation or installer pickup. No changes (except deleting line items) can be made to the order

C

Closed

Order has been completed and sold

X

Canceled

Order has been canceled
