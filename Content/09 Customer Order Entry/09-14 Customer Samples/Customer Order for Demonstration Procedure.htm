Customer Order for Demonstration Procedure

This section does not apply to Automotive Branches

Step

Action

Processing a Demonstration

1

From POS desktop, select Store -> Customer Order Entry -> Add Order -> Select Demonstration from Customer Service Options -> Enter customer information.

2

Enter the required parameters and press Accept [F1].

3

Enter each product.

4

Select Ready for Delivery from the Delivery Menu.

5

Print the Confirmation of Delivery and have customer sign it.

Store Expense

1

From POS desktop, select Store -> Point of Sale -> Press Page Down key -> Select Store Expense.

2

Select Demonstration Expense from Prime-Sub List.

3

Select the correct customer order.

4

Enter the appropriate parameters.

5

Select All Unsold Lines

Note: All demo items listed on the Customer Order should now display on the screen.

6

Press Comment [F9] to enter comments about the order.

7

Press Total [F12] to complete.

8

Staple a copy of the Store Expense Transaction to the signed Confirmation of Delivery and retain the documents.
