Test Drive Overview

Test Drive is a program that helps store managers and sales representatives encourage customers to “test drive” both new and existing GOM products by extending a 50% discount (off customer’s best price)  or 40% discount off Emerald’s list price on select products.

Stores may monitor use of the Test Drive Program using the Store Accountability Coupon Usage Report available through Inquiry [F4]. However, there are no requirements to print and retain reports.

Stores in EAD, MWD, SED, and SWD will use the Test Drive Coupon to process these transactions.  Please access the Test Drive Coupon if a new copy is needed. Some districts and divisions may use barcoded hero cards or marketing to encourage test drive of products by customers. Prior to providing a serialized barcode, employees must verify the customer’s sales history. Stores in Canada will use a serialized barcode coupon to process these transactions.
