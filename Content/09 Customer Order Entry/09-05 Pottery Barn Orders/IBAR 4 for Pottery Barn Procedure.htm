IBAR 4 for Pottery Barn Procedure

Step Action

1

Does the order require tinted paint?

If Then
Yes Create an order only for the paint to be tinted using DIY -> Procedure is complete.

From the POS desktop, select Store -> select Merchandise Management from the Main Menu.

3

Enter your Employee Number and Password.

  • Cost Center

Release the IBAR.
