Pottery Barn Order Disposition Procedure

Step Action

1

From POS desktop, select Store -> Select Customer Order Entry from the Main Menu.

2

Enter your Employee Number and Password.

3

Select Order Disposition from the Order Entry Menu.

4

Order lines that have not been processed on point-of-sale transactions will return to the screen. Use the ARROW Key to select the entry that you would like to review. Press ENTER.

Note: Order Disposition should be used to correct tinted paint orders so they can be properly linked to the IBAR.

5

Select Process Order Line from the list.

6

Select Link Order Line to IBAR from the Order Disposition Menu.

7

Enter the Control Number or use one of the other Search Options to lookup the correct IBAR. Press ACCEPT [F1].

8

Select the IBAR.
