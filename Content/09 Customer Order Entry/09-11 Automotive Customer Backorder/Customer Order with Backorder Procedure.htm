Customer Order with Backorder Procedure

Step

Action

Complete Order Header

1

Complete your order header as you normally would with the following exception:  Delivery must be selected as the Customer Service Option.

Complete Order Detail

1

Complete the order detail as you normally would.

Shipment Detail

1

Select N to the prompt “Are lines on this Order ready for Delivery?” Press ACCEPT [F1] to proceed.

2

Select Print Picking Ticket, choose sort method, and print.

3

Select Update/Review Orders from Customer Order Entry Menu.

4

Use Order Lookup to find your order. Press ACCEPT [F1] to view a listing of all orders matching your query.

5

Press ENTER to select the order you wish to view.

6

Move through the COE until the prompt for “Are lines on this Order ready for Delivery?”   Enter Y.

7

Select the Store Truck as the carrier from the SCAC Code List.

8

Select the Delivery Date. Calendar navigation instructions are at the bottom of the screen.

9

Confirm the Ship Quantity.

Note: If the Ship Quantity is less than what is available, enter only the amount being shipped. Partial shipments are allowed; there is no need to create a new order for backordered items. See Fulfilling Backordered Products below for handling backordered products. The order with backordered products will remain in Customer Order Entry.

10

Do you want to create a Bill of Lading?

If Then
Yes Select Y to submit a Bill of Lading Request to BOLDOC. Press ACCEPT [F1] to proceed
No Select N to create a Manual Bill of Lading or your Third Party Carrier is providing one for you

Note: This will communicate your transaction information to HEARS. If you see an error message during any of the transaction submissions, please note the on-screen message and call the helpdesk. A Bill of Lading is required for all orders, regardless of transport.

11

Select Y to open HEARS to complete the Bill-of-Lading.

12

Process the Sales Transaction.

13

The system will print a Confirmation of Delivery showing the order quantity and the quantity shipped. Send this along with the order to the Customer.

14

The system will generate an “Items to be Ordered” report if the material is required before the next STAR Order Scheduled Delivery Date. You must get the material locally from another Branch, an Outside Vendor, or via Emergency STAR Order.

Note: If the material is not needed until after the next STAR Order Scheduled Delivery Date, it will automatically be included in the next STAR Order as Memo Demand and the Items to be Ordered report will not print.
