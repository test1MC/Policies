Daily Customer Service Report

[From POS desktop select Store -> Customer Order Entry -> Reports -> Daily Customer Service Report]

 
  • Review this report daily.
  • The min max for any backordered item should be reviewed and adjusted as necessary.
  • Items that show “in-stock” should be reviewed for possible errors. These variances should be reviewed thoroughly and the order filled if the product is on hand.
  • If the perpetual inventory is incorrect, it should be corrected after a review of the cause.
  • The report can also be reviewed or printed for any period. A date range can be entered for a week, month, etc. The fill rate goal will be taken from this report.
