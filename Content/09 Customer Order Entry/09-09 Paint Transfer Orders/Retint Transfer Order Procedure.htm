Retint Transfer Order Procedure

If an order needs to be re-tinted at the Service Center/Tint Facility select option Transfer Order from the Order Entry Menu to transfer the order back to the Service Center/Tint Facility from which it was originally filled. You can transfer orders which have been sold or closed.  This is critical as the color information will be updated on the order to keep an accurate record of the customer's color history, but the order cannot be transferred to a different location. Before transferring the order back to Service Center/Tint Facility, include a note in the Special Instructions of the Customer Order notifying the location of what actions are required for what product(s).

Step

Action

1

From the POS desktop, select Store -> Customer Order Entry -> Update/Review Order.

2

Select No when prompted "Review Only Quick Orders?"

3

Enter the search parameter.

4

Select the correct order.

Note: If the order is NOT closed, make any line item changes you may require. If the order is closed, you will not be able to edit the Order Header or Line Item information.

5

Press Enter to advance to the Special Instructions Window and include the re-tinting information.  Press Accept [F1].

6

Return to the Order Entry Menu and select Transfer Order.

7

Select the order you wish to transfer.

Note: You will only be able to transfer the order to the original transfer location.

8

Once the order has been re-tinted, it will be transferred back to your location. If you choose to charge the customer for the tint, you will need to process a Point-of-Sale Transaction.
