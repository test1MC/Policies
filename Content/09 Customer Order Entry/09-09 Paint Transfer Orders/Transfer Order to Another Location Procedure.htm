Transfer Order to Another Location Procedure

Step

Action

1

Create Customer Order per customer request.

Note: Select Delivery from the Customer Service Options Menu. This option must be selected for all transfer orders, even if the order is not to be delivered. Select NONE when prompted to indicate if the order is Ready for Delivery. The Service Center/Tint Facility mark it as ready when they have fulfilled the order and will use the Confirmation of Delivery tickets to obtain customer will call pickup signatures.

2

Transfer the Customer Order to the Service Center/Tint Facility/Store:

Step

Action

1

From POS desktop, select Store -> Customer Order Entry -> Transfer Order.

2

Search in the Order Lookup Window or press Accept [F1] to view all orders. Press Enter.

3

Enter the store number of the transfer location. Press Accept [F1].

4

Choose YES when prompted, "Transfer this order?" Press Enter to confirm.

5

Once the transfer function has been completed, the originating location no longer has control of the order and changes cannot be made. Orders with the status "In Update" may be orders that have been transferred.

Note: An ISR Purchase Order is automatically created through the transfer process. Receive against it once the order is transferred back to your location. Once an order has been transferred back, the "Transferred In" number in the Black/Yellow Status Window will increase.
