Transfer Order Help

Transfer Type

Description

Mistaken Transfer

If you discover that you have transferred or re-transferred an order by mistake to the Service Center/Tint Facility, call the location immediately and ask that they transfer the order back. It is imperative this occurs as soon as possible so they do not mistake the order as a new order and fill it.

Retint after it has already been transferred back to the store

The order can be transferred back to the Service Center.  It can only be transferred to the original transfer location [Service Center].  You will not be able to transfer the order to another location.

Closed Order

Once an order is closed, the Order Header or Line Item information cannot be edited.  If the order is not Closed but has already been fulfilled or partially fulfilled by the Service Center, follow up with Special Instructions and call the Service Center so that the order is not filled twice.

Product Finishes facilities

If you do not want to ship back to the Product Finishes facility, but instead have the order sent directly to the customer, change the header by accessing the Special Menu, clicking IST, and entering the Receiving Store Number and the ISR Request Number. Then, overwrite what appears in the ShipTo Address to the address you require. Press Accept [F1].
