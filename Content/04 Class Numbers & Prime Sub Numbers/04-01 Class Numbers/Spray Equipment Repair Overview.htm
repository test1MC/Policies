Spray Equipment Repair Overview

Spray Equipment repair items sold using a sales number will properly relieve your perpetual inventory. If you lump together all items that go into the repair and use a class number, your perpetual inventory will not be adjusted properly unless you perform an Inventory Adjustment. The following class numbers can be used for spray equipment.

Description Class Number
