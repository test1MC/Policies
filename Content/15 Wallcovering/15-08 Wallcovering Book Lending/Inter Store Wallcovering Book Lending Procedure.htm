Inter Store Wallcovering Book Lending Procedure

Wallcovering books may be lent between stores. To lend a wallcovering book to another store, an ISR/IST Transaction should be processed:

Step

Action

Store Lending Book

1

Process an IST Transaction to initiate the lending of the book when notified of the ISR creation by the borrowing store. 

Note: For more information on processing IST Transactions, please see Inter Store Transfer (IST) Overview.

2

When processing IST Transaction:

  • Use Sales Number 161-9618. 
  • Press COMMENTS [F9] on the line item and identify the book title.

3

When notified by the lending store that they are ready to return the book, create an ISR Purchase Order for the return of the book and share the purchase order information with the borrowing location.

4

Process an ISR Purchase Order. 

Note: See Internal Purchase Orders Overview for more information on processing ISRs.

5

When processing the ISR Purchase Order:

  • Use Sales Number 161-9618.
  • Press COMMENTS [F9] on the line item and identify the book title.

Store Borrowing Book

1

Process an ISR Purchase Order. 

Note:  See Internal Purchase Orders Overview for more information on processing ISRs.

2

When processing the ISR Purchase Order:

  • Use Sales Number 161-9618.
  • Press COMMENTS [F9] on the line item and identify the book title.

3

When done borrowing the book, notify the lending store that they may create an ISR Purchase Order for the return of the book.

4

Process an IST Transaction to return the book when notified of the ISR creation by the lending store. 

Note:  For more information on processing IST Transactions, please see Internal Purchase Orders Overview.

5

When processing IST Transaction:

  • Use Sales Number 161-9618. 
  • Press COMMENTS [F9] on the line item and identify the book title.
