Wallcovering Book Lending System Parameters Procedure

Once you have District Manager approval to operate a wallcovering lending library and have been provided the required deposit amount, you must set the system parameters in POS to operate your wallcovering book lending system: 

  • Required Deposit Amount: Enter the amount your District Manager established or the actual value of 0.00 will be used. This amount will return to the screen for all wallcovering books loaned out at your store.
  • Due Date: Changing the number of days from check-out that books are due will adjust the system reporting of overdue books for all customers. For more information, please see 09-03A W/C Book Lending Controls.

Follow this procedure to set your wallcovering book lending system parameters:

Step

Action

1

From POS desktop, select Store -> Manager Functions -> System Control -> WC Book Lending Controls.

2

Enter the Number of Days from Check Out that Books are Due. Press ENTER.

3

Enter the Default Amount of Deposit. Press ENTER.

Note:  The G/L Accounts used for wallcovering book check-out and check-in display on the bottom of the system control screen.
