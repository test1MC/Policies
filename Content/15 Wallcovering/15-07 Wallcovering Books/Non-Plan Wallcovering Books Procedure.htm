Non-Plan Wallcovering Books Procedure

Before ordering wallcovering books directly from a distributor, first check to see if the wallcovering book information is available at SWAT. If the book does not exist at SWAT, follow this procedure:

Step

Action

1

Place a Purchase Order using Sales Number 161-9618 All Other WC Books.

Note: This sales number should ONLY be used for ordering and receiving the wallcovering book. Do NOT use this sales number for ordering wallcovering from the book.

2

If a number has not been assigned, ask the distributor for a Book Number to register the book. 

Note: This new book number should be used when ordering wallcovering from the book.

3

Add the wallcovering book to the Wallcovering Book Register.

4

If a label is not on the back cover of the wallcovering book, add the information to the Wallcovering Book UPC Label Request Form.
