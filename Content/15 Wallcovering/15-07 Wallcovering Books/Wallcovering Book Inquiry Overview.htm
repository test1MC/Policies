Wallcovering Book Inquiry Overview

The store computer system validates against the SWAT Database anytime a wallcovering book sales number or pattern number is referenced. Search tips:

  • When prompted for the book name, enter a key word to return every book with that specific key word in the book name.
  • When prompted for a pattern number, enter the exact pattern number.
  • Sherwin-Williams' private label wallcovering books always begin with “SW” before the book name.
  • Wallcovering book names on the SWAT database do not normally contain articles in the beginning of the book name.  For example:
    • A Place in the Country -> Place in the Country
    • The Creative Edge -> Creative Edge
