Edit BOL Request Procedure

The BOLDOC Edit Selection Screen allows users to view all documents currently available in BOLDOC. The lower window of the Edit Selection Screen displays order information to help when creating a Bill-of-Lading. By selecting on an order, you may view information such as Ship To Address(es) and Customer PO Number(s).  If multiple orders were submitted for a single Bill-of-Lading, you will see both sets of information. All Bills-of-Lading require additional information entry and some editing before a final Bill-of-Lading may be generated.

Step Action

1

From POS desktop, select Store -> Customer Order Entry -> Delivery Menu -> Open BOLDOC -> Select the Customer Order.

2

Select the pending BOL or select Edit.

3

Specify the Regulatory Rule and whether or not the Consumer Commodity Exemption may be taken for the shipment. Select OK.

Note: By default, US Land will display for shipments originating in the US with the consumer commodity indicator enabled. Canadian Land will display for shipments originating in Canada where consumer commodity does not apply.

4

Review all the information displayed on the Edit Selection Screen:

  • Ship Date
  • Shipment Terms
  • Be aware of the following in regards to Shipment Terms:
    • The RemitTo (Invoice) Address defaults as the order ShipTo Address for BOLs designated as Collect Shipments.
    • When Shipment Terms are designated as 3rd Party, the Third Party Remit To Window becomes enabled when the RemitTo Address is not the same as the BillTo Address.  Select on the icon to access the Third Party Remit To Window it when required.  Enter the Third Party Remit To Information and select OK.  If you select Default, the Third Party Remit To Information will default to the Customer BillTo Address.
    • Customer’s addresses, account, and order number information

Note: The Edit Screen displays information about the entire order. The upper window will display information about the order while the lower window will display the submitted line items.

5

Review all the information displayed on the Shipped Items Window. Line Items displayed in red require additional information before a BOL can be generated.

6

Edit the product information in the Shipped Item Edit Screen.

7

Line Items with the BOL Code Manual require an assigned code.

If Then

Product was not assigned a Bill-of-Lading Code.
  • The product is new and the information and has not yet been provided to HEARS for product classification.
  • The product is an outside buy and has been assigned the generic Size Code 99. 

Note: HEARS requires specific size information in order to classify a product.  Until the issues with product set-up have been addressed, manually resolve the BOL Code information for the product each time it is submitted. Not including these products on a BOL is not an option. Stores will be audited to ensure all required products for delivery are included.

8

Is this product for an Outside Buy Request?

If Then

Yes

Complete an Outside Buy Regulatory Information Form found in the Forms Repository.

No

Continue to Step 9.

Note: Notifying the Regulatory Affairs Group with product issues will allow them to research and correct product information so it is properly classified within HEARS.

9

Create Repack Cartons for all items with quantities noted in the Remaining Loose Units column.

10

Once there are no red-line items and no repack quantities, select OK on the Edit Selection Screen and return to the main screen for printing.

Note: The Shipped Items Window displays basic item line information required to complete the BOL. Hover over a line item to see Product Description.

11

Print a single SDS for any product, if necessary.

Note: You may also print SDS for all products on the order when you generate the actual Bill-of-Lading.

12

Select Apply and Save on the Edit Screen to save changes.

There are several fields concerning product information in the Shipped Item Edit Screen that need to have values provided if they do not exist.  Review all fields, even if they have values.

Step Action

1

Select the line item from the Shipped Items Window.

2

Press Enter or select Edit Item.

3

Review the following fields: Empty Container Weight, Tare Weight, Unit Weight, Product Weight per Unit Container Count (the actual number of units shipped), Containers/Carton (number of containers per carton). If shipping A/P product and this value is unknown, leave the field blank and select the Unknown Checkbox, and Package Type (drum, pail, or carton)

Note: All weights display on BOLDOC screens in pounds, which are converted to kilograms when the document is printed for Canadian stores. Product classified as Size Code 99 is typically assigned Package Type of Carton (CTNS).

4

Verify the BOL Code Information:

If Then

The BOL Code appears in the available listing

Select the correct BOL Code.

Note: The Hazardous Flag displays in the right-most column.

None of the entries listed fit the product

Select Manual Entry. Include a Manual Entry Shipping Description and indicate whether or not the product is hazardous. When complete, select OK to save the information or Cancel to exit.

Note:  Hazardous and regulated product must include a UN Number and a complete and valid entry from the Hazardous Materials Table per Federal regulations.  Be aware that invalid descriptions are unlawful and can result in a citation and fine.

5

Select OK to save information or Cancel to exit the Edit Screen without saving. 

Note: Once you all information for each item has been reviewed/resolved, the line should display in black in the Shipped Items Window of the Edit Screen.  If the line is still red, there is still some information that is not completed. The information modified in the Shipped Item Edit Screen displays in the Shipping Description Window.
