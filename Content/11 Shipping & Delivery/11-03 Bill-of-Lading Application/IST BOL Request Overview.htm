IST BOL Request Overview

In order to create a BOL for an IST, the process must be initiated from within the IST transaction.  Once the IST is completed, it will prompt the user to select printing options for Packing Slips and Receipts, select the option to create a Bill-of-Lading.  If you select Y to create a BOL for the IST, the system will automatically open the Bill of Lading Request Screen. Complete the screen and access BOLDOC just as you would with a Customer Order.  

Sherwin-Williams Delivery Service Stores: the store creating the IST is required to create the Bill-of-Lading for the delivery.
