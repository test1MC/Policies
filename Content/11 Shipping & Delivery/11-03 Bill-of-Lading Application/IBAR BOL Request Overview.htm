IBAR BOL Request Overview

Bills-of-Lading may only be requested for the following Return IBAR Codes:

Stop Sale – To Plant or DSC, Stop Sale – To Vendor, Quality Return – PPR, Emergency. Con-way will create the Bill-of-Lading for returns handled by them using the information provided.  Once completed and the IBAR released the IBAR, open BOLDOC.  You must open BOLDOC when creating the IBAR.  If not, you will not be able to use BOLDOC to create a BOL for the IBAR and there is no method to resubmit the information.
