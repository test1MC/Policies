Navigating BOLDOC Overview

Upon opening BOLDOC, the Release Notes Dialog Screen is visible. This screen will be used to communicate major changes made to BOLDOC or urgent messages regarding BOLDOC usage.  Release Notes can also be accessed from the Help Menu. Similarly, there are several other helpful features:

Part Description
File Menu

Provides access to same options available on the Toolbar:  Edit, View, Preview, and Print.  You may also Exit BOLDOC.

File Restore

Provides the user a way to default the Bill-of-Lading information back to the original data submitted from SOURCE.  You may only restore a document in Editing or Edit/Ready status. This will place the document back into a status of Confirmed. 

View Menu

Provides another method to Refresh BOLDOC data.  Using Refresh will allow any orders submitted from SOURCE after BOLDOC was opened to display. 

BOLDOC allows the use of function keys:

Function Shortcut Keys Description
Edit [CTRL + E]

Allows editing of the selected document.
Preview [CTRL + P]

Preview of BOL document. BOL number not assigned.

Note: A document can only be previewed if editing is complete and all data is resolved.
Generate [CTRL + G]

Launches the print process for finalized BOL document.

Note: This option assigns the next available BOL sequence number to the document. A document can only be generated if editing is complete and all data is resolved.
Delete [CTRL + D]

Delete the data for the selected row.
Restore

 

Restores the order data for the selected row to how it came over from SOURCE. All BOLDOC edits removed.
Exit [CTRL + X]

Exit BOLDOC.

Note: If having trouble connecting to BOLDOC, use the Tools Menu option called Check Connection. Selecting this will check the server connection and notify you of any issues.
