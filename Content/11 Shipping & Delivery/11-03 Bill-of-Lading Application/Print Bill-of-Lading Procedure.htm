Print Bill-of-Lading Procedure

Bills-of-Lading may be printed for documents which have been edited and all data has been resolved. These documents will be in Edit/Ready status on the Edit Selection Screen.  Documents in Edit/Ready status have all action options available to them. Before a document is edited, the only options available are Edit (edit selected document) and Refresh (refreshes edit selection view to include newly submitted documents.)

Step Action

1

Select Print in the toolbar of the Edit Selection Screen.

2

Select YES when prompted.

Note: This option is not reversible. Bills-of-Lading are "pre-numbered" forms the document cannot be edited once a final version is printed. If changes are required after you print, you will need to delete the request and resubmit in SOURCE.

3

Select the number of copies to print.  BOLDOC defaults to three copies:

  • Store Copy (to file with delivery paperwork)
  • Carrier Copy
  • Customer Copy

Note: To print an SDS for a single product, use the option available from the Edit Selection Screen.

4

BOLDOC now assigns the Bill-of-Lading Number.  The Bill-of-Lading copies and any other requested documents will generate.  Select File Print.

Note: Once a Bill-of-Lading has been generated the Edit and Preview options are no longer available and the document is in Printed status.  However, you may View the final Bill-of-Lading and may still continue to print copies.
