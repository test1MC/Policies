Create Carrier Record Procedure

Step Action

1

From POS desktop, select Store -> Manager Functions -> File Maintenance -> Carriers -> Add.

2

Enter the 4 position alphanumeric SCAC and Carrier Description:

If Then

Setting up the Store Truck (employee transport)

Create one Carrier Record regardless of the number of store trucks.  The SCAC Code should be XXXX and the Description should be Store Truck.

Setting up the Will Call record

The SCAC Code should be WCAL and the Description should be Will Call.

Setting a record for a new 3rd party carrier (not preloaded into the system)

Ask the carrier for their SCAC Code.

3

Change the Status: Active value to YES.

4

Select a Carrier Type.

Note: This will let employees know the type of carrier selected.

5

Enter a Preferred Status of YES.

Note: Completing this field will cause this carrier to display at the top of the search list, which will make selecting the appropriate carrier easier.

6

Complete the Shipping Terms Field. This defaults the Shipping Terms when this carrier is selected for a BOL.  If necessary, the Shipping Terms can be edited within BOLDOC.  Select Prepaid for the Store Truck and Will Call.

7

Press ACCEPT [F1] to save the new record.
