BOLDOC Carrier Maintenance Overview

In order to submit Customer Order information to BOLDOC, the user will need to specify a Carrier. Several hundred carriers will be preloaded into the system in non-preferred active status. A listing of these preloaded carriers may be viewed in the system from either the Display by SCAC Code or Display by Carrier Name option on the Carrier Maintenance Menu.  Managers are responsible for Carrier Maintenance. The Carrier Maintenance function is used to add and change Carrier information. And, it may be used to specify "preferred" carriers, making searches for carriers more user-friendly. Store Managers should be aware of the following Carrier Maintenance they should perform:

  • A one-time setup must be performed to add the Store Truck (employee transport) and Will Call as carriers. Managers may also need to add a carrier record for a local common carrier, which has not been loaded into the system.
  • The records for CTS, Yellow Freight, and any other oft used common carriers should be marked as “preferred” carriers and Shipment Terms should be added to their records.
