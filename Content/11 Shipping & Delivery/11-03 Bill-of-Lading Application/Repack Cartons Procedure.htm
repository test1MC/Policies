Repack Cartons Procedure

Once all the issues are resolved with items for which a BOL Code could not be automatically assigned, you may need to create Repack Cartons.  Look to the far right column entitled Remaining Loose Units.  If you have any loose units in your shipment, you must specify repack carton information. If you are creating a Bill-of-Lading for a Customer Delivery, use the pick ticket as a guide to creating Repack Cartons.

Step Action

1

Select New in the Repack Cartons Window.

Note: The Available Items Window displays all the available shipped items by BOL Code, their loose Units, and the BOL Code Descriptions of all the BOL Codes assigned to products with loose units displays at the bottom of the page.  This display is here to help determine which products may be packaged with other products since some products cannot be packaged with others based on their BOL Code.

2

Use the action buttons to add or remove items.

Note: In future releases, restrictions will be imposed on the user regarding which items may be packaged together based on regulatory requirements.  For now, repack cartons should be created by users familiar with packaging regulations. The Repack Carton Window displays information for those items that have been identified by the user as packaged in the repack carton being created or edited.

3

Enter the actual value as the Total Weight Override for the repack carton regardless of whether or not the value is the same as the estimated weight.

4

Select OK to save the carton information.

Note: You must create and save each carton individually and repeat this process until you have "repacked" all the loose units in cartons.

5

Select on the key for a carton to view the items contained in the Repack Carton.

Note: All created Repack Cartons will display in the Repack Carton Window.
