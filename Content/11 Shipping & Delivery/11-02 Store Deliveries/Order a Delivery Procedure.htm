Order a Delivery Procedure

Step

Action

1

From POS desktop, select Store -> Customer Order Entry -> Add Order

2

If required, enter a Customer PO# and/or Customer Order Job #.

3

Select Delivery as the Customer Service Option.

4

Indicate if none, some, or all of the order lines will be delivered.

5

Select YES when asked if the order is to be delivered.

6

Enter the Customer Account Number or use LOOKUP [F3].

7

Enter the ShipTo Address for the order. If the ShipTo is the same as the BillTo, press ACCEPT [F1] and answer Y to the prompt to use the BillTo Address as the ShipTo.

Note: If the customer is exempt from tax and your store is not using Tax Record Cards, verify that the Tax Exception Flag has been set or press [Shift] + TAX? [F11]

8

Press ACCEPT [F1] to move to the Order Detail.

Note: If the customer is tax exempt and the store uses Tax Record Cards, select one to apply to the order.

9

Enter the products and quantities requested by the customer. Press ACCEPT [F1].

10

Special Instructions may be added, if necessary.

11

Select None when prompted to indicate which order lines are ready for delivery. This should not be completed until the order has been picked.

12

If needed, print a Picking Ticket from Print Options.

13

Pick and pack the order.

14

From POS desktop, select Store -> Customer Order Entry -> Update/Review Order.

15

Use Order Lookup to find the order. Press ACCEPT [F1] to view a listing of all orders matching your query.

16

Press ENTER to select the order.

17

Update Header Information, products, and quantities, if required.

18

Indicate if some or all of the order lines are ready to be delivered.

19

There are three delivery methods that can be utilized: Please see Delivery Order Method Procedure.
