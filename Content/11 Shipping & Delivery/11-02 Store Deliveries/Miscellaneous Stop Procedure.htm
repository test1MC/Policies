Miscellaneous Stop Procedure

Step

Action

1

From POS desktop, select Store -> Customer Order Entry -> Submit Order for Delivery -> Add Miscellaneous Pickup/Delivery

2

Select a Customer Account, if applicable. 

3

Enter a Delivery Date.

Note: “Anytime” is the default for Miscellaneous stops.

4

Enter the Pick-Up Address or the Delivery Address, but not both.

Note: If something needs to be picked-up and delivered, create two separate miscellaneous stops.

5

Use Special Instructions to inform the driver on the specifics of the pick-up. Enter information about total weight, hazardous information, and the nature of the pick-up because there is no line item selection or order type.
