Delivery Order Types Overview

In addition to being scheduled at the end of a delivery order, deliveries can also be scheduled by submitting an order from the Customer Order Entry Menu. From POS desktop, select Store -> Customer Order Entry -> Submit Order for Delivery -> Submit Delivery Order Request. Select Order Lines to be delivered.

Order lines are fed from various functions and display as Order Types, including:

Part

Function

Store

Delivery Order to customer. Lines will only display if they are marked as Ready for Delivery in COE.

IST

Product to be delivered to another SW store location.

ISR P/U

This is a temporary Order Type that will allow stores using Centralized Delivery to schedule a pick-up of IST goods from another SW location.

Repair

This selection allows you to transfer spray equipment dropped off at one store location to another store location for repair. When returning a repaired unit to an originating store, the repair location must schedule the return from the IST.

Return

Pick-up of goods to be returned from a customer site.

Will Call

Pick-up of product at DSC for delivery to requesting store or customer. When delivering the Will Call to the customer location, the store must create both a Will Call Purchase Order and a Customer Order for delivery. Both need to be scheduled for delivery.

Vendor P/U

Allows you to schedule a pick-up of goods from a vendor based on an External Purchase Order.

Once selected for delivery, a line item will disappear from this listing. Lines not scheduled for delivery within seven days will also be removed from the listing. Be sure to select each line.

Complete the address for your order based on the Order Type selected:

Part

Function

Store, IST, Repair

The Store, IST, and Repair Order Types will import an address from either the customer order or the store address file. The address may be edited if needed.

ISR P/U, Return, Vendor P/U

The ISR P/U, Return, and Vendor P/U Order Types will all prompt you to enter a Pick-Up Address.

Will Call

You will need to select from the DSC/Plant List for pick-up when selecting the Will Call Order Type.

If required, geocode the entered address. If the address does not pass GPS validation, you will be required to correct the address before you are able to submit the order to Dispatch. If this occurs:

  • Choose to enter the address again or manually locate the address on a map. If you select to locate the address on a map:
    • Read the geocoding instructions and press ENTER to launch the browser.
    • The browser will open the map to the general area of the entered zip code. Use the + and - keys to zoom to the location of the delivery. See Fig A.

    • You can then narrow to individual plots of land, and clicking on a plot will retrieve its address. To select a location, select Mark Location. See Fig. B.

    • When you see the message that your location has been successfully marked, close the browser.
