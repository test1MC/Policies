Delivery Order Procedure

Process or cancel a Delivery Order using a Store Vehicle or Cartage Service, a Delivery Order using Multiple Delivery Methods, or a Delivery Order using SW Delivery Service

Step

Action

Delivery Process
1
If Then
Delivery Order Using a Store Vehicle or Cartage Service Proceed to Step 2.
Delivery Order Using Multiple Delivery Methods If entering an IST or Customer order, select N to the prompts, “Are lines on this Order ready for Delivery?” or, “Schedule a delivery for this transaction.” Press ACCEPT [F1]. Repeat for all transactions being combined.
Delivery Order Using SW Delivery Service

Proceed to Step 2.

2

From POS desktop, select Store -> Customer Order Entry -> Delivery Menu -> Add Delivery -> Enter the transaction lookup criteria (narrow search by transaction type, date range, and sort preference). Mark select order(s) with the ENTER key (orders with the same account can be sent to the same location on one delivery). Press ACCEPT [F1].

3
If Then
Delivery Order Using a Store Vehicle or Cartage Service

Select the XXXX Store Truck as the carrier from the SCAC Code List.

Note: If your truck is in use, scroll and select the Carrier name of the service.
Delivery Order Using Multiple Delivery Methods Select a Carrier. If using a Store Truck serviced by the SW Delivery Service, select a delivery priority.
Delivery Order Using SW Delivery Service

Select XXXX Store Truck as the carrier from the SCAC Code List.

Note: Rename any carrier named Store Truck that you added in the past. Leave only the XXXX Store Truck, which sends orders to the delivery service.

4

Select the Delivery Date -> Select Priority or Delivery Time (Available Order Times will show based on the Delivery Date you have selected) -> Select YES to submit your order to Dispatch for scheduling.

Note: POS will save delivery locations used for customer accounts. You may select a previously used address from the list returned to the screen or press CANCEL [F5] to use the address entered on the Customer Order.

5

Confirm the Ship Quantity.

Note: If the Ship Quantity is less than what is available, enter only the amount being shipped. See Additional Steps below for the backordered products procedure. The order with backordered products will remain in Customer Order Entry.
6
If Then
Delivery Order Using a Store Vehicle or Cartage Service Proceed to Step 7.
Delivery Order Using Multiple Delivery Methods

Select Y to submit a Bill-of-Lading Request to BOLDOC and/or schedule delivery routing (SW delivery service only). 
Delivery Order Using SW Delivery Service Select Y to submit the order for Scheduling/Routing and to open BOLDOC.

7

Do you want to create a Bill-of-Lading?

If

Then

Yes

Select Y to submit a Bill-of-Lading Request to BOLDOC. Press ACCEPT [F1] to proceed.

No

Select N to create a Manual Bill-of-Lading or if your Third Party Carrier is providing one for you.

Note: This will communicate your transaction information to HEARS. If an error message appears during any of the transaction submissions, note the message and call the helpdesk. A BOL is required for all orders.

8

Select Y to open HEARS to complete the BOL.

9

Process the Sales Transaction.

Additional Steps: Backordered Products

1

Once the backordered products have arrived, from POS desktop, select Store -> Customer Order Entry -> Update/Review Orders. Search for the backordered customer order.

2

Select Y to the prompt, “Are lines on this Order ready for Delivery?” Press ACCEPT [F1].

3

Select the Store Truck as the carrier from the SCAC Code List.

4

Select the Delivery Date. Instructions are at the bottom of the screen.

5

Confirm the Ship Quantity of the backordered products.

6
If Then
Delivery Order Using a Store Vehicle or Cartage Service Select Y to submit a Bill-of-Lading Request to BOLDOC. Press ACCEPT [F1].
Delivery Order Using Multiple Delivery Methods

Select Y to submit a Bill-of-Lading Request to BOLDOC and/or schedule delivery routing (SW delivery service only).  Press ACCEPT [F1].
Delivery Order Using SW Delivery Service Select Y to submit the order for Scheduling/Routing and a Bill-of-Lading Request to BOLDOC. Press ACCEPT [F1].

Note: This will communicate your transaction information to HEARS and/or SW Delivery Services. If an error message appears during any of the transaction submissions, note the on-screen message and call the helpdesk.

7

Select Y to open HEARS to complete the Bill-of-Lading.

8

If necessary, print the Confirmation of Delivery tickets.

9

Process the Sales Transaction, select Shipped Lines Not Billed.

Additional Steps: Canceling Delivery

1

From POS desktop, select Store -> Customer Order Entry  -> Delivery Menu  -> Cancel Delivery.

2

Select COE from the Order Types Menu. Press ACCEPT [F1].

3

Highlight the order to be cancelled and press ACCEPT [F1].

4

Enter Y to cancel the order. Contact Dispatch to confirm the shipment cancellation.
