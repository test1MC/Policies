Employee Delivery Procedure

TAG Stores participating in the Sherwin-Williams Delivery Service ONLY

Use a carrier code when an employee is taking an IST order to another store or making a store delivery using their personal vehicle (rather than the hub).

Step

Action

1

From POS desktop, select Store -> Manager Functions -> File Maintenance -> Carriers.

2

Select Add.

3

Enter EMPL as the SCAC code and use Store Employee as the description.

4

Complete the necessary information for the BOL header.

5

When scheduling a delivery of an IST or standard order, EMPL – Store Employee will be available as a carrier option on the Shipment Detail screen.

Note: Change the BOL to Yes (it does not default to yes as it does for store truck). A BOL will be generated with the proper header information.
