Locate State Tax Bulletin Information Procedure

For questions regarding taxability and potential exemptions, stores should first consult the state Sales and Use Tax Bulletin for the state in which the transfer of goods will occur.  To look up the State Tax Bulletins and State Tax Forms for each state, Canada, and Puerto Rico, reference Locate State Tax Bulletin Information Procedure.

Many states allow the use of the Uniform Sales & Use Tax Exemption Certificate Multi-jurisdiction form for resale ONLY.

Step Action
1 Find the Sales & Use Multi-Jurisdiction – Resale Only form in the Forms Repository.
2

Search form name or keyword “jurisdiction” in the Form Name field. All available forms containing the search word will appear below in the form description field.
3

Please include the following information on ALL forms listed below prior to sending:

  • Store Number
  • Customer Account Number
  • Job Number
