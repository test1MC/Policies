Assume Store Liability Procedure

Step Action

1

Run a basic correction transaction for tax collection as detailed in Collect Tax Procedure.

Always select the payment method of cash.

2

Process a Miscellaneous Cash Disbursement for the amount of the tax, which should have been collected on the original sale.

Code the disbursement to G/L Code "SALES TAX ADJUSTMENT" (7618-001). 

Include the COMMENT [F9] - CASH ACCOUNT TAX CORRECTION. Press ACCEPT [F1].

Enter employee password and press ENTER.

3

Staple a copy of the original cash exempt transaction and a copy of the tax correction together and retain in the tax exemption certificate file folder for audit purposes.
