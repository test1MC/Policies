Apply Use Tax Procedure

Use tax reporting is an automatic process. Use tax accuracy depends totally on the store properly identifying use tax products. Automated use tax reporting will help to simplify recording and reduce the time it takes to report transactions subject to use tax.

Step

Action

1

Press SHIFT + TAX? [F11] to activate the Tax Exception Flag. 

2

Ensure that the correct installation address displays in the shipping address area. The use tax jurisdiction for the transaction is that of the installation site - this information will be used to determine which tax rate should apply.

3

Press ACCEPT [F1] to move to the order detail.

4

Apply use tax by product.  Use tax should be applied to all product (defined as subject to use tax by state) being physically installed at the installation site except for labor.  Move to the product line where use tax should be applied and press SHIFT + TAX? [F11]. Select Use Tax from the Tax Exception Menu. 

Note:  The letter U displays in the T(ax) column to confirm that use tax will be calculated for that line item.
