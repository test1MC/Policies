Use Tax in Neighboring States Procedure

If Then
In a non-use tax state and installing in a use tax state POS will calculate the use tax.
In a use tax state and installing in a non-use tax state Validate that the appropriate sales tax is properly applied to the transaction.
The store is installing merchandise in a neighboring state and use tax applies for the merchandise Follow the procedure below.

 Note: In all cases, ensure that the installation address (including zip code) is correctly noted in the ShipTo Address. 

Step

Action

1

Refer to the Use Tax Summary By State chart to verify that use tax applies in the neighboring state.

2

When bidding on the job, make sure to account for use tax. Remember that the store will be charged for use tax based on the cost of the material.

3

Be sure to include a ShipTo Address, including the correct zip code of where the product will be installed. The zip code will be used by Taxware to determine the sales/use tax rate for the transaction.

4

Process the transaction following use tax processing instructions.

5

Was the use tax transaction processed using the above procedure?

If Then
Yes Stop. There is no need to submit the Floorcovering Use Tax Report.
No

The store must submit a completed Floorcovering Use Tax Report to Corporate Tax each month. Proceed to step 6.

Note: The address is noted at the bottom of the report.
6 Find the Floorcovering Use Tax Report in the Forms Repository. Search form name or keyword “use tax” in the Form Name field. All available forms containing the search word will appear below in the form description field.
