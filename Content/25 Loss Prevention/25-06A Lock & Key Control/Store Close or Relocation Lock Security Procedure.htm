Store Close or Relocation Lock Security Procedure

Follow these steps to handle a Store Closing/ Relocating based on the service provider for the store's locking system.

Step

Action

Bass Security

1

Contact the District Manager, City Manager, or District Coordinator.

Note: The district personnel will request that Bass Security supply them with construction cores and keys to be used when the location is returned to the landlord. Bass Security will ship the store an interchangeable construction core that is compatible with the existing one.

2

The last day the store is open, change the SW Medeco locks for the construction cores provided by Bass Security.

3

Return the Medeco cores and keys to Bass Security in the provided container within seven days.

Midwest Security Inc

1

Contact Loss Prevention.
