Lost & Unreturned Keys Procedure

Follow these steps to handle a Lost/Unreturned/Broken Key based on the service provider for the store's locking system.

Step

Action

Bass Security

1

Contact Bass Security (1-866-956-1815, Option #3) to request new keys and re-core.

2

Provide the store number and the unique code imprinted on the key.

Note: All keys issued to the same location have the same code.

3

After receiving the new cores, sets of keys, and return kit from Bass Security, use the control key to remove the existing core and replace it with the new one.

4

Obtain issued keys from all keyholders and re-issue new ones.

5

Return the old cores and all corresponding keys to Bass Security using the return kit.
