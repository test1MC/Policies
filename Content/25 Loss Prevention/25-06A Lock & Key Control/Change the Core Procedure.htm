Change the Core Procedure

Follow these steps to Change the Core based on the service provider for the store's locking system.

Step

Action

Bass Security

1

Turn the control key in the existing core a quarter clockwise.

Note: The existing core will slide out.

2

Install the new core using its control key.

3

Turn the new control key a quarter counter-clockwise to lock the new core in place.

Midwest Security Inc

1

Contact Loss Prevention to report the occurrence and to obtain replacement lock and key hardware.

Note: Loss Prevention will contact Midwest Security Inc to ship replacement cores, keys, and a control key in order to change locks. Equipment will be shipped to the branch overnight.

2

Remove the old cores and install the new cores with the control key.

Note: Detailed instructions how to change cores will be provided with the replacement hardware.

3

Ship the original cores, keys, and the control key back to Midwest Security Inc. Packaging and shipping slip to return the old cores, keys, and control key will be included in the package with the new cores and keys. 

4

Maintain an updated and complete Internal Security Control Record (located in the Forms Repository) listing current employees that hold branch keys, the safe combination, and/ or alarm codes.
