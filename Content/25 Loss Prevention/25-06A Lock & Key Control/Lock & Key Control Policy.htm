Lock & Key Control Policy

  1. All locks and special locking devices must be kept in good working order.
  2. Key locks and cores must be changed immediately whenever a key is lost.
  3. If a key is lost or stolen, a Loss, Offense and Incident Report (LOI) must be completed. For more information see Loss, Offense, & Incident Policy
  4. When a keyholder is transferred, suspended, dismissed, or resigns:
    1. If the Medeco Key Mark Interchangeable Locking System is being used, there is no need to re-core the lock if the keyholder has returned the key.
    2. In some locations with night deliveries where the Medeco Key Mark Interchangeable Locking System is not utilized, the outer door locks must be changed immediately.  Outer door locks with cylinder door inserts should have a protective steel ring and a bolt throw shield installed. The ring will prevent the removal of the cylinder from the exterior door. 
  5. Locations using the Medeco Key Mark Interchangeable Locking System should be aware of the following policies:
    1. Stores will be issued a control key, which will be used to change the core when necessary. This control key should be kept in the safe, locking file, cabinet, cash drawer, or other secure area within the store. Contact Loss Prevention for more information on correct storage of secure items.
    2. Keys are property of The Sherwin-Williams Company. If a key is lost or the keyholders fail to return their key upon leaving their position with the company, new cores and new sets of keys will need to be issued within twenty-four hours.
  6. If a former employee is unable to return the key within twenty-four hours, the Manager must use assess the situation and determine whether or not to rekey the lock.
  7. If an issue exists which may threaten the security of Sherwin-Williams employees and property, the Manager should follow Replace Emergency Lock Procedure
  8. Retail locations should have a maximum of two key access points per location. All other access doors, which may remain unlocked during business hours, should be fitted with thumb latches on the inside and blanked out on the outside.
  9. Failure to return used cores and all corresponding keys to any Sherwin utilized security and locking system service provider in a timely manner may result in an audit infraction. Internal Audit will verify that all keys ordered for a location are accounted for in the annual audit.
  10. Automotive branches should be aware of the following additional policies:
    1. All key/core change requests must be submitted and approved by Divisional Loss Prevention Management.
    2. All core control keys are to be returned to Midwest Security and not to be retained in the branch.
    3. Only branch managers are authorized to perform lock changes.
    4. Automotive branch locations are not required to have safe combinations changed when a keyholder departs.
