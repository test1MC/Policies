Alarm System Policy

  1. Each Manager should be assigned a confidential passcode.
  2. Passcodes may not be shared under any circumstances.
  3. Only the Store Manager and Loss Prevention are allowed to add or delete codes, as well as modify the emergency call list for their location.
  4. If a Manager leaves the store due to transfer, termination, or steps down to a non-management position, the passcode must be deleted immediately with the alarm vendor.
  5. When using a borrowed Manager, contact the alarm vendor 24 hours in advance to receive a temporary passcode.
  6. If at any point it is suspected that the passcode has been compromised, contact the alarm vendor immediately to have the old passcode canceled and a new passcode issued.
