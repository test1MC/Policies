Door & Window Security Policy

  1. All warehouse windows must be locked at all times.
  2. All windows and doors must be kept in good repair.
  3. Freight doors must remain locked unless they are being used for receiving or shipping merchandise.
  4. A signaling device should be installed at the rear entrance for deliveries. Install a buzzer, for example.
  5. The front door should have a door-opening signal. The building must be secured anytime it is unattended.
