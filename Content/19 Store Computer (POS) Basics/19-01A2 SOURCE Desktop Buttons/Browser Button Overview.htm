Browser Button Overview

When selecting the Browser Button, input an employee number and password. This will connect to the SOURCE Intranet System and provide access to web browsing. In the SOURCE browser, employees have access to:

  • Information Library
  • Policies and Procedures Manual
  • Product Data Sheets
  • Forms Repository
  • Human Resources Options
  • Links
  • SDS Files
  • Sales Tools
  • SOURCE Messages

Note: The company has a vested interest in preventing the browsing of sites which could be destructive to the POS or sites which could be considered offensive, exposing the company to potential legal action. With the exception of malicious sites, high bandwidth sites (which may prevent priority transactions such as credit card validation, customer lookup, etc.), and sites that are not operational on the Mozilla browser, web browsing is now open on the POS. Review SOURCE Intranet System Overview for more information on the SOURCE Intranet System.
