Inquiry Button Overview

Reports for store specific information can be generated using The Store Computer System. This is done using the Inquiry Button. Inquiry is a “hot key” report function, which means that it can be pressed at any time to access store information. Browse anything from Credit Card Authorizations to Price Record Cards.

A Store Manager Password will be required to review and print certain reports. For information on password maintenance, see System Control.

Note: Review Inquiry [F4] Overview for information on utilizing all of the options available in Inquiry.
