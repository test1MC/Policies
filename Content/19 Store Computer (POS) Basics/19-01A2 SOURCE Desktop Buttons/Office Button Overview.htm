Office Button Overview

The Libre Office suite is available for additional basic word processing, spreadsheet, and presentation needs not met by other resources and applications (such as Store Reports, Forms Repository, HEARS, etc.). Libre Office functions very similarly to Microsoft Office.

In addition to creating new documents, LibreOffice allows employees to open and to edit preexisting templates for presentations, spreadsheets, and forms, many of which are available in the Forms Repository.

Note: When selecting LibreOffice, there will be a prompt to enter a password for Manager Access. All documents created while in Manager Access are confidential which means they cannot be shared. Use this mode for documents to keep private, such as Employee Performance Appraisals. To create Sharable Documents, select No for Manager Access.
