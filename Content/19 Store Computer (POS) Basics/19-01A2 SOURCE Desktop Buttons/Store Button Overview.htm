Store Button Overview

The Store Button provides access to the following options:

  • Point of Sale
  • End of Day
  • Manager Functions
  • Help Desk
  • Miscellaneous
  • Attendance and Time Entry
  • Customer Order Entry
  • Merchandise Management
  • Color Menu
  • Work Order System
  • Floor Covering Menu (if installed)

Note: Review Chapter 08-01A Sales Transactions for detailed POS Processing & Store Instructions.
