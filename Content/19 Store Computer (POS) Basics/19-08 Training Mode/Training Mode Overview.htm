Training Mode Overview

Training Mode allows new employees to practice entering transactions on the system. These transactions will not print and will not be polled by the central computer as a valid transaction. Store Management must enter the system manager password to activate training mode and review any transaction run during training.

Note: Training mode is only available for POS Transactions on the Sale/Payment Menu and not available for Customer Order Entry, MMS, Color Functions, or Floorcovering.
