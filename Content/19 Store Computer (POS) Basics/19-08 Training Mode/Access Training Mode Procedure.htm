Access Training Mode Procedure

Action Process

Entering Training Mode

From POS desktop, select Store -> Point of Sale -> Sale/Payment Menu -> Enter Training Mode -> Enter System Manager Password -> Enter Employee Number and press Enter.

Note: The screen will display in different colors when training is on. For example, the Sale/Payment and Special Menus will display in light blue.

Exiting Training Mode

Press CANCEL [F5] twice to exit the INQUIRY [F4] screen -> Select Sale/Payment Menu -> Exit Training Mode.

Note: Exit Training Mode will not display on the Sale/Payment Menu until after Enter Training Mode is selected.
