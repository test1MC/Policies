Review Training Mode Transactions Procedure

Reviewing in Training Mode is only for Store Management.

Step Action

1

From POS desktop, select Store -> Point of Sale -> Sale/Payment Menu -> Enter Training Mode -> Enter System Manager Password -> Enter Employee Number and press Enter.

2

Select INQUIRY [F4] to review training transaction information.

3

Enter the Date Range.

4

Press Enter until you reach Voids: Enter 4-Training Only, press ACCEPT [F1]. A list of all training transactions for the date range will return to the screen.

5

Press CANCEL [F5] one time.

6

Choose between the following selections or press CANCEL [F5] twice to exit the INQUIRY [F4] screen.

7

Select Sale/Payment Menu -> Exit Training Mode.
