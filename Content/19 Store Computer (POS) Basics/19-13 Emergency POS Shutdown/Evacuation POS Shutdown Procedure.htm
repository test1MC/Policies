Evacuation POS Shutdown Procedure

Step

Action

1

Run a close on all terminals but do not run End of Day.

2

Once the close has printed, start a backup on the Main POS System -> Cancel [F5] terminals back to Source Desktop Screen -> Select Source and choose backup -> Press Accept to run backup.

Note: Backup should take anywhere from 25-30 minutes to complete.

3

Once backup has completed, Cancel [F5] terminals back to the Source Desktop Screen -> Press Ctrl- Alt- Delete -> Select Shutdown and then OK to proceed.

4

Shutdown and power off all computers (Main POS, 2nd POS, FEX PC, Manager’s, etc.). Power off and unplug all printers, fax machines, power surges, etc.

5

When all systems have powered down, cover them.

6

Take one of the servers, preferably the manager’s, and secure it while transporting it.

7

Go into Manager’s Office, locate and unplug the Satellite box (labeled PES 5000, HUGHES 5000 or HNS 7700S).
