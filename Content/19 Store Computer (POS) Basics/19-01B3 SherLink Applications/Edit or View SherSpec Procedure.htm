Edit or View SherSpec Procedure

Step Action

1

From POS desktop, select Browser -> Access SherLink homepage -> Presentation Tools -> SherSpec.

2

Select View or Edit a Specification.

3

Enter appropriate parameters to search Specifications.

4

From listing, select the Spec #.

5

Select Duplicate Spec.

6

Choose a customer from database or enter SW Account Number.

7

Review Customer Details and select or add Customer Contact.

8

Add additional products or edit existing selections, if necessary.

9

Update Surface Preparation, if necessary.

10

Select Preview Specification.

11

Edit Options.
