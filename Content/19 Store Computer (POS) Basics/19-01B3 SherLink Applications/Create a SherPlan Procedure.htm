Create a SherPlan Procedure

Step Action

1

From POS desktop, select Browser -> Access SherLink homepage -> Territory Management -> SherPlan.

2

Select Create/Edit Plan.

3

Complete each step in the SherPlan Task Bar:

Task Description

This step allows reps to review their performance from the previous year in order to build a new plan. This information automatically updates and will display current data as well. This step includes sales information, key accounts, opportunity accounts, new accounts, existing accounts, and SherCall entries. 

This step asks reps to review their TCSR-CAC Report and describe opportunities for growth and potential weaknesses in their markets.

This step allows reps to review the four sales buckets: key accounts, opportunity accounts, new accounts, and existing accounts. Then, answer questions regarding how to improve in the upcoming year.

This step allows reps to review information from their previous year’s SherPlan and visit competitors in order to gather information about them.

This step allows reps to review lost customers or non-repeating jobs and anticipate how to improve sales.

This step asks reps to refer to their Dashboard End of Year Preview, TCSR Dashboard Report, or MicroStrategy Dashboard in order to determine which accounts qualify as key accounts in the upcoming year.

This step asks reps to refer to their Dashboard End of Year Preview, TCSR Dashboard Report, or MicroStrategy Dashboard in order to determine which accounts qualify as opportunity accounts in the upcoming year.

This step asks reps to refer to their Dashboard End of Year Preview, TCSR-CAC Dashboard Report in order to see the total sales resulting from new accounts and how to improve growth in this group.

This step asks reps to refer to their Dashboard End of Year Preview, TCSR-CAC Dashboard Report in order to see the number of existing accounts and how to improve sales in this group.

This step allows reps to review their gap analysis of these buckets and summarize their observations.

Note: Once a step is completed, select Back to Plan or Other Steps in the top right-hand corner to continue working. The completed sections will appear in green with a check next to them.

4

Once complete, select Print Plan or Print Goals.

Note: As the year progresses, reps should refer back to their plan and make any necessary adjustments.
