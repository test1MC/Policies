Create SherSpec Procedure

Step

Action

1

From POS desktop, select Browser -> Access SherLink homepage -> Presentation Tools -> SherSpec.

2

Select Create a New Specification .

3

Enter the customer’s SW Account Number and press the Customer Lookup icon.

Note: If necessary, select Add New Customer to include a new customer and enter the required information.

4

Review Customer Details and select Customer Contact.

5

Enter Project Details and select Next Step.

6

Select the proper Schedule, Substrate, and Label, or choose Other and provide an explanation.

7

Enter Products (using Sales/Rex #) or use the Product Lookup.

8

Add Surface Preparation, if necessary.

9

Select additional documents to include with the specification. Options are featured in the picture below:

10

Preview Specification.

Note: Several additional options are available at the bottom of the screen: Duplicate Spec, Create Price Quote, and Create Custodian.

11

Choose Print and Delivery Options.
