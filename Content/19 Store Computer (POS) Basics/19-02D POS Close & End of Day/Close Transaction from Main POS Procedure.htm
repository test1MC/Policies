Close Transaction from Main POS Procedure

For stores with more than one POS, the main system should be closed last. You can identify the main system by referring to the Login Screen. The main system is identified as ‘pos####’ (#### = store number). Other POS systems will be identified as clients (clntx####). Contact Store Computer Operations at 216-566-2740 if you cannot identify the main system.

Step Action

1

Rather than having multiple deposits per night, stores will now only have a single deposit for all terminals. Each terminal will still need to be closed, but the last terminal (main POS) closed will provide the amounts needed for the deposit and the combined deposit amount.

If Then

Other terminals are still open

Count the secondary terminals’ drawers. See Drawer Count Procedure.

All terminals have been closed except for the main POS

Count the drawer and continue to Step 2.

Note: Only close the main POS after all additional units as well as the Manager’s Workstation have been closed and returned to LOGIN.

2

From POS desktop, select Store -> Point of Sale -> Select Close from the Sale Payment Menu. All of the information entered at the manager’s office system will be on the Close screen.

3

Follow the prompts to input cash and coin amounts.

Note: This information will automatically populate if the Pre-Close Drawer Count was already completed.

4

The computer system will prompt you to enter the dollar amount of each check. After completing the Close Transaction, the check amount will return to the screen with the difference displaying in the [+/-] Column.

Note: Press CANCEL [F5] to review the check amounts if there is a difference between the transaction check amount and the actual dollar amount of the check. Verify that the validated amount (back of the check) matches the check dollar amount (front of the check). If you choose to continue, enter the reason for the check variance when you see the Check Variance screen.

5

The computer system will automatically balance the credit card transactions processed for the day. It can only balance credit cards run through the POS. Manual forms not run through the POS will not be balanced.

6

Press ACCEPT [F1] after entering all Cash, Checks, and Credit Cards into the Close Worksheet. The over/short amount will return to the screen. Verify the over/short amount by entering the exact dollar amount. Enter the reason for the over/short.

Note: A maximum of three comment lines can be used to describe specifics about a close. Include any specific comments that would help to explain any discrepancies found on the Close transaction.

7

Enter Y to the prompt, "O.K. to continue?" Answer [Y] or [N].

Note: Before answering [Y], review all of the information and make any necessary changes. If errors are discovered after accepting the close amount, the Close transaction will need to be voided.

8

If everything is correct, press ACCEPT [F1]. If not, make any necessary additions or corrections and press ACCEPT [F1].

9

The screen below will prompt for the deposit bag number:

Note: All cash and checks from each terminal will be put into a single deposit slip.

10

The following message will appear: "Is the Deposit Bag Number Correct?" Enter Y [Yes].

Note: Deposit slip will be validated after entering the Deposit Bag Number.

11

When prompted by the following message: "Is the Validated Amount on the Bank Deposit Ticket Correct?" Answer [Y] or [N]. The last terminal will print the combined deposit slip.

Note: As part of the Close, the system will generate a Check Log from the information entered during check transactions.  To review the Check Log: from POS desktop, select Store -> Press Inquiry [F4] -> Daily Store Information -> Check Log.
