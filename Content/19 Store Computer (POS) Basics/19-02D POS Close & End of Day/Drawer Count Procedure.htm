Drawer Count Procedure

Drawers Counts are required to perform a Close transaction. Do not count money on the POS wrap counter in plain view of the store’s windows. Count all currency and checks in the manager’s office. You are required to leave the main POS open until the store is closed so that you can process any last-minute customer transactions. Other POS terminals can be closed prior, if time permits. There are two options for completing the Drawer Count:

  • Count the drawer while processing a Close transaction on the main POS.
  • Enter the Drawer Count from any POS terminal prior to Close. When it is time to process the Close transaction, the drawer count information will automatically populate in the Close transaction count screen.
Step Action

1

From POS desktop, select Store -> Point of Sale -> No Sale -> Count Cash for Store Close. Remove the cash from the drawer and then close the drawer.

2

Select Miscellaneous from the Main Menu.

3

Select Enter Drawer Count from the Miscellaneous Menu.

4

Select the correct terminal number for the drawer that you are counting.

Note: Terminal numbers can be found by referencing any POS transaction screen or printed transaction.

5

Count all of the cash and coin contained in the cash drawer. If there are no over/short issues, the screen below resembles the normal Close transaction screen. Press ACCEPT [F1] when done.

6

All information entered will appear in the Close transaction screen when you close the terminal:

Note: This screen will ask for the terminal’s drawer currency deposit amount. 

Note: To review deposit bag information: From POS desktop, select Store -> Inquiry [F4] -> Store Accountability -> Deposit Bag Report.
