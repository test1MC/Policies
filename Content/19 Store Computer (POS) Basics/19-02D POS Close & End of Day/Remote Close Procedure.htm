Remote Close Procedure

This feature is only to be used if the point of sale terminal is not operating properly or if instructed by the HELP DESK. This procedure can be completed at the Manager's Office Terminal.

Step Action

1

From POS desktop, select Store -> Miscellaneous -> Remote Close.

2

Read the Reminder message:

"This selection should only be used when instructed by the HELP DESK. It is used when the Point-of-Sale Monitor or Keyboard are inoperable. If you want to enter the Drawer Count ONLY - press Cancel [F5] and select option 1 (Drawer Count). Otherwise, follow the system prompts to proceed with the Remote Close."

3

Press Enter and begin the Close procedure referenced above.

Note:   After performing the Remote Close do not forget to select End of Day from the Main Menu. End of Day will prepare the store for polling and will activate the printing of the Journal Tape.

Note: This procedure does not take the place of properly closing the point of sale. This procedure should only be followed if instructed by Store Computer Operations Help Desk or reported to the Help Desk the following day.
