Downtime Processing Forms Overview

Below is a listing of forms that should be used in the event of a system malfunction. These forms are found in the Forms Repository on SOURCE.  Search for the form by keyword in the Form Name field. All available forms containing your search word will appear below in the form description field.Print blank copies of these forms and store them with the Manual B-Prefix Purchase Order and Manual POST forms.

Should these forms be lost in a natural disaster, a neighboring store can provide Manual B-Prefix Purchase Orders or Manual POST forms by completing a fixed asset transfer form. The District Office can order Manual B-Prefix Purchase Orders or Manual POST forms until the computer system is properly functioning.

Form Downtime Function

Manual Close Form

(keyword ‘close’)

Process a manual close.

Manual Attendance & Time

(keyword ‘attendance’)

Record attendance.

Note: It is the Store Manager's responsibility to see that each employee completes the Manual Attendance and Time Worksheet.

Manual Bill of Lading

(keyword ‘lading’)

Process a Bill-of-Lading via BOLDOC.
