Set the Target Fund Procedure

District Managers will set the operating fund amount; however, the target fund must be set for each point-of-sale terminal with a cash drawer. The Manager Password is required to perform this function.

Step Action

1

From the POS Desktop -> Main Menu -> Point-of-Sale -> Sale/Payment Menu -> Special Menu -> Set Target Fund.

2

Enter the Manager Password and press Enter.

3

Enter the Target Fund amount and press Enter.

4

File any written approvals from the District Manager in the Target Fund folder in Drawer Three.
