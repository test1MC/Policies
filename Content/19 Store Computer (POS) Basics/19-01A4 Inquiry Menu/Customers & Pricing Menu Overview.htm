Customers & Pricing Menu Overview

Customers and Pricing Menu provides access to:

  • Price Record Cards
    • See View Price Record Cards on Inquiry Menu Procedure.
  • Products and Prices
    • An Inventory Record for each item resides at the store and at Headquarters. This information will assist you with inventory control and ordering.
    • Press Inquiry [F4] -> Products and Prices -> Enter the Product Sales Number or use Lookup [F3] to find the product information ->All pricing options will return to the screen for the product selected -> Press Cancel [F5] to return to the menu.
      Note: No changes can be made to the Product or Prices in this area. This feature is for viewing only. If product information is changed in the HQ database, you may need to pull this information down to your local database. Please use the Product Refresh Procedure.
  • Store Specials
    • Press Inquiry [F4] -> Store Specials -> Enter the System Manager Password -> This will return a query for the Version Number for the Store Specials -> If the Version Number is not known, press [F3] to review all Store Specials entered in the Manager Functions -> After viewing the list, press [F5]. Select Print Immediately to print a copy of the report.
  • Price Schedule Print Log
    • Press Inquiry [F4] -> Select Price Schedule Print Log -> Enter the System Manager Password -> After viewing the list, press [F5]. Select Print Immediately to print a copy of the report.
