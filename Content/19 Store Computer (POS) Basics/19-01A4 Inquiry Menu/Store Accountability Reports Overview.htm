Store Accountability Reports Overview

This menu provides several options to review specific accountability reports.

Report

Function

Over/Short

Verify overages and shortages and to make sure that the Loss Offense and Incident (LOI) Reporting Process is complete. See Store Accountability Menu Reports Procedure.

For more information, see Over/Short at Close Procedure.

Product Quality Report Transactions

View Customer Satisfaction Transaction Reports. See Store Accountability Menu Reports Procedure.

Pricing Overrides

Ensure the accuracy of all Price Record Cards and to monitor employee overrides at the point of sale. See Store Accountability Menu Reports Procedure.

Reprints Log

Review the Reprints Report and challenge any transaction with excessive reprints. See Store Accountability Menu Reports Procedure.
Manual Forms Inquiry See Manual Forms Inquiry Procedure.
Coupon Usage Report

Store managers should review this option to monitor Coupon Usage.

Press Inquiry [F4] -> Store Accountability Menu -> Coupon Usage Report -> Modify the parameters to search for information by date range, employee number, etc. -> Use the generated report to manage the coupon process.

Review Coupons Overview in the for more information on Coupons.
Gift Card Report

Press Inquiry [F4] -> Select Store Accountability Menu -> Gift Card Report -> Enter dates.

Review Gift Card Overview for detailed Gift Card Instructions.

Store Expense Report

 Press Inquiry [F4] -> Store Accountability Menu -> Store Expense Report -> Modify the parameters to search for information by date range, employee number, G/L code, etc. -> Review all expenses from the beginning of the month.
Open/Close Drawer Count Report

Drawer counts by denomination entered during the Open and Close Processes are retained and can be reviewed in this report. A System Manager Password is required.

Press Inquiry [F4] -> Select Store Accountability Menu -> Open/Close Drawer Count Report -> Enter System Manager Password.
