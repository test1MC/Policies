Miscellaneous Menu Overview

Miscellaneous Menu provides access to a range of options and information:

  • Size Codes
    • Press Inquiry [F4] -> Select Size Codes -> Use [Page Up] and [Page Down] to view all Size Codes.
      Note: This feature is for viewing only.
  • Territories
    • Press Inquiry [F4] -> Select Territories -> Enter a portion of the Territory Sales Representative's Name or press [Enter] to return all Sales Representatives in the country.
      Note: This feature is for viewing only.
  • Store Locator
    • There are a number of search categories available to locate stores: by name, by number, by division/area/district, by city, by zip code, by area code.
  • Bulletins
    • Select Bulletins to review any downloaded bulletins at any time during the day. Bulletins display every morning and print with the end of day journal during the selected time period.
  • CAC Codes
    • View current Customer Account Classification (CAC) Codes by selecting this option.
      For more information, see Customer Account Classification (CAC) Codes Overview.
  • SWAT Reprints
    • SWAT Purchase Orders and Customer Order Memos can be reprinted or viewed on the screen at any time prior to completing a transaction. A combined list of all Shipped Direct and Ship to Store Transaction Information is available. Adjustment transactions do not have the same support information, so adjustment information is not available to view or print. When processing a Generalized Inquiry for SWAT transactions, enter customer type one (1) when prompted to enter the account number. A list of SWAT retail transactions including purchase order information will return to the screen along with other type one retail sales.
      For more information, see Sherwin-Williams Answer Team (SWAT) Overview.
  • SPR Review
    • All Stores Purchases Reports (SPR) print at the end of the day and should be used for batching purposes. New SPR Review Reports can be generated by selecting SPR Review. This report lists all ISR (match to incoming IST), expense transactions, and IBAR transactions.
      Review for more information on Stores Purchases Reports.
  • Color Menu
    • See Color Menu Procedure.
  • Reprint Registers Menu
    • Transaction Registers automatically print after processing the Close Transaction.
    • Press Inquiry [F4] -> Select Reprint Registers Menu -> Select a register -> Enter the date.
      Review End of Day Registers Procedure for more information on Transaction Registers.
  • Bin Location Inquiry
    • See Bin Location Inquiry Procedure.
  • Giftcard Balance Inquiry
    • Press Inquiry [F4] -> Select Giftcard Balance Inquiry -> Swipe card or type in the Gift Card Number to retrieve the card’s balance.
