Color Menu Procedure

This section does not apply to Automotive Branches.

The Color Menu provides access to Color Base Checker and Color Formula.

Step

Action

Search Color Base Checker

1

Press Inquiry [F4] -> Select Color Menu -> Select Color Base Checker -> Enter a partial or complete Color Number or Color Name.

2

To quickly move to Wood Classic Colors, enter WC and press [Enter].

Note: Notice that the Recommended Base displays at the bottom of the Color Lookup Screen.

3

Enter a partial or complete Color Name.

Search Color Formula

1

Press Inquiry [F4] -> Select Color Menu -> Select Color Formula.

2

Utilize inquiry capabilities from the Color Types Menu to quickly review a formula.
