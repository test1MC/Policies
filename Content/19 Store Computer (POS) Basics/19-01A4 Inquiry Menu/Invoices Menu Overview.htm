Invoices Menu Overview

Invoices Menu contains three menus:

  • Inquiry Menu
    • See Inquiry Menu Procedure.
  • by Transaction Number
    • By Transaction Number retrieves a transaction by its number. You can then view and print this report. If printed, the transaction will clearly say REPRINT 02/08/94 under the Sherwin-Williams header. The computer system will internally count each time a transaction is reprinted.
  • Suspend/Installer Inquiry
    • See Suspend Inquiry & Installer Inquiry Procedure.
