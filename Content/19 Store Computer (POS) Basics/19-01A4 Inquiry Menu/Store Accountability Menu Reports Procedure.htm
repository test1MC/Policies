Store Accountability Menu Reports Procedure

Access Over/Short, Customer Satisfaction Transaction Reports, Pricing Overrides, or Reprints Log.

Step

Action

1

Press Inquiry [F4] -> Store Accountability Menu -> Select required report (Over/Short, Customer Satisfaction Transaction Reports, Pricing Overrides, Reprints Log) -> Enter the System Manager Password and the date range for the required report.

2

Use the following chart to proceed with the returned report:

To

Then

View

Select View on this screen.

Print a Copy

Select Print Immediately.

Save

Select Send to Reports File for future use.

Exit

Press Cancel [F5].
