Daily Store Information Menu Overview

Daily Store Information contains a variety of reports specific to the workday:

Part

Function

Employee/ Territory Sales

Allows access to total sales for the store. While reviewing the report, notice that sales are listed for each employee and each territory.

Journal Tape

Outlines each transaction that has been run for the selected day(s). If the end of day journal printing is not successful, use this selection to print another journal tape. Print one day at a time because information will not properly reflect each day's transactions.

Check Log

This log is automatically created when the proper information has been entered at the Point of Sale.

For additional information, see Check Log Overview.

Attendance and Time

Used to view Attendance and Time using specific criteria:

  • Detail By Date and Detail By Employee allow the employee to review hours worked (recorded by the employee) and unworked hours to be paid (recorded by management) for each day in the current biweekly period.
  • Summary By Date and Summary By Employee allow the employee to review hours worked and unworked for the current biweekly period. This summary can be reviewed at any time during the current biweekly period.

Draft Log

Review all Store Draft Transactions by transaction type. It is automatically created and will print along with the end of day journal.

For additional information, see Store Drafts Policy.
