Suspend Inquiry & Installer Inquiry Procedure

Suspend Inquiry Menu retrieves Suspended Transactions Not Recalled and Recalled Transactions:

Part

Function
Press Inquiry [F4] -> Select Suspend/Installer Inquiry.

Suspended Transactions Not Recalled

Displays all Suspended Transactions that have not been recalled in the selected date range. Select View on the screen, Print Immediately, or Send to the Reports File for future use.

Recalled Transactions

Displays complete Transaction History for the recalled transaction in the selected date range. The final transaction will be the first transaction shown in each grouping. Each of the other transactions in the grouping will be previous suspends and recalls that resulted in the final transaction. Select View on the screen, Print Immediately, or Send to the Reports File for future use. Each transaction grouping is sectioned off by two lines: Beginning of Recalled Transaction Group and End of Recalled Transaction Group.

Installer Inquiry Menu retrieves Customer Transactions Not Paid to Installer and Completed Customer/Installer Transactions:

Part

Function
Press Inquiry [F4] -> Select Suspend/Installer Inquiry.

Customer Transactions Not Paid to Installer

Review a date range for all outstanding installation charges that have not been paid to installers.

Completed Customer / Installer Transactions

Review a date range for the original customer transaction information tied to an installer payment transaction.
