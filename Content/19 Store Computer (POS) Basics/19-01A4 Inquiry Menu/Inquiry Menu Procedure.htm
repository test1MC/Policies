Inquiry Menu Procedure

Inquiry Menu allows for the entry of specific parameters to search for POS, COE, or MMS Information. Transaction information will be retrieved according to the range specified. Generalized Transaction Inquiries are defaulted to the broadest parameters, but you can change any area to tailor the inquiry as needed.

The Inquiry Menu also provides access to transaction information using the following search fields:

  • By Customer Name and Ship To/Job Name.
  • By Customer Order/Control Number.
  • By Bill To (which can be used even without an account number).

Generalized Inquiry is used when processing reports for SWAT transactions. To begin, enter customer type one (1) when prompted to enter the account number. A list of SWAT retail transactions, including purchase order information, will return to the screen along with other type one retail sales.
