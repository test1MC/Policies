Inquiry [F4] Overview

The store computer system allows you to view or generate reports for store specific information. This is done using the Inquiry Button. Inquiry is a hot key report function, which means it can be pressed at any time to access store specific information. The Inquiry Menu offers many options and they are organized into seven categories:

  • Daily Store Information
  • Invoices
  • Sales History
  • Inventory
  • Store Accountability
  • Customers and Pricing
  • Miscellaneous

Note: Viewing and printing certain reports on the Inquiry Menu requires a Store Manager Password. For information on password maintenance, see System Control.
