Inventory Menu Overview

Inventory Menu provides access to:

  • Return Disposition Log
    • Return Disposition Log is a listing of all customer returns over a specified date range.
  • Unit Movement
    • See Unit Movement Procedure.
  • Wallcovering Book Library
    • Wallcovering Book Library accesses a menu with five options:

      Part

      Function

      Overdue Books

      Review overdue books for each customer.

      Books Checked Out

      Review all books that are currently checked out or search by specific book or customer.

      Book History

      Review book-lending history for all books and customers or search for a specific book or customer.

      Book Library Register

      Review wallcovering books in your wallcovering book library.

      Books Expiring

      		Review books expiring. Enter the expiration date range if it differs from the default range.
