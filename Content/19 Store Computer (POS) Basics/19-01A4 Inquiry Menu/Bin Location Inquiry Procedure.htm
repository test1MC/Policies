Bin Location Inquiry Procedure

For locations with inventory stock utilizing bins, the Bin Location Function is a way to record stock by bin location. Bin location information can then be used to run store inventory reports by bin location. This information can be used to generate Customer Order Pick Lists by bin location.

Step

Action

1

Press Inquiry [F4] -> Select Bin Location Inquiry

Reports may be ordered by either a specific Sales Number or specific Bin Location. The Lookup [F3] function will work on either field for narrowing criteria, or both fields may be left blank in order to review all products in all bins.

2

Reports may be sorted for viewing by Sales Number, Bin Location, Dept-Sub-Cat, Pattern/Blend ID or Batch Number.

3

Use the following chart to proceed with the returned report:

To

Then

View

Select View on this screen.

Print a Copy

Select Print Immediately.

Save

Select Send to Reports File for future use.

Exit

Press Cancel[F5].

Note: Bin location information cannot be used in the annual inventory count process as a way to establish counting locations. For more information, see MMS Report Types Overview.
