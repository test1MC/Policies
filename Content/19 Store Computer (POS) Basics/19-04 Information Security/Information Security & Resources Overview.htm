Information Security & Resources Overview

Information Security (Chinese, French, Italian, Portuguese, Russian, and Spanish) [Information Security documents can be viewed but not printed.]

Sherwin Williams has its own Information Security Team that provides the following services:

  • Administers security programs.
  • Assists in investigation of computer crime.
  • Evaluates and recommends security products and services.
  • Assists Audit with the development of effective monitoring programs and audit trails.
  • Assists in development of business recovery plans.
  • Trains appropriate personnel to initiate and manage decentralized security program.
  • Delegates limited security authority where necessary and practical.
  • Conducts periodic risk assessments and security reviews.

To help keep the system secure, review Information Technology’s Newsletter Series. These releases provide important information that will help to protect employee information.

Note: Some information outlined in these newsletters, such as password maintenance, does not apply to SOURCE. In other words, follow the Policies and Procedures manual where applicable. Reference Employee Password for more information on Employee Passwords on SOURCE.
