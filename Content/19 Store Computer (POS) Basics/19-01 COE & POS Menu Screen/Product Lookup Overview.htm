Product Lookup Overview

To launch Product Lookup, press LOOKUP [F3] on a new line item. The system will return the following lookup options:

Lookup Type

Description

Description

Enter a partial product name (first several letters/numbers) to return a listing of all items within the range.

Product No./Size

Enter a rex number and size code to return all products within the classification. Please see the example listed below.

Sales Number

Enter a partial sales number to review a selection of products that relate to the entry.

Dept., Sub, Cat., Descr

All products in the defined area will be returned to the screen. Press LOOKUP [F3] to display the Sub Dept. within the selected Dept. Press LOOKUP [F3] again to display the Categories within the selected Dept., Sub Dept.

Manufacturer Part Number/Name

After selecting Manufacturer Part Number/Name from the Product Lookup Menu you have several choices:

  • Enter a portion of the Manufacturer Part Number or press Enter to leave the Manufacturer Part Number blank.
  • Select One Manufacturer from the Manufacturer Menu to search for products for a specific
  • Select ALL Manufacturers if you are not sure of the manufacturer after entering a portion of the Manufacturer Part Number.
  • You will be given the option to select the specific manufacturer from the Manufacturer List. Remember that you will only see the first 100 products for the manufacturer.

Note: You are always better off entering a few digits of a part number and a specific manufacturer to limit the number of products returned to the screen.

Wallcovering Books/Patterns
  • The store computer system will validate the wallcovering book name or pattern number against the S.W.A.T. database at Headquarters anytime you look-up a wallcovering book/pattern on the computer system. When prompted for the book name, enter a key word to return every book with that specific key word in the book name. When prompted for a pattern number you must enter the exact pattern number. This will return a list of books where the pattern is available.
  • Sherwin-Williams’ private label wallcovering books always begin with (SW) before the book name. Remember that the (SW) wallcovering books should be considered a first choice when placing wallcovering orders.
  • Wallcovering book names on the S.W.A.T. database do not normally contain articles in the beginning of the book name. For example: A Place In The Country is Place In The Country, The Creative Edge is Creative Edge.
