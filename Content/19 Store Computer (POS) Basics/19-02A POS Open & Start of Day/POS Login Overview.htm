POS Login Overiew

Before beginning any procedures on the Store Computer System, the following opening activities must be performed. You will be required to follow this process every morning except on Tuesdays. On Tuesday mornings, you will be prompted to log in because of system reboots.

Follow these Start of Day procedures before entering any Attendance and Time information on the Manager’s Computer System:

Step Action
1 After pressing Start [Esc] on the main computer system, you will be asked if you want to quit polling. Answer YES and review and respond to each system message (review examples of system messages below).
2 Be patient and do not touch the keyboard until prompted for a username. Enter scs to login.
3 After logging into the computer system, you will be able to properly record Attendance and Time on any terminal.
4 If this sequence is not followed, the system may lock. If this occurs, please contact Store Computer Operations at 216-566-2740
5 Once logged on, System Messages will appear. Use the movement keys to scroll through bulletins, or press F1 [ACCEPT].

Backup Error

Backup Error  Version: 00000

AN ERROR OCCURRED DURING BACKUP

This message will appear If an error occurred during the backup procedures. The Computer System MUST be successfully backed up before proceeding to Point of Sale.

Note: On the Store Computer System, there may be an asterisk moving slowly across the top of the display screen or the message Waiting For Line Connect. Press Start [Esc] to continue. The communication will be interrupted and the backup will begin.

If you encounter any problems, call the Store Computer Operations Help Desk at 216-566-2740.

Bulletins

Review all Bulletins from Headquarters. Take any necessary action prior to opening the Computer System. All Bulletins will print at the end of the day on the manager's office printer. If you would like to view the bulletins press F4 [INQUIRY] then select Bulletins.

Note: When Bulletins are received they will be displayed each morning to make sure that everyone in the store has viewed them.

Waiting For Line Connect

The Waiting For Line Connect message indicates that the network is waiting for Headquarters’ main computer to call for the store's information. If this message appears when opening the store, press the START (ESC) key.

Note: If encountering any problems, call Store Computer Operations at 216-566-2740.
