Operation Keys Overview

Advance through transactions using the Operation Keys. Hold down [Shift] and press the corresponding key to access the function above.

Part Function

F1 [Accept] / [Shift] + F1 [CALC]
  • Accepts the data entered and functions as a selection key.
  • [Shift]: Displays a calculator pad over the screen.

F2 [HELP]
  • Returns an explanation and a list of options for the current field.

F3 [LOOKUP] / [Shift] + F3 [FC COLOR]
  • Returns a listing of valid options to input into the current field. Search for information by following the system prompts in the lower left corner of the screen.
  • [Shift]: Begins the Floorcovering Color process.

F4 [INQUIRY] / [Shift] + F4 [REPORT]
  • Returns a selection menu for a variety of information. Inquiry can be performed at any time while using the Store Computer System.
  • [Shift]: Press this key at any time while running a transaction to retrieve a Reports Menu.

F5 [CANCEL] /

[Shift] + F5 [EXIT]
  • Moves back to the previous input screen. While in POS, it is possible that the transaction will be voided prior to canceling the entire transaction.
  • Returns to the Main Menu. While in POS, it is possible that the transaction will be voided prior to exiting the transaction.

F6 [COLOR]
  • Used to begin the Sher-Color tinting process.

F7 [VIEW] /

[Shift] + F7 [FC AREA REQ]
  • When selected during a transaction, this button returns the input to the customer information field.
  • Navigates to required Floorcovering Fields.

F8 [CLEAR] /

[Shift] + F8 [VOID]
  • Erases the last response entered.
  • Cancels a transaction. The reason for the void will be printed after making a selection from the Reason Code Menu.

F9 [COMMENT] /

[Shift] + F9 [PRC METHOD]
  • While entering customer purchases, specific product information can be entered, including: formula information, name of paint color, etc.
  • Displays the source of the price being returned to the screen. Examples of Price Sources are: Retail, Pro Buy, Price Record Card.

F10 [SUSPEND] /

[Shift] + F10 [RECALL]

The Suspend/ Recall [F10] function allows you to save your place on a Quick Order or Sales Transaction should some other POS activity take priority.

For more information, see Suspend or Recall Procedure.

F11 [DISC] /

[Shift] + F11 [TAX?]
  • Allows a percent, price, or dollar discount for individual line item entries that were not previously discounted from a retail promotion or a Customer's Price Record Card.
  • Returns a tax type menu, including: No tax, Percent of tax, or Dollar amount of tax.

F12 [TOTAL / PROCEED] /

[Shift] + F12 [SUBTL]
  • Completes a transaction and proceeds to the Form of Payment screen. Also—in select applications—this key will enable you to move to the next screen.
  • Deactivated in the system.
