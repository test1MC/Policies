Help Desk Menu Overview

Help Desk provides access to the following menus:

  • Communications
  • Hardware Setup
  • System Configuration
  • Hardware Tests
  • Miscellaneous

Most options will require a Help Desk Password. The charts below outline specific menu options and their functions.

Part Function

Communications:

Enable Diagnostics

Allows Store Computer Operations Help Desk to check for system problems prior to calling for service. The Help Desk will give instructions to Enable Diagnostics. Enabling Diagnostics will not interrupt the use of the computer system, but  DO NOT pick up or use the polling telephone line while the Help Desk is running the diagnostic tests.

Disable Diagnostics

If necessary, the Help Desk will instruct stores to Disable Diagnostics.

Polling

Allows the Help Desk to poll the store computer system in the middle of the day. Normally, the store computer system is polled after hours.

Hardware Setup:

Configure POS Terminals

Used to set up the computer system. Only use these functions with the guidance of Store Computer Operations Help Desk.

View/Edit MQ Options

 

Equipment Serial Numbers

Records the serial numbers of store computer equipment which are used by service technicians when store computer equipment is replaced.

Configure Color Station

This process will be completed with Help Desk assistance. Call 216-566-2740 for additional information.

Configure Label Printer

This process will be completed with Help Desk assistance. Call 216-566-2740 for additional information.

System Configuration:

Display Parameter

Parameters on the store computer system are necessary to process various store computer system transactions. These will always be the same on the store computer system. Parameters include the store's address, tax identification numbers, and banking information.

Help Desk/Depot Address

Print a mailing address to send equipment to the Help Desk or Depot Maintenance.

Credit/Gift Card Downtime

Provides the downtime credit card processing phone numbers for the facility.

TAX Information

Provides taxing jurisdiction information the current tax revision information.

Hardware Tests:

Run Backup Diagnostics

When a problem occurs with the backup, Store Computer Operations will be able to perform a diagnostic test.

Validation Printer

Store Computer Operations will be able to test the Validation Printer when a problem occurs.

Mag Stripe Reader

If a problem is detected with the Mag Stripe (Credit Card) Reader the HELP Desk will instruct on how to test the reader.

Bar Code Scanner

If a problem is detected with the Bar Code Scanner, Store Computer Operations will instruct on how to test the scanner.

Invoice Printer

Someone on the Help Desk will instruct on how to use this feature to test the invoice printer.

Satellite/WAN Operation

These programs are used to test communication between the store POS system and headquarters using the Hughes satellite dish installed at the store. When necessary, detailed instructions will be provided by Store Computer Operations Help Desk personnel.

LAN Connection Test

Assesses Local Area Network connection.

Miscellaneous:

Run Downloaded PGM Files

Tests communication between the store POS system and headquarters using the Hughes satellite dish installed at the store. When necessary, detailed instructions will be provided by Store Computer Operations Help Desk personnel.

Clear MMS/COE Status

Unlocks a Purchase Order (PO) that is In Receiving status or In Update status. Verify that the PO is not being updated or received on any store POS system terminal before using this option, and check the SYMBOL PDT (hand held terminal). Follow the prompts on the screen to unlock the PO.

Note: For more information, see Hand-held Troubleshooting Procedure.

Reset Inventory Locations

Unlocks a physical inventory location that was locked in error.

Configure Intermix Stores

This process must be completed by the Store Computer Help Desk. Call 216-566-2740 for instructions.

Inventory Status

Provides a quick overview reference for report printing and recording activity.

Coupon Print/View Report

View or print B.
