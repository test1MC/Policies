Review & Process the Store Email Quarantine Procedure

Step

Action

1

Open the email received from sw.user.quarantine.release.

Note: Quarantine emails only show items quarantined within the past 24 hours of the email date.  If the quarantine has not be reviewed for several days, review all quarantine messages for that time by using the EUQ Link in the email.  Items are deleted from quarantine after thirty (30) days.

2

How will you process items in quarantine?

If Then
From email Proceed to Step 3.
From End User Quarantine Link Proceed to Step 4.

Note:  Processing requests from the End User Quarantine Link is the recommended processing method:

  • Allows for safe listing of email addresses
  • Requires less processing time
  • Allows for mass processing
  • Allows access to all messages in quarantine

3

Process quarantined messages using the links to the left of each message:

Action

Result

Delete

Removes message from quarantine listing

Release

Releases selected message to inbox for review

Note:  A new web page will open to confirm the actions for each processed email.

4

Click on the link to access the End User Quarantine.

Note:  Link is only valid for fifteen (15) days.

5

Process quarantined messages:

Step

Action

1

Review message(s) for those which should be safelisted and released.

2

Select item(s) to process for safelist and release by checking the box to the left of the quarantined entry.

Note:  It is recommended that item(s) safelisted are also released.

3

Click the Safelist Button in the toolbar.

Note:  The email address of selected item(s)  will be whitelisted to prevent emails from future quarantine and allow for delivery to inbox. Confirmation will appear in the toolbar.  Item(s) remain in quarantine view.

4

Click the Release Button in the toolbar.

Note:  Selected item(s) will be released to the inbox and will be removed from quarantine  listing.

5

Select  remaining item(s)  in quarantine to process for deletion by checking the box to the left of the quarantined entry.

6

Select Options ->  Delete All to remove all remaining messages from quarantine.

Note:  It is recommended that once all items have reviewed and processed for safelisting and release, this option is used to clear quarantine.

7

Close quarantine email.

Note:  Additional processing options include:

Action

Result

Confirm

Notes

Delete

Removes message from quarantine listing.

Item(s) will be removed from quarantine listing.

 

Not Spam

Reports message as “not spam”  but does not release message or safelist email address.

Confirmation will appear in the toolbar. Item(s) remain in quarantine view.
