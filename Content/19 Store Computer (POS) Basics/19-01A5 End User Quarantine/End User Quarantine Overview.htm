End User Quarantine Overview

The End User Quarantine (EUQ) is a corporate measure to better secure Sherwin-Williams from external threats, viruses, and unwanted email by allowing employees to control receiving emails they need and eliminating unwanted emails. EUQ is a daily web based service that allows users to see and release messages that have been stopped in quarantine and to safelist senders so that they are not quarantined in the future. 

Stores will receive an email message with a link to the quarantine showing probable spam received in the past 24 hours—if no messages received meet this criteria, no email will be received by the store. A high percentage of messages in quarantine will be obvious spam and bulk email. Once a specific email address has been added to the safelist, emails from this address will be delivered to the Inbox.

For any questions, contact the SCO Help Desk at 216-566-2740 or scohelpdesk@sherwin.com.
