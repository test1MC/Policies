SherLink Password Update Procedure

Follow these steps to update the SherLink password anytime there is a change in personnel. The District Manager may delegate the responsibility of changing the SherLink Password to a Sales Representative or a Store Employee.

  • If you are a Sales Representative, call (800) 566-2997 (Option #1, Option #2) to change the SherLink password.
  • If you are a Store Employee, follow these steps to change the SherLink password:

    • If

    Then

    You know the old password

    Step

    Action
    1 From POS desktop, select Browser -> Access SherLink.

    2

    Select Edit Profile in the upper right of the screen.

    3

    Scroll to the bottom of the page.

    4

    Enter the new password two times.

    5

    Select Submit Edits.

    You do not know the old password

    Step

    Action

    1

    If you are already logged in to SherLink, select Logout in the upper right corner of the screen. Select Yes to “Are you sure you want to log out?” If you are not already logged into SherLink, then access SherLink at www.sherlink.com. The end result of either of these methods is a SherLink Log-In Screen.

    2

    At the SherLink Login screen, enter the following as your User ID: the letter [s], your [store number], then the letter [r].

    Example: Store Number 1234 = [s1234r].

    3

    Select Forgot Password?

    4

    Enter the letter [s], your [store number], then the letter [r]..

    5

    A new password will be emailed to your store email address.

    Note: Call 1-800-522-8861if you have any problems resetting the password.
