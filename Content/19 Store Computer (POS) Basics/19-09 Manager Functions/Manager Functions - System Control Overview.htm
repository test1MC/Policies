Manager Functions - System Control Overview

System Control allows Store Management to make any necessary adjustments to the POS system.

Feature

Function

Change Manager Password

Changing the manager password will require a minimum entry of two letters and two numbers. The computer system will prompt to enter the New Password two times. The double entry will validate the new password.

Note: The System Manager Password should be changed at least once every ninety days for security purposes. For forgotten System Manager Passwords, call Store Computer Operations at 216-566-1740. They will notify the District Manager anytime if assistance is needed with the password.

View Last 3 Manager Passwords

 

Quickly review the last three expired system manager passwords. Store Management can see if the password is changed at least once a quarter, according to Company policy.

Outside Phone Line Access

Adjust telephone access by store location. The Store Computer Operations Help Desk will instruct employees to modify the setting if required.

WC Book Lending Controls
This section does not apply to Automotive Branches

Adjust the number of days the customer is allowed to keep the books and the default dollar amount for Wallcovering Book lending.

Changing the Number of Days From Check Out That Books Due will adjust the system reporting of overdue books for all customers. Changing the Default Amount of Deposit will adjust the lending dollar amount. 

Note: District Manager approval is required for these action.

COE Picking Ticket Copies
  • COE Picking Ticket Copies: Change the number of default picking ticket copies that will print from the laser printer. 
  • Picking Ticket Format: Print or suppress the hand written comment lines and print all the headings on all the pages in the picking ticket or the first 6 lines of headings starting from page 2 to increase the number of lines being printed.
  • COE Headings: Print all of the headings on all the pages in the customer order or only the first 6 lines of headings starting from page 2 to increase the number of lines being printed.

Configure Formula Printing

Choose whether to have formulas printed on customer receipts, customer picking tickets, or customer orders.  Choose to have Sherwin Williams Formulas, Competitive Formulas, and Custom Formulas to be displayed or not be displayed. The system is defaulted to Yes.
