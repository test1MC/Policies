Adjust File Listings Procedure

File Listings contains master file reports that have been loaded into the store computer system by the main computer at Headquarters. These reports can be printed on the Manager’s printer when selected from the Reports menu. The information in these files is used to properly access the correct product and customer related information. This information can also be used in a report form to enable better management of the store business.

Step Action
1 After selecting one of the files, a system prompt will give instructions to either View Immediately or Send to Reports File for Future Use.
2 The Reports file can be accessed after pressing the Reports key located in the operations portion of the keyboard.
3 Press Cancel [F5] to stop the processing of any report selected from the File Listing menu. Since several of these listings are extremely long, this allows exit out of this area if an error was made.
