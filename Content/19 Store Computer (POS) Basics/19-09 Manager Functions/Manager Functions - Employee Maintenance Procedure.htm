Manager Functions - Employee Maintenance Procedure

Step Action
1

Someone in store management must enter employee information into the computer system for each employee by selecting Main Menu -> Manager Functions -> File Maintenance -> Employees.
2

A message will appear stating that employee information needs to be correct for Payroll and that each employee should only be assigned one employee number. Do not reassign an existing employee number.

Note: Employee Numbers (1-99 excluding 97) are available on the computer system. Remember that during transactions, only the employee name and not the employee number will return to the screen.
3

After selecting one of the following options, the Employee Maintenance window will be returned to the screen. Modifications can be made to the Employee file by following the system prompts in the lower corner of the screen.

Note: Deactivated employees will be blocked from processing any point-of-sale transaction in the system. Employee numbers should be deactivated when the employee is promoted to another location or no longer works for the company. View deactivated employees by selecting Main Menu -> Manager Functions -> Employees -> Print/View Employee Detail.
