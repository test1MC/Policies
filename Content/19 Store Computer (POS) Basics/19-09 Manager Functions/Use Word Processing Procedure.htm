Use Word Processing Procedure

Corresponding with customers, the District Manager, Division, and Group staff is made easier with Word Processing on the store computer system. It is an easy-to-use word processor that allows employees to produce letters, memos, and forms, while being totally integrated with the store computer system. Lookup [F3] customer and product information, or use Inquiry [F4] to find previous transaction detail.

Word Processing is available from all terminals on the store computer system. It is recommend to, if possible, use the manager's office system to perform Word Processing functions.

Step Action
1

From the POS Desktop -> Store Button -> Miscellaneous or Manager Functions.

Note: Miscellaneous is not password protected. Manager Functions is password protected.
2

Word Processing contains two specific areas: Documents and Templates.
