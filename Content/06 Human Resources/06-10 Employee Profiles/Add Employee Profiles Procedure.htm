Add Employee Profiles Procedure

Step

Action

1

From the POS desktop, select Store -> Select Manager Functions from the POS Main Menu.

2

Enter your Employee Number and Password.

Note: Only managers can access Manager Functions.

3

Select File Maintenance -> Employees -> Add.

Note: Additional functionality includes Change, Deactivate, Reactivate, and Print/View Employee Detail.

4

Complete New Hire fields:

Field

Info

Employee Number

This number will be used to process activities for Attendance and Time and to process Point of Sale transactions.

Social Security Number or GEMS ID

Enter the employee’s SSN and GEMS ID.

If

Then

TAG stores

Press Lookup[F3] and select the employee’s name and GEMS ID. If the employee is not listed, contact the District Coordinator to obtain the employee’s GEMS ID.

Automotive branchess or PF facilities

Contact the Regional Coordinator for the employee’s GEMS ID

Caribbean stores

Contact Doretta Battle (doretta.r.battle@sherwin.com) to obtain and enter the new hire’s GEMS ID.

Note: If the Coordinator is absent or unable to provide the GEMS ID, contact HR Shared Services (HRSS) at hrss.tag@sherwin.com for assistance.

Title

Press Lookup[F3] in the Title field to prompt a listing of job titles.

Exempt Status

Non-Exempt Employees are eligible for overtime, while Exempt Employees are not.

MMS Security Level

Depending on the employee’s job title, enter 0-4. See MMS Security Levels Policy  for more information on security levels and job titles.

Run POS

Dictates employee’s use of POS.

Run COE

Dictates employee’s use of COE.

Borrowed

Indicates if the employee is borrowed from another store.

A/R Access

Dictates employee’s A/R Access.

PRC Security Level

If

Then

0

Employee will not be able to alter Price Record Cards.

1

Employee will be able to alter Price Record Cards.

Note: Only members of store management are allowed to alter customer Price Record Cards.

5

Once all required fields are completed, press Accept [F1] to submit the profile.

Note: Once submitted, the form will clear. New employee payroll file information will be downloaded to the Store Computer System within 48-96 hours after the Store Manager has contacted Division Personnel to hire a new employee.
