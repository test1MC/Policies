H1N1 Pandemic Policy

  1. The Store Manager or Assistant Manager is responsible for entering Absence Reasons for all employees.
  2. The Store Manager and Assistant Manager must enter any "unworked" hours for each employee daily.
  3. The appropriate number of hours must be entered based on each “unworked” hours code.

Note: The Store Manager should talk to the District Manager if any discrepancies arise. For more information, see 28-04A Absences.
