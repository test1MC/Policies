Secondary POS Station Installation Overview

Do not power down any of the ThinkCentre computers via the power switch. Only perform a logical shutdown when directed by these instructions or by the Store Computer Operations Help Desk.

These instructions will walk you through installing equipment at an additional POS.

The basic steps include:

  1. Installing new Lenovo 10ANS07C00 PC and POS peripherals.
  2. Certifying operation.

The main POS and managers stations must be installed prior to any additional POS (points of sale).

  1. Main POS and Main Manager
  2. Secondary POS stations
  3. Tint Station
  4. Additional Manager Stations

Please call the Sherwin-Williams Store Computer Operations Help Desk at 216-566-2740 with any installation questions.
