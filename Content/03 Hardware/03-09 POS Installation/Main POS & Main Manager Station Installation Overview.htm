Main POS & Main Manager Station Installation Overview

Do not power down any of the Thinkcentre computers via the power switch. Only perform a logical shutdown when directed by these instructions or by the Store Computer Operations Help Desk.

The basis steps include:

  1. Installing new Lenova 10ANS07D00 PC’s.
  2. Certifying operation.

Subsequent instructions for any other installations at this location must not be attempted until this instruction pack is complete.
