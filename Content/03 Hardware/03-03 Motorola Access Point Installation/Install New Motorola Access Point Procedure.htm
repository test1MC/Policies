Install New Motorola Access Point Procedure

Step Action

1

The new Motorola unit comes ready to go. Plug the blue patch cable into the same port that the Cisco router was plugged into.

2

Plug the power cord from the power injector into the surge protector.

3

Once the power has been plugged in, it will take about eight minutes for the access point to be ready to work. When it is ready, the amber light will remain on steady and the green light will flash slowly. Turn on your AML scanner and it should connect. If it does not connect within two minutes, call the Help Desk.

Once it is connected, try doing some cycle counts in the farther areas of the store to make sure it is communicating properly. If everything is working properly, peel off the back of the mounting tape on the back of the bracket on the Motorola access point and mount it up as high as possible.

4

Make sure to put the Motorola on top of the network cabinet and not on the inside.  If the Motorola is located in the office, make sure that it is put up high and not on the floor.

If it does not work properly, call the Help Desk (216) 515-2740.
