Automatic Tinter Installation Overview

Do not power down any of the ThinkCentre computers via the power switch. Only perform a logical shutdown when directed by these instructions or by the Store Computer Operations Help Desk.

These instructions will walk through installing equipment at the tint station. The basic steps include:

  1. Installing new 10ANS07C00 client and peripherals.
  2. Certifying operation.

The main POS and main Managers station must be installed prior to installing any additional stations.

Installation Order:

  1. Main POS and Main Manager
  2. Secondary POS stations
  3. Tint Stations
  4. Additional Manager Stations

Please call the Sherwin-Williams Store Computer Operations Help Desk at 216-566-2119 with any installation questions.
