Install Tint Station Client Hardware Procedure

Step Action
Install Tint Station Client

1

Install a Tripp Lite surge protector at the existing tint stations. Use an open AC receptacle. 

Note: Do not plug into another power tap or extension cord.

2

Turn on the Tripp-Lite surge protector.

3

Unpack a Lenovo ThinkCentre PC model number 10ANS07C00. Place it on the tinter. Plug the power cord into the back of the PC and the other end into the Tripp Lite surge protector.

Lenovo 10ANS07D00 Server / Lenovo 10 ANS07C00 Client

4

Unpack the LENOVO 17” Display and place on top of the Lenovo ThinkCentre PC.  Connect the display to the Lenovo ThinkCentre.  Plug in the power cord to the Tripp Lite surge protector.

5

Connect the mouse and keyboard into the back USB connections on the Lenovo ThinkCentre.

6

Place the keyboard template (found in the instruction packet) over the function keys on the keyboard.

7

Unpack the Motorola scanner gun and cable.

8

Connect the cable to the Motorola scanner gun.

9

Plug the Motorola scanner gun into the front USB port on Lenovo ThinkCentre PC.

10

Unpack the Zebra label printer.

11

Assemble the power cord and plug it into the back of the printer and the new Tripp-Lite surge protector.

12

Attach the USB cable from the back of the Zebra label printer to the PC.

13

Install a roll of labels into the printer, as shown on the inside of the printer.

14

Connect the 14’ Ethernet cable from the Lenovo ThinkCentre to the network jack.

15

In the network cabinet, install the short Ethernet patch cord for the corresponding data jack on the patch panel to the new ATI switch. Verify there is a LINK/ACTIVITY light lit.

16

Turn on all the equipment.

17

The STORE COMPUTER SOURCE desktop should appear within 60-90 seconds.

18

Click on STORE.

Note: The Main Menu should appear.

20

Select Option 4 – Help Desk from the Main Menu.

Note: The Help Desk Menu should appear.

21

Select Option 19 - Bar Code Scanner and press ENTER.

22

The following screen will appear:

23

Use the Motorola barcode scanner gun to scan the barcode on any available product, verify the UPC number is captured.

24

Press CANCEL [F5] to return to the HELP DESK menu.

25

Select Option 16 – Label Printer to test the label printer.

26

Perform the label printer test, when complete.

27

Press CANCEL [F5] to return to the Help Desk Menu.

28

Press CANCEL [F5] to return to the Main Menu.

29

Use the Velcro cable wraps in the instruction kit to tidy up the cables under the desk and counters.

30

The installation of hardware for this station is complete.
