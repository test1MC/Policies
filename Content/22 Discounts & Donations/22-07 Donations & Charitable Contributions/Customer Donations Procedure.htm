Merchandise donation (including: mistints, obsolescence, and clearance) of 25 gallons or more
  • Complete the Materials Acceptance Agreement - Donate (available in the Forms Repository) for any less-than-perfect donated merchandise.
  • Have the Acceptor [customer] sign the agreement.
  • Retain the signed agreement and District Manager email response approval for 10 years in a folder titled [Materials Acceptance Agreement]
  • Mail a copy of the signed agreement to Division Loss Prevention and to:
    The Sherwin-Williams Company
    ATTN: Sam Skubak
    Corporate Legal
    101 Prospect Avenue
    Cleveland, Ohio 44115
  • Process a Store Expense Transaction using G/L Code 7615-001 CONTRIBUTIONS OTHER.

Note: For more information, see Store Expense Procedure.

Process a Store Expense Transaction using G/L Code 7615-001 Contributions Other.

Cash Donation

Process a SWeep Expense Transmittalusing the G/L Code 7614-001 Cash Donation.

Note: For more information, see Expense Invoice Submission Procedure.

Customer Donation (customers may pledge a donation in support of a charitable event in which Sherwin-Williams is either supporting or participating)
  • From POS desktop, select Store -> Point of Sale -> Sale
  • Enter customer information
  • On the line entry screen, press Enter to access the Non-Merchandise Code Menu
  • Select Charitable Donations
  • Input the quantity of 1
  • Charge the amount of the donation.
