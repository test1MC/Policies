View CAC Code on Customer Account Procedure

Step Action

1

From POS desktop, select Store -> Manager Functions -> Price Record Cards -> Print/Display.

2

Enter the Customer Account Number or press Lookup [F3] to look up the Customer Account.

3

Enter the A/R Job Number for the account.

Note: The CAC Code can be viewed on the right-hand of the screen underneath the Credit Limit.

4

Change Customer CAC Code Classification.

5

Call the District Credit Office/Service Center to make any customer classification changes.
