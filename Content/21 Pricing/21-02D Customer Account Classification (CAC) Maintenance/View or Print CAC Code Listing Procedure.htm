View or Print CAC Code Listing Procedure

Step Action
View CAC Codes

1

Access the Inquiry Menu by selecting INQUIRY [F4] on POS.

2

Use the arrow keys to navigate to CAC Codes under Miscellaneous.

3

Press Enter to select CAC Codes.
Print CAC Code Listing

1

 From POS desktop, select Store -> Manager Functions -> File Listings -> CAC Code File.

2

Press the Shift Key + REPORTS [F4] to access the Reports Menu.

3

Press Enter to select the CAC Code Master File List.

Note: You have the option of to display, print, or delete the report results.
