PRC Name Cleanup Help

You may encounter the following error messages when completing PRC Cleanup:

Error Message Resolution

"CPR edit is disabled at this time / Web service down"

See 37TUPRC Cleanup Disabled Time FramesU37T.

"Account xxxx-xxxx-x is locked - currently in use by xxxxxx"

Two employees attached to the same location cannot be in the application at the same time. Coordinate with the other employee.

"PRC name not reviewed"

Indicates that only the Job name was reviewed, but both the Job and PRC names need to be reviewed.

"Job name not reviewed"

Indicates that only the PRC name was reviewed, but both the Job and PRC names need to be reviewed.

"Review Failed/System Error Try Again"

The system did not accept the updated name. Please try again.
