PRC Name Cleanup Overview

To facilitate E-Commerce orders, stores are required to clean up Job and Price Record Card (PRC) names. With online ordering, customers will select their account and job account information online, which will prompt their pricing prior to the order being placed.

Below is an example of a customer’s view for online ordering and pricing, and the bolded text represents the Job number and PRC name that the customer has selected. Because of this visibility and transparency, it is essential to perform a PRC Cleanup so that customers select the correct job number and name.

The PRC Cleanup application has been created to expedite this initiative:

  • The application will only show Jobs and PRCs that need to be reviewed, which includes (1) rollup jobs that are not tied to the TRC system, and (2) PRCs with multiple PRCs on a single job.
  • An automated system cleanse will name all Job (1) /Card (1) accounts as Main Account, and jobs with only one PRC will have matching names.
  • Remember that Job and PRC names will be visible to your customer, so they need to be meaningful and business appropriate.
