DVE Add for PRC 1 & PRC 2 or Higher Procedure

Step Action

1

Access PRC Maintenance -> Lookup the customer -> Select the appropriate account and PRC.

Note: For more information on PRC Access and Maintenance, see Price Record Card (PRC) Maintenance Overview.

2

Select Add an Exception.

3
If Then

Entering SKU for PRC 1

Enter the SKU.

Note: The system will determine whether the product is eligible for Category or Brand Pricing. Refer to Dollar Value Exception (DVE) Overview for more information.

Entering Category for PRC 1

Proceed to Step 9.
Entering SKU for PRC 2 or higher

Enter SKU, Category /Size, or Rex/Size Continue to enter the SKU level DVE. Uncheck the Create Category box and then select the current price for other sizes or sheens.

Note: The system will determine whether the product is eligible for Category Pricing. When a SKU is entered and the category is safe, then the category exception will be added. On PRC 02 or higher, the system provides the option to convert Safe Category SKUs to category pricing. Refer to Dollar Value Exception (DVE) Overview for more information.
Entering Category for PRC 2 or higher

Proceed to Step 14.

4

If applicable, enter a price difference between singles and fives. Select Next.

5

Single and five size codes will return with a range of pricing information. Complete one of the following actions to establish pricing:

  • Select Current Price
  • Select Suggested Price
  • Enter Custom Final Price

6

Select Next.

7

Enter Competitive Information (vendor, product, price) and select Next.

8

Review the Estimated Impact data and select Next.

9

Enter the Category.

Note: The system will determine whether the product is eligible for Category or Brand Pricing. Refer to Dollar Value Exception (DVE) Overview for more information.

10

If applicable, enter a price difference between singles and fives. Select Next.

11

Single and five size codes will return with a range of pricing information. Complete one of the following actions to establish pricing:

  • Select Current Price
  • Select Suggested Price
  • Enter Custom Final Price

Note: If any SKU level exception(s) exists for this category, they will be displayed and checked for deletion. To keep the SKU level exception, uncheck it. If kept, the SKU level exception will override the category exception.

12

Select Next.

13

Enter Competitive Information (vendor, product, price) and select Next.

14

Review the Estimated Impact data and select Next.

15
If Then

No other changes are needed

Select Apply.

Additional changes are needed

Select Add An Exception, Edit, or Cancel.

Note: For DVEs below the division’s approved price (floor), the system will prompt for confirmation to send to District Management for review and approval. See Pricing Approval Overview for more information.
