Edit or Delete DVE Procedure

Step Action

1

Access PRC Maintenance -> Lookup the customer -> Select the appropriate account and PRC.

Note: For more information on PRC Access and Maintenance, see Price Record Card (PRC) Maintenance Overview.

2
If Then

Edit
  • Select Edit on the appropriate exception.
  • Follow the screen prompts and make any necessary changes.
  • Select Next.

Delete

Select Delete on the appropriate exception.

3

Review the Estimated Impact data or the exception flagged for delete.

4

Select Apply once all changes have been made.

Note: For DVEs below the division’s approved price (floor), the system will prompt for confirmation to send to District Management for review and approval. See Pricing Approval Overview for more information.
