Update Customer Information Procedure

Step Action

1
If Then
Editing customer information

From the POS desktop, select Browser -> Access Source homepage -> Select Customer Information from the Credit Tab under Source Operations-> Enter the Account # -> Select the EditButton -> Enter the Customer Information -> Proceed to Step 2.

Note: Any field with a box displayed can be modified.
Looking up customer record From the POS desktop, select Browser -> Access Source homepage -> Select Customer Information from the Credit Tab under Source Operations-> Enter the Account # -> Procedure is complete.

Note: Accounts Payable Contacts and E-mail information details are expanded. Select the check box to collapse the information.
Deleting customer information

From the POS desktop, select Browser -> Access Source homepage ->Select Customer Information from the Source Operations Tab -> Enter the Account # -> Select the EditButton -> Select the Delete button -> Proceed to Step 3.

Note: If you selected delete by mistake, click on the undelete button to make the customer account active.

2

Does your customer wish to receive marketing phone calls from the store?

If Then
Yes Leave the box blank
No Select the box and a check mark will display indicating the customer does not want to be contacted

Note: This indicator is used for marketing or prospecting calls only.

3

Select Accept to save changes.
