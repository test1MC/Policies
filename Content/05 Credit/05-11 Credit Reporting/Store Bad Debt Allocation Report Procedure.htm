Store Bad Debt Allocation Report Procedure

Step

Action

1

From the POS desktop, select Browser -> Access Source homepage ->Select Credit from the Source Operations Tab.

2

Select  Store Bad Debt Report.

3

Select a valid Month and Year.

 

Note: By default, month and year fields are pre-populated to the previous month of the current year.

4

Select Ok to open the spreadsheet with Open Office.

5

Select Comma in the Separator Options.

6

Select Ok to open the spreadsheet.

Note: If the date selected is before September 2013, a blank spreadsheet will be created.

7

If you receive the following error message, please try to log on again.

Note: If do not belong to a valid store or you leave the screen idle for a long period, you may get an error message. If you are a valid store user and the problem persists, please contact the Sherwin support group via e-mail collectionsupport@sherwin.com
