Create Contractor Accounts Procedure

Request/Search Contractor or Rollup Job Accounts

Step

Action

1

From the POS desktop, select Browser -> AccessSOURCE homepage -> Operations -> Credit -> Account Job Processing.

2
If Then
Contractor Job Account Select Job Processing
Rollup Job Account

Select Roll Up Job or Tax Record Card Job

3
If Then
Creating new request Select Create New Request -> Proceed to Step 4
Searching for contractor Select Advanced Search -> Procedure is complete.

4

Enter the Account # or select the Lookup icon to search for the account -> Press Submit Account.

5

Enter the required information.

Note: While entering the required information, follow these guidelines:

  • Be descriptive with the Job Name (ex. “Elm Elementary” not “school”).
  • Specify correct CAC Code as this can affect the marketing and specials presented to the customer.
  • Indicate whether or not it is a National Account. If uncertain, enter do not know so the NA Team can review and make the determination.
  • Add an explanation to any Yes responses.

6

Press Submit.

Note: If required information is missing, an error message will appear at the top of the screen. Required fields can be identified by a red asterisk*.

Note: Submitted jobs are reviewed by the District Credit Office or Financial Services. Once approved, the Job Account will be available in the Account Receivable System.
