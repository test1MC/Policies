Dispose of Rental Unit (Store Closing) Procedure

Two weeks before closing the store, the rental program will be discontinued.  This should allow customers time to return any outstanding units and the store time to process the units for disposition.  The Store Manager should remove any rental units, displays, and signage from store front at this time.  Rental units can be disposed of as soon as the rental program has been discontinued.   Several options exist for the disposal of rental units:

  • Sell Rental Equipment:  The store will need to complete the Gross Profit Worksheet.
  • IST Rental Equipment:  The store will need to complete the Gross Profit Worksheet.
  • Retirement
