Generate Shelf Labels Procedure

Generate Shelf Labels by Plan-O-Gram Sequence, Shelf Labels on Demand, or Shelf Labels for Vendor Direct Purchases.

Step Action

1

From POS desktop, select Browser -> Access SOURCEhomepage -> Operations -> Processes -> Shelf Labels on Demand.
2
If Then
Generating Shelf Labels by Plan-O-Gram Sequence Select the Use Plan-O-Gramicon -> Select the correct POG # title.
Generating Shelf Labels on Demand

Enter a Sales Number, or REX Number, or UPC Number.
Generating Shelf Labels for Vendor Direct Purchases

Enter a Vendor Direct Sales Number.

3

Enter the label quantity and DIY selling price, including the decimal point.

Note: Ensure that the shelf label price matches the DIY price from the store computer system. Verify the system DIY price by selecting Inquiry [F4] and then selecting Products and Prices [22].

4

Select Add to Sheet.

5
If Then
Generating Shelf Labels by Plan-O-Gram Sequence Proceed to Step 6.
Generating Shelf Labels on Demand

Determine the type of labels needed:

If Then

Labels with Prices

Select Print Labels.

Labels without Prices

Select the Print Shelf Labels without Prices box on the lower left side of the screen. Then select Print Labels.

Note: These labels can be used in the back of the store in the front if State law requires individually priced items.

Labels with Prices and Product Number instead of SMIS

Select Print Product Number instead of Sales Number. Then select Print Labels.

Labels  without Prices and Product Number instead of SMIS

Select both Print Shelf Labelswithout Prices and Print Product Number instead of Sales Number. Then select Print Labels.

Quick Scan Reference Chart

Select Print in Quick Scan Reference  Chart format. Then select Print Labels.

Warehouse Label - Global

Select PFG. Then select Print Warehouse Labels.

Warehouse Label - Canada

Select Canada. Then select Print Warehouse Labels.
Generating Shelf Labels for Vendor Direct Purchases Proceed to Step 6.

6

Print the labels:

Step Action

1

Place the correct number of sheets of label stock in the Lexmark Printer’slower paper drawer.

Note: If there are issues loading the paper, try feeding the label stock through the multipurpose feeder on the printer. Then, configure the printer to accept paper from the multipurpose feeder by selecting the PAPER MENU -> Configure MP submenu. Finally, press the right or left menu keys until the LCD panel displays First. Save this setting.

2

Select Print Labels from the Shelf Labels on Demand page.

3

Select the small printer button displayed at the top of the page.

4

Select OK.

Note: 28 price labels will print on one page.

7

Press the X at the top right of the screen to close the Shelf Label Request Screen.

Note: If additional Shelf labels need to be printed select Delete or Delete all to start the process over.

Note: For Vendor Direct Purchases, complete a Product Add/Maintenance Request via SWeep if a barcode does not display for the vendor direct product and the UPC number is known. For Plan-O-Gram Sequences, please reference the Plan-O-Gram document or email the Category Development group with any questions.
