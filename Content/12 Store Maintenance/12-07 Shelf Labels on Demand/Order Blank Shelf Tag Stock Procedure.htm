Order Blank Shelf Tag Stock Procedure

Step Action

1

From POS desktop, select Browser -> Access SOURCE homepage -> Operations -> Order -> Order Express.

2

Under Search Catalog enter the item number: A101154PD.

3

Add the quantity.

Note: There are 32 sheets per pack. Stores may only enter up to two packs per order.

4

Select Add to Cart.

5

The system will generate the order to be shipped to the store.
