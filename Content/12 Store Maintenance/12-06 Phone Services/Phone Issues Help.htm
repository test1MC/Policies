Technical Line Issues:

Issue

Remedy
  • Dead line(s)
  • Static on line(s)
  • Line cuts in and out
  • Cross talk issues (voices heard on line)
  • Incoming caller receives busy signal
  • Incoming caller hears rings but store phone does not ring
  • Any issue impeding inbound/outbound communication
  • Call Client Services Assurance (216-566-2997 option 3)
  • Client Services Assurance will:
    • Create a ticket
    • Contact the local provider
    • Have the provider check the line
  • Where indicated:
    • Send out either the local provider’s field technician and/or an internal line technician if it is determined that the problem is within the building (not an external telephone line problem).
  • If it is determined that the problem is equipment related please call Finance and Procurement (216-566-1837). They will walk you through the process for determining whether to order new equipment or repair the existing device(s).

Other Line Issues:

Issue

Remedy

Unable to place Long Distance Calls; store personnel can make local calls, however, toll calls are not going through.

Call Finance and Procurement (216-566-1837). Finance and Procurement will contact the provider and make sure that the long distance pick code is entered correctly.

Unable to place local calls, or main line is not rolling into secondary/additional lines (Hunt group exception). When the main line is busy while the other lines remain open, incoming calls are not moving over to the other lines.
