Order Fixed Asset Policy

  1. The District Manager is responsible for initiating a request to purchase a Fixed Asset.
  2. All requests must be approved by the Division office.
  3. The Store Installation Manager is responsible for processing the order and payment of the invoice.
  4. The Store Installation Manager is responsible for Mixing/Tinting Equipment installation and arrangement if required.
  5. The Division Engineer is responsible for installation of all equipment purchased by the Engineering Department.
  6. Fixed Asset Accounting will assign an asset number and add the asset to the Fixed Asset Register.
