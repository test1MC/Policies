Transfer or Retire Fixed Assets Report Procedure

Step

Action

1
If Then
Store From POS desktop, select Browser -> Access Source homepage -> Select FACTFinder from Store Reports on the Source homepage.
Non-store location From POS desktop, select Browser -> Access mySherwin -> Select FACTFinder from Applications on the mySherwin homepage.

Note:  For more information on accessing or using FACTFinder, see FACTFinder Overview.

2

Select Fixed Assets from the Report Dropdown.

3

Enter the Division and Cost Center for which you wish to query or use the search function.

4

Select Run Report.

5

Export the file by selecting Export then select To XLS.

6

Select Open with: LibreOffice then select OK.

7

Select File and Save as a Copy.

Note: Save the entire Fixed Asset file and use the Store Number as the name for it (i.e. 1234.xls). Do not delete any lines. 

8

Select Edit Document in the upper right-hand corner of the screen.

If

Then

Transfer

Type T in the Transfer column followed by the cost center that the item was transferred to. (i.e. T-1234)

Retire

Type R in the Retire Column.

Partial transfer or retire

Add the quantity.  i.e. 2 T-1234 (for transfer) or 2 R.

Sales

Type S in the Retire Column followed by the proceed amount.  i.e. S - $1000.

9

Save changes and send an email to the District Manager with the attached file.

Note: The District Manager will email the file to Stores.FAQuestions@sherwin.com.
