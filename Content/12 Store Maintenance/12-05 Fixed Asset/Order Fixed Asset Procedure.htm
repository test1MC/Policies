Order Fixed Asset Procedure

Step

Action

1

Contact the District Manager to order a fixed asset. The District Manager will initiate a written request and forward to the Division Office for approval.

2

When approved, the Division will forward the request to the Store Installation Manager or Field Engineer.

Note: The Engineering Department orders all PSR Room related equipment excluding Air Mixers, Drum Racks and grounding needs.

3

Retain in the store file a copy of the purchase order sent from the Store Installation Manager.

4

Any invoices received by the store, ordered by the Installations Department, for a fixed asset should be sent to the Store Installation Manager for payment. Any invoices received, ordered by the Engineering Department, should be sent to the Project Engineer.

5

Any problems with the installation of the fixed asset should be directed to the Store Installation Manager or Project Engineer.

6

Check the next Fixed Asset Register to ensure the asset is listed properly. If not, notify Fixed Asset Accounting at:
