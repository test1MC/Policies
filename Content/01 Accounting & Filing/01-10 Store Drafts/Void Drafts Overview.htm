Void Drafts Overview

Error

Action

Transaction was voided before the completion and the draft was not written. The draft can be used for a new transaction.
  • Process another transaction using the same draft number

Draft was written but the transaction was voided before completion. Draft cannot be used for a new transaction.
  • Call 216-515-4660 to report the voided draft/transaction (216-566-2287 – stores in Canada ONLY)
  • Write [VOID] across the face of the draft and include the reason for the void.
  • Cut out and destroy the signature section.  Staple the Voided Store Draft (all copies) to the transaction, if available.  Retain in the Store Draft file.

Transaction was completed and the draft was written. Draft cannot be used for a new transaction.
  • Call 216-515-4660 to report the voided draft/transaction (216-566-2287 – stores in Canada ONLY)
  • Write [VOID] across the face of the draft and include the reason for the void.
  • Cut out and destroy the signature section.  Staple the Voided Store Draft (all copies) to the transaction, if available.  Retain in the Store Draft file.

Voided drafts should be reported to StoreDrafts@sherwin.com and include the following information:

  • Store Draft Number
  • Explanation of if it was processed through POS and if it is replacing a draft
  • POS Transaction Type
