Store Draft Retention Policy

  1. The store copy of the store draft must be retained in the store draft book.
  2. The store manager must sign the register after reviewing each transaction and receipt. Place the register with the attached receipts (received when you made the purchase) in a folder titled VALIDATED STORE DRAFT REGISTER - Keep all VOIDED drafts intact in the Validated Store Draft file.
    1. If it is an original version of the Store Draft, attach the store copy and receipt to the Store Draft Register.
    2. If it is a new version of the Store Draft, retain all store copies in the store draft book. Then, return it to the store safe or locked filing cabinet.  The store draft will eventually only contain the store (yellow) copies.  Retain the partially used store draft book in the store safe or locked filing cabinet until all drafts have been used.  After using all store drafts in the store draft book retain it in the Validate Store Draft folder with the Store Draft Registers for the entire year.
