Downtime Processing Procedure

Step Action

1

Complete Manual POST Form by including the following information:

  • Transaction date
  • Customer name
  • Address
  • City
  • State
  • Phone number
  • Proper identification
  • Complete transaction information
  • Explanation for writing the downtime draft

2

Manually complete the draft, then immediately call 216-515-4660 (216-566-2287 stores in Canada ONLY) in accounting to register the check for payment. 

Note: This process needs to be done the same day the draft is written.

3

Process all downtime store draft payments by first selecting Enter and validate a manual form when the Store Computer System is operating,

4

Press the COMMENTS key to capture the downtime approval number.
