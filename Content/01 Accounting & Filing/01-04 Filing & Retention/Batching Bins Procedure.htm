Batching Bins Procedure

Step

Action

1

Place all of the daily SALES TRANSACTIONS in the container entitled [Sales Transactions] throughout the day. While performing the batching routine paperclip all sales transactions together and place in the Daily Transaction Envelope. These daily envelopes are to be stored in a Batching Box (Batching Box = Any Box).

2
If Then
RETURNS AND CORRECTIONS (Cash & Charge)

Place the store copy of the Returns and Corrections in bin, entitled: RETURNS AND CORRECTIONS. Returns and Corrections are to be stapled to the back of the Return/Correction Register.

The store manager will approve the Return/Correction Register and review Return Disposition on the store computer system. If a return is made by store draft, the store copy of the store draft must be stapled to the daily Store Draft Register. (New Store Drafts must be retained in the store draft book until all drafts have been written, then placed with Store Draft Registers).
ALL OTHER DAILY TRANSACTIONS Paperclip forms together and place in the Daily Transaction Envelope.
