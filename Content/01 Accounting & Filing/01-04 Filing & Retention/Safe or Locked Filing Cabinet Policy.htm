Safe or Locked Filing Cabinet Policy

  1. Items must be stored in a secure area (not necessarily a store safe).
  2. Use the Filing Storage Labels to create folders and a retention system for these items and forms.
  3. If any forms are missing, file an LOI report.

Find the labels in the Forms Repository. Search form name or keyword “filing” in the Form Name field. All available forms containing your search word will appear below in the Form Description field.
