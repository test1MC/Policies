End of Day Registers Procedure

Step

Action

1

Return and Correction Transactions must be stapled to the back of the Returns and Corrections Register. You are not required to staple transactions to any of the other registers. Leave all other copies of the transactions with the daily batch.

2

Cash Disbursements Register:
Staple the signed and dated receipts (received when you made the purchase) to the back of the Cash Disbursement Register. The store manager must sign the register after reviewing each transaction on the register and receipts. Place the register with the attached receipts in a folder entitled Cash Disbursements Register. Leave all copies of the transactions in the daily batch.
Return/Correction Register:

3

Attach copies of all daily Returns and Corrections to the back of the Return/Correction Register. The store manager will approve the Return/Correction Register and review Return Disposition on the store computer system. The Manager's signature on the Register designates that the following review process has been completed for all returns and corrections. (Manager is required to review each return and correction for each of the following areas but the manager is only required to sign the Return/Correction Register).

4

A copy of each refund and correction listed is attached to the register.

5

All Cash Refunds and Correction transactions have been signed by the customer and the processing employee.

6

Customer name, address, city, state zip and phone number is completed for each transaction.

7

If a return is made by store draft, the store copy of the store draft must be stapled to the daily Store Draft Register. Place the register with the attached transactions in a folder entitled Returns / Corrections Register, Store Drafts Register.

8

Attach the store copy of the * store draft and receipt (received when you made the purchase) to the Store Draft Register. The store manager must sign the register after reviewing each transaction on the register and receipts. Also, keep all VOIDED drafts intact in the Store Drafts Register file. Leave all copies of the store draft transactions in the daily batch.

9

The store copy of the newer Store Drafts will be retained in the store draft book and kept in the store safe or locked filing cabinet until all store drafts have be used. When all store drafts have been used, place the store draft book in the Store Drafts Register file with the Store Draft Registers. Place the register with the attached receipts (received when you made the purchase) in a folder entitled Store Drafts Register. Leave all copies of the transactions in the daily batch.

10

Miscellaneous Register :
The store manager must sign the Miscellaneous Register after reviewing each Miscellaneous transaction on the register. Place the register in a folder entitled Miscellaneous Register. Leave all copies of the transactions in the daily batch.

11

Void Register:

The store manager must sign the Void Register after reviewing each voided transaction on the register.

12

Place the register in a folder entitled VOID REGISTER. Leave all copies of the transactions in the daily batch.
