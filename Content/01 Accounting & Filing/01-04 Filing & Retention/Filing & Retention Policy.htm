Filing & Retention Policy

  1. Computer-generated forms for each point of sale must be retained separately for each terminal.
  2. Each day, month, and year, the Store Manager is responsible for verifying that the forms are retained and filed for the previous day's business.
  3. At the end of the year, the documents should be moved to a records retention area. Documents are kept in this area for the time period designated on the schedule. For more information see, Filing & Retention Batch Box Overview.
  4. If necessary, verify the number of transactions by referencing the Transaction Count report in the REPORTS file. Transaction Count Reports automatically delete after seven days.
  5. Once validated all Manual POST forms must be retained in numeric sequence. Perform a [Manual Forms INQUIRY] to verify the validation of all manual forms.
  6. DO NOT throw away any copy of a point of sale transaction that is not given to the customer. A copy of every transaction signed by the customer or employee must be retained.
  7. Every store's filing cabinet(s) should be set up according to the Filing System and Record Retention Schedule and all documents must be retained according to this list.
  8. Retain all previous years’ documents in covered boxes (gallon paint boxes). Keep all boxed records in a dry clean area. Also, make sure that all boxes are easily accessible.
  9. Label each box with the contents, the dates of the records, and the date the records will be able to be destroyed. Records that will have the same destroy date should be boxed together. Find the labels in the Forms Repository. Search form name or keyword “filing” in the Form Name field. All available forms containing your search word will appear below in the form description field.
  10. Three containers should be available to help with your daily batching.
    1. Sales Transactions
    2. Returns and Corrections
    3. All Other Transactions
  11. The following day, Sales Transactions, and All Other Transactions will be retained in one Daily Transaction Envelope. (Daily Transaction Envelope = any envelope) MAKE SURE THAT YOU INCLUDE THE DATE ON THE FRONT OF THE ENVELOPE. These daily envelopes are to be stored in a Batching Box (Batching Box = Any Box).
  12. Return and Correction Transactions are to be stapled to the back of the Return/Correction Register.
  13. Retain customer invoices in a folder entitled MAIL TO CUSTOMER if your customer is not on central invoicing and requests that you mail invoices. Invoices must be mailed at least once a week. If you are required to mail an invoice copy to your customer, be sure to select [Prefers Laser] on the customer Price Record Card.
