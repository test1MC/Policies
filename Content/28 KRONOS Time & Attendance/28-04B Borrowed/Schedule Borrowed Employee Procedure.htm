Schedule Borrowed Employee Procedure

Step

Action
1

Which borrowing method is being used to schedule?

If Then
Home Store Schedules Employee to be Borrowed

This is the preferred method for borrowing. When pre-scheduled using the steps below, employee hours will automatically transfer to the timecard when the employee clocks in (the employee will not need to use the transfer function in QTS). This method for borrowing must be used when:

  • The person being borrowed is a Store Manager.
  • Borrowing between DADs in the same division.
Borrowing Store Schedules Employee to be Borrowed

This process can be used to schedule an employee to work at your store location. This process will not:

  • Put the employee on the schedule
  • Allow for borrowing of Store Managers
  • Allow for borrowing between DADs in the same division.

2

Access Kronos using the Time Stamp button on the POS toolbar. Use your mySherwin credentials to log in.

3

From the Manager Landing Page, select the Schedule Planner Widget.

4

Launch a full-screen view of the Schedule Planner by selecting the maximize button ().

5

Select Select Dates Button. Set the Start Date and End Date to the time period in which the employee needs to be scheduled at another store.

6

Add or edit a shift for the day in which the employee will work at another location.

7

Select the cell in the Labor Level Transfer Column. Select Search.

8
If Then
Home Store Schedules Employee to be Borrowed Expand the store that the employee will be borrowed to in the organizational map.
Borrowing Store Schedules Employee to be Borrowed

Open the Store Dropdown. Select the store number of the store at which the employee will be working.

9
If Then
Home Store Schedules Employee to be Borrowed Expand the Work in Store Section. Select the Job they will work at the borrowed store.
Borrowing Store Schedules Employee to be Borrowed Proceed to Step 13.

10

Select Apply.

11

Select on the cell in the Labor Level Transfer Column. Select Search.

12

Open the Store Dropdown.  Select the store number of the store at which the employee will be working.

13

Select Apply.

14

Select Save.
