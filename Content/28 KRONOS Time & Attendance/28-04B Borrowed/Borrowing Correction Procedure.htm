Borrowing Correction Procedure

If an employee was not pre-scheduled to work at another location or did not use the transfer function in QTS, a manager at the home store will need to adjust the hours worked after returning to the home store location before the close of the pay period. This process will need to be repeated for all shifts worked at the non-home store location.

Step Action

1

Access Kronos using the Time Stamp button on the POS toolbar. Use your mySherwin credentials to log in.

2

From the Manager Landing Page, select the Employee Name.

3

Select on the blank cell in between the In Punch and Out Punch worked at the other store location.

4

Select Search.

5

Open the Store Dropdown. Select the store number of the store at which the employee worked.

6

Select Apply.

7

Select Save.
