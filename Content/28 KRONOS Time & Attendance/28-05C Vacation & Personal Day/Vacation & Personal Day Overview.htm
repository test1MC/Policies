Vacation & Personal Day Overview

Use the Request Time-off function in Kronos to request paid vacation days and personal holidays. For information on reviewing accruals, requesting, viewing, retracting / cancelling, and manager entry of vacation time, please see Request Time-Off Procedure.  For information on manager approval of requested vacation time, please see Manager Approval of Time-Off Requests Procedure.
