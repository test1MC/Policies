Add Pay Code to Timecard Procedure

Step Action

1

Access Kronos using the Time Stamp button on the POS toolbar. Use your mySherwin credentials to log in.

2

Launch a full-screen view of the Schedule Planner by selecting the Schedule Planner Widget.

3

Select the employee for whom you need to make a change.

4

Select Add Pay Code.

5

Enter the Effective Date, Pay Code, and Amount (hours, minutes, full day, half day). 

Note: Make sure the Override Shift Box is ticked (this will delete the original schedule and replace it with this change).
