Time-Off Request Overview

Managing time-off in Kronos is a fully integrated process. Employees make requests which are approved (or denied) by management. If approved, the request for time-off will be automatically added to the schedule and automatically added to the timecard. Employees should be encouraged to submit time-off requests as soon as possible so that this information can help create the best schedule for the store and save managers extra steps in processing an ad hoc request and re-posting a schedule during a pay period.

Employees are able to request time-off for:

  • Vacation
  • Personal days
  • Jury duty
  • Excused time off without pay
  • Local sick time (where available)

Managers will handle the following requests directly on an employee’s timecard as employee’s cannot request these types of time-off:

  • Disability (sick days)
  • Funeral leave
  • Excused paid time off

When employees request time-off, they can let the system know whether or not they want to be available or unavailable to work that day. For example, if an employee wants to take a half day of vacation and be scheduled to work the other part of the day, they would choose to request vacation time and be available to work. If the employee wants to take a vacation day and not be scheduled, they would choose to request vacation time and be unavailable to work. Note that even if the employee chooses to be available along with their time-off request, this does not guarantee they will be scheduled.
