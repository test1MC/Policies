Open Shift Request Procedure

Employees may request to work open shifts:

Step Action

1

Access Kronos using the Time Stamp button on the POS toolbar. Use your mySherwin credentials to log in.

2

From the Manager Landing Page, select the MyCalendar widget.

3

Hover the mouse over Open.

4

Select the arrow that pops up to the right.

5

Select Request Open Shift.

6

Select Submit.
