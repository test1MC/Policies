Kronos Availability Procedure

Setting availability is a manager only function.

Step Action

1

Access Kronos using the Time Stamp button on the POS toolbar. Use your mySherwin credentials to log in.

2

From the Manager Landing Page, select the Schedule Planner widget.

3

From the Schedule Planner widget, select the employee name -> Availability

Note: On the Schedule Planner pane, select the maximize button () to launch full-screen. All employees will default to available for all shifts until base availability is specified.

4
If Then

No availability has been assigned

New Availability Pattern form will automatically launch.

Availability has been assigned

Existing patterns will return. Select to Add Pattern, Edit existing pattern (pencil symbol), or Delete existing pattern (x symbol).

5

To add or edit availability pattern, complete all fields:

Field Description
Anchor Date

Date the pattern begins.
Availability Start Date

Date the employee will begin to work the pattern.

Note: For best practice, the anchor date and availability date should match and be set for the first day of the pay period.
End Date

Date the availability pattern ends (if base availability will remain the same for the foreseeable future, check the Forever box).
Pattern Name

Example: Joe’s Fall Semester
Pattern Type

Select Base.

Define Pattern

Set weekly or daily availability patterns.

Note: If adding multiple patterns, select the + button.

6

Set availability:

If Then
Setting availability for the whole day

Select/highlight the appropriate day block(s) and select the corresponding availability tag (only use available or unavailable tags).
Setting availability for part of the day

  • Select/highlight the appropriate day block(s) -> Hourly view

  • Drag to select/highlight the appropriate 15-minute time blocks

  • Select the corresponding availability tag (only use available or unavailable tags)

  • Select Apply

Note: To select/highlight multiple days, hold down the Ctrl button while making selections.

7

Select Apply.

Note: For key-holders whose availability coincides with open or close, add 15-minutes of availability before the store open/after the store close. This will account for their keyholding activities.
