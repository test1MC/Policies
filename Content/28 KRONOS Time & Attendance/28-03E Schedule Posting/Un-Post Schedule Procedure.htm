Un-Post Schedule Procedure

Step Action

1

Access Kronos using the Time Stamp button on the POS toolbar. Use your mySherwin credentials to log in.

2

From the Manager Landing Page, select the Schedule Planner widget.

3

Select Tools -> Manage Schedule Posts.

4

Select on the Check Box at the top of the Posted Schedules Listing to select all employee schedules for the pay period.

5

Select Un-post.

6

Confirm your wish to un-post schedules. Select Yes.

7

Select Done.
