Missed Punch Alert Procedure

Step Action

1

Access Kronos using the Time Stamp button on the POS toolbar. Use your mySherwin credentials to log in.

2

Review the Missed Punch Alert Widget from the upper center of the screen.  f you have requests to process, the ticker will alert you to the number of pending requests.

Note: You will need to review Inbox for the associated message from the employee’s attestation alerting you to the corrections which need to be made.

3

Select the Missed Punch Alert Widget.

4

Select an employee name to view the timecard.

5

Locate the missed punch and select the cell.

6

Enter the time to complete the punch. Select out of the cell.

7

Select Save.

Note: Once you have approved or refused, the request alert button will decrease the number of pending requests (you may need to select the refresh button next to the alert symbols to update).
