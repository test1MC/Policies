Request Time-Off Procedure

Use the Request Time-off function in Kronos to request paid vacation days, personal holidays, jury duty time, and unavailable (unpaid time-off for doctor’s appointments, etc.). Multiple requests may be made at the same time. All requests will be submitted to your manager for approval.  Store manager requests will automatically approve. If approved by management, the auto-scheduler will account for this time-off and the requesting employee will not be scheduled during the specified time frame. 

Note: If you must request sick time or a bereavement (funeral) day, speak with someone in store management as they will need to schedule this time for you.
