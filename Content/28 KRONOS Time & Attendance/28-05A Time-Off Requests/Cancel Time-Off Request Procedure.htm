Cancel Time-Off Request Procedure

Cancel a time-off request from within your calendar. When canceling a multi-day request, the entire request will be canceled. If you only wish to cancel some of the request, a second request for the desired time off must be re-submitted.

Step

Action

1

Access Kronos using the Time Stamp button on the POS toolbar. Use your mySherwin credentials to log in.

2

Select My Calendar from the Widget Panel (right side of screen).

3

To cancel, hover over the request name and select the play button to the right of the request name.

4

Has your manager approved this request?

If Then
Yes Select Cancel.
No Select Retract.

5

Select Submit.

Note:  The request will still be visible in your calendar but will display a retraction/cancellation request when you hover over the request for details.
