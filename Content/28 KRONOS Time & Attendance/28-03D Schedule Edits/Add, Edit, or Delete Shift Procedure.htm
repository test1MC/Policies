Adding Shift

Select corresponding employee’s name -> Select Add shift -> Specify Start Time and End Time -> Select Apply.
Adding Shift Note

Select shift box -> Select Edit -> Add note in the Shift Label Field -> Select Apply.
Editing Shift

Select shift box -> Select Edit -> Make revisions -> Select Apply.

Note: Edit a Kronos-suggested shift when you want to change the length of a shift, adjust the start or end time, or re-assign the shift to another employee.
Deleting Shift

Select shift box -> Select More actions -> Select Delete.
