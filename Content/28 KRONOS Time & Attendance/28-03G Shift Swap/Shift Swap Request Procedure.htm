Shift Swap Request Procedure

Step Action

1

Access Kronos using the Time Stamp button on the POS toolbar. Use your mySherwin credentials to log in.

2

Select MyCalendar Widget.

3

Select Schedule Detail Tab.

4

Select More Dropdown.

5

Select Request Shift Swap.

6

Enter the date and time of the shift you want to swap.

7

Enter the date you would like to work.

8

Select the employee you are swapping with.

9

Select Submit.

Note:  The request is not official or scheduled until a manager has approved the swap. Check Inbox for approval notification.
