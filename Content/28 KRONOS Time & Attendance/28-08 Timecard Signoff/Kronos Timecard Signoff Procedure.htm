Kronos Timecard Signoff Procedure

By managing daily exceptions and approvals, pay period approval should be a rubber-stamp process, ensuring employees are paid correctly and on time.

Step

Action

1

Access Kronos using the Time Stamp button on the POS toolbar. Use your mySherwin credentials to log in.

2

Select Timecard Signoff from the Widget Panel (right side of screen).

3

Select Pay Period for Approvals Tab.

Step

Action

1

Select All Home and No Pre-Hires from the Hyperfind Dropdown.

2

Select the Time Period to approve: Previous Pay Period.

3

Select Save.

4

Select the Exceptions Tab.

Step

Action

1

If you see an employee with missed punches, make the necessary corrections on their timecard and approve the timecard.

5

Bypass the Employee Requests Tab by selecting the double arrows. You can manage these requests for future time-off in your daily review.  These do not need to be resolved in order to signoff on the timecards.

6

 

Select the Signoff Timecards Tab.

Step

Action

1

Select the Select All Rows Button.

2

Select the Approval Button -> SignOff.

3

Confirm that you do, indeed, wish to signoff when prompted.


7

To verify a successful signoff, select the Group Edit Results for Signoff Tab. Are there any failures?

If Then
Yes Proceed to Step 8.
No Proceed to Step 9.

8

Message

Resolution

Timecard already signed-off

No action required

Missed Punches

  • Correct missed punches
  • Perform signoff again

9

Clear the widget for the next pay period.
