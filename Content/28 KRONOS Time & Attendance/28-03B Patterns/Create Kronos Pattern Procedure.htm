Create Kronos Pattern Procedure

Step Action

1

Access Kronos using the Time Stamp button on the POS toolbar. Use your mySherwin credentials to log in.

2

Launch a full-screen view of the Schedule Planner by selecting the Schedule Planner Widget.

3

Select employee name -> Schedule Pattern.

4

On the Schedule Planner pane, select maximize to launch full-screen. A new Schedule Pattern Form will launch.

5

To add pattern, complete following fields:

Field Data Value
Anchor Date

The anchor date and start date should match and be set for the first day of the pay period.
Start Date

The anchor date and start date should match and be set for the first day of the pay period.
End Date

Date the pattern ends. Check Forever Box to lock pattern for foreseeable future.
Define Pattern Week

6

Set pattern:

Step Action

1

Select cells for days and type in hours to be worked.

2

Select Apply and OK.
