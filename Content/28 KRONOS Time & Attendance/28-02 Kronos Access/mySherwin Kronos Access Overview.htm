mySherwin Kronos Access Overview

Through mySherwin, you have access to the full Kronos application excluding the TimeStamp function to punch-in and punch-out. This provides managers and employees some flexibility in that these things can be reviewed remotely while not in the store; (anywhere mySherwin is available to you). 

Employee Type

Functions Available

Managers
  • Respond to Employee Requests
  • Resolve Alerts & Exceptions
  • Edit & Post Schedule
  • Timecard Signoff for Pay Period

Employees
  • Make Timeoff Requests
  • View Current & Future Schedule

Access Kronos from the MyApps section of mySherwin. If you do not see Kronos in myApps, select the Personalize Link and add Kronos to your MyApps.
