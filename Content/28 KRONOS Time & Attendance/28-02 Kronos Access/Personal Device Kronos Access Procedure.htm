Personal Device Kronos Access Procedure

When accessing Kronos through mySherwin from a personal device, please be aware of the following recommendations:

  • PCs and laptops are the recommended devices to use to access Kronos via mySherwin on a personal device.
  • The Kronos app for iPhones, Ipads, Android devices is not configured for use. You will not be able to access Kronos on these devices.
  • Device must be flash-enabled
  • Internet Explorer 11 (32- & 64-Bit) recommended for best performance. 
  • Other supported browsers include:
    • Chrome Browser (Google Chrome 41+)
    • Firefox 35+
    • Apple Safari 9 on OSX 10.9
  • Supported operating systems include:
    • Windows 8 (64-Bit only)
    • Windows 8.1 (64-Bit only)
    • Windows 7 (32- & 64-Bit)
    • Windows Server 2012 (64-Bit)
    • Windows Server 2012 R2 (64-Bit)
    • Windows 10

The following information is being provided to show how you may download Flash on various browsers. Please be aware that these instructions may be effected by the browser version you have installed on your personal device. Follow these instructions once you have attempted to access Kronos through mySherwin via your preferred browser:

Step

Action

Google Chrome

1

Browser will display a white page. Select the Shield Icon.

2

Select Load unsafe scripts.

3

Select Get Flash.

4

Select Allow.

5

Select anywhere on the page. When you see this icon, you will be logged into Kronos.

Microsoft Edge

1

Browser will display a white page. Select the Shield Icon.

2

Select See all content.

3

Select Get Flash.

4

Select Always Allow.

5

Again, the browser will display a white page. Select the Shield Icon.

6

Select See all content. You will be logged onto Kronos.

Microsoft Internet Explorer

1

Browser will display a white page. Select Show all content.  You will be logged into Kronos.

Mozilla Firefox

1

Browser will display a white page. Select the i Icon in the address bar and the > (next) button.

2

Select Disable protection for now.

3

The browser may ask you to install the Flash Player. Follow the on-screen instructions to install the Flash Player. 

4

Once the Flash Player is installed, close and open Firefox again.  Logon to mySherwin and Kronos, again.

5

Browser will display a white page. Select the i Icon in the address bar and the > (next) button.

6

Select More Information.

7

Select the Permission Tab and Allow under Activate Plugins/Adobe Flash. You will be logged onto Kronos.

As with any use of Sherwin applications on a personal device, your experience may differ from your usual or expected experience on your Sherwin device. You may call the IT Service Desk at 216-566-2997 with any questions. Be aware that they are unable to assist with technical difficulties on personal devices.
