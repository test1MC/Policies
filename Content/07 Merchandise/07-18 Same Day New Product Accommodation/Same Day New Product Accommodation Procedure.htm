Same Day New Product Accommodation Procedure

Applies Only to Automotive Branches

The Same Day New Product Accommodation is used when a customer requires product not available through POS same day. 

Step Action

1

Does your customer have an SW Charge in good standing?

If Then
Yes

Is this request being made in the last two business days of the month?

If Then
Yes

Process a deposit for the customer.

Note: For more information on how to process a deposit, refer toProcess Deposits Procedure
No

Process a SWeep request for the new sales number.
No

Process a deposit for the customer.

Note: For more information on how to process a deposit, refer to Process Deposits Procedure.

2

Complete a Manual B-PO Form and fax to the vendor to order the product. 

 Note:  Make arrangements with the vendor to pick up or have the product delivered to the branch.

3

Complete a Manual POST Form to release the material to the customer:

  • Use the correct part number.
  • Use the correct description.
  • Calculate price correctly to achieve appropriate Gross Margin.
  • Calculate tax correctly.

Once the new sales number has been set up, complete the following process to finalize the purchase order and customer order:

Step Action

1

Did your customer pay a deposit for the accommodation?

If Then
Yes

From POS desktop, select Store -> Customer Order Entry -> Update/Review Orders -> Search for and update the customer order created for the deposit:

Step Action

1

Delete the unprocessed Deposit Line Item.

2

Add the newly created Sales Number to the order with the correct quantity and pricing as shown on the Manual POST Form.
No

From POS desktop, select Store -> Customer Order Entry -> Add Order ->Create a Customer Order for the newly created Sales Number based on the Manual POST Form.

2

Create and release a confirming purchase order based on the Manual B-PO Form used to order the product from the vendor:

Step Action

1

Override the A-PO Number generated by MMS with the confirming B-PO Number on the Manual B-PO Form.

2

Select Special as your Order Option.

3

Select Yes when prompted to indicate if the order is Direct to the customer.

4

Select the related Customer Order to pull into the Purchase Order.

5

Press LOOKUP [F3] at the line items to bring the item(s) into the purchase order.

3

Did your customer provide a deposit for this order?

If Then
Yes

Process the Manual POST Form in POS as a Correction Transaction:

Step Action

1

From the POS desktop, select Store -> Point of Sale -> Select Enter & Validate Manual Form from the Sale/Payment Menu.

2

Enter the Manual Form Number.

3

Press PAGE DOWN and select Correction from the Special Menu.

4

Select the related Customer Order to pull into the Correction Transaction.

5

Enter the transaction information from the sale of the deposit.

6
  • Positive Side – Deposit (No Tax)
  • Negative Side – Product(s)
No

Process the Manual POST Form in POS as a Sale Transaction:

Step Action

1

From the POS desktop, select Store -> Point of Sale ->Select Enter & Validate Manual Form from the Sale/Payment Menu.

2

Enter the Manual Form Number.

3

Select Sale from the Sale/Payment Menu.

4

Select the related Customer Order to pull into the Sale Transaction.
