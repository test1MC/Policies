Place a Special Blend Order Procedure

Refer to the Division Blending Station Manual for current procedures. If a Blending Station Manual is not available, follow the procedures below:

Step

Action

Placing a new order for a blend

1

Fill out an Order for Special Blend form

2

When the blending station has made the special mix, it will assign a Permanent Blending Station REX Number to the color.

3

The blending station will ship the order freight collect, using an Inter Store Transfer (IST) Transaction.

4

When the IST is received, calculate the expected gross profit percentage and the selling price, using the unit cost on the IST.

5

Label the containers with the Class Number 8300 and the Appropriate Gross Profit Percentage.

6

Include the selling price on the label as well.

7

If the merchandise is shipped directly to the customer, the blending station will send the IST to the store the same day the material is shipped to the customer. When the IST is received, bill the customer accordingly.

Placing a repeat order for a blend

1

Telephone the blending station to place the order or complete a Stock Request (#23674). If the order is within the blending station criteria, the station will ship the completed order on an IST.

2

Calculate the expected gross profit percentage and selling price, using the unit cost listed on the IST.

3

Label the containers with class number 8300 and the appropriate gross profit.

4

Include the selling price on the label as well.

5

Handle GOM Breakdowns

6

Use the actual sales number of the product, the decimal equivalent for the quantity, and the selling price times 4, for the unit price.
