Duron Spray Equipment Transfer Procedure

This section does not apply to Automotive Branches, Product Finishes Facilities, MAB Paints Stores, or SW Paint Stores

In general, all spray equipment and parts should be ordered by stores on their regularly scheduled STAR orders.  Customer Transfer Orders and the Store Stock IST process should only be used to fulfill immediate emergency need for delivery.  If a Duron store has an immediate need for spray equipment and/or accessories, which cannot wait for the next STAR Order delivery the store may:

If

Then

Place a Will-Call Order on the warehouse

Coordinate the order with Cleveland Customer Service [1-866-DURON-98] and the warehouse Shipping Office.

Place an order with the SDC

The order must be for delivery.  If the order is not for delivery, place a Will-Call on the warehouse.

Place an order with the Spray Shop using the Transfer Order Process

This process should only be used if the warehouse/SDC does not have the stock to fulfill the order and the Spray Shop has the stock in their store inventory.  The Spray Shop will need to coordinate any delivery requests with the SDC.
