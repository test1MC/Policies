KST Orders Policy

  1. Stores should create an Emergency, Customer Direct, or Will-Call Order; however, should NOT release this order in the system.  If the SMPO Code for pricing is known, this should be included in Special Instructions.
  2. Stores MUST call Customer Service [ARCH: 1-800-797-8010 #3] before releasing the order in SOURCE. Customer Service will instruct the store to release their order in SOURCE when appropriate.
  3. Products requiring special pricing may not be placed on a STAR Order.  These products must be placed on an order by themselves using one of the following order types: Emergency, Customer Direct, or Will Call.
