Old-Age Orders Policy

This section does not apply to Automotive Branches.

Follow the process for placing Southeast Division Old-Age Inventory Orders. Eastern, Midwestern, and South West Divisions may order old-age inventory using the Garage Sale program available through mypaintconnection.

  1. Stores should create an Emergency Order or Customer Direct Order for their Old Age Order but should NOT release this order in the system.
  2. Stores MUST call Customer Service [Arch: 1-888-677-8673] before releasing the order in SOURCE.  Customer Service will instruct the store to release their order in SOURCE when appropriate.
