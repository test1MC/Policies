Marine Orders Policy

This section does not apply to Automotive Branches

  1. Marine Paint Orders should be placed using the Emergency Purchase Order. No Emergency Order fee will be assessed for placing this order.
  2. Stores should create an Emergency Order but should NOT release this order in the system.
  3. Stores MUST call Customer Service [Arch: 1-888-677-8673 / CC: 1-888-792-4364] before releasing the order in SOURCE. Customer Service will instruct the store to release their order in SOURCE when appropriate.
