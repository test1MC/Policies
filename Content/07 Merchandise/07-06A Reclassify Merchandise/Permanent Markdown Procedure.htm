Permanent Markdown Procedure

Step Action

1

Stores Group buyers will send the Markdown notice to all stores prior to the markdown date.

Note: Advertising initiates the selling price markdown each time a new selling price goes into effect (via monthly Point of Purchase POP kit).

2

The POP kit will contain current selling price sign(s) for the month. If you do not have stock on an item simply discard the sign(s). Place the sign in front of the products so that the display is clearly marked.

Note: Do not mark down the merchandise prior to the markdown period because the computer system will only correctly price during the identified markdown period.

3

The computer system will automatically price the customer's invoice correctly while processing a sale transaction for the marked down merchandise.
