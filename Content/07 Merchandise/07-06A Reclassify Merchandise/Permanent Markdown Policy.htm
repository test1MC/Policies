Permanent Markdown Policy

  1. You are required to comply with the Permanent Markdown Notices to eliminate discontinued/obsolete merchandise.
  2. All items on Permanent Markdown are to be assumed store stock only. At the time of receiving the markdown, no product will be available from the DSCs. No product is to be ordered from vendors for sale at Markdown Prices.
  3. Do not change the markdown on products, unless otherwise indicated on the notice.
  4. The gondola should be merchandised to the current Plan-o-gram once the markdown merchandise is completely sold from the shelf. If the store receives a new Planogram Book, set the gondolas to the new plan immediately.
