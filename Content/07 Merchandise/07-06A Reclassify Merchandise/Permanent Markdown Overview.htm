Permanent Markdown Overview

The Permanent Markdown process provides a system where discontinued/obsolete products are marked down in all Sherwin-Williams' stores to expedite the sale on select merchandise. Min/Max levels will be automatically set to zero to prevent reorders from the DSC for all discontinued/obsolete merchandise.

Note: All Permanent Markdown selling prices will end in $.00. This is to [Flag] that the product is discontinued/obsolete and unavailable for reorder. Point of Purchase (POP) signs for all markdowns will show original List Price and new selling price (showing [value] to customer) to entice quicker sales.
