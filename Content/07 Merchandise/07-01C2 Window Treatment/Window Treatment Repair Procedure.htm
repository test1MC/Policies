Window Treatment Repair Procedure

Occasionally, you will need to return window treatment to the vendor for repair. When this happens, process a purchase order to the vendor.

Step Action

1

From the POS Desktop -> Main Menu -> Customer Order Entry -> Add Order -> Enter the Vendor/Style Number associated with the window treatment that is being repaired.

2

Instead of entering the width and height, enter the word Repair or the name of the part being ordered in the width and height fields -> Press ACCEPT [F1] and leave all other fields blank

3

After entering the quantity and cost, press the COMMENTS [F9] key to capture additional information about the repair.

4

Receive the repaired window treatment against the purchase order when it arrives at the store.

5

Sell the repair or part using the appropriate Vendor/Style Number.
