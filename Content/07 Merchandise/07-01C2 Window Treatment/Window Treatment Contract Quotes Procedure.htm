Window Treatment Contract Quotes Procedure

Step

Action

1

From the POS Desktop -> Main Menu -> Customer Order Entry -> Add Order -> Enter the Vendor Number or use LOOKUP [F3] when prompted for the vendor -> Enter the appropriate style number for the window treatment.

2

When the Options bar pops up on the screen, enter the contract quote name instead of entering the Width and press ACCEPT [F1].

Note: It is not necessary to enter each blink for a contract quote.

3

Enter a quantity of one -> Enter the total unit cost for the contract quote.

Note: Press the COMMENTS [F9] key to capture any specific window treatment information that you were not able to include in the Options area.

4

Once the purchase order is completed, call the vendor to place the order -> Release the purchase order.
