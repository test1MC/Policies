View Window Treatment Class Numbers Procedure

There are two ways to view window treatment class numbers:

Option

Action

1

From the Line Item Screen in POS/COE -> press LOOKUP [F3] -> Dept., Sub., Cat., Desc -> Draperies & Window Treatment

2

Main Menu -> Merchandise Management -> Reports -> Window Usage By All Parts -> Enter -> Draperies & Window Treatment -> Appropriate Sub Department -> All Products
