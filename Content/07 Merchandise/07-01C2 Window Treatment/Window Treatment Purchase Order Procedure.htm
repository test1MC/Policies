Window Treatment Purchase Order Procedure

For measuring Window Treatments, see Window Treatment Measuring Guide. Use the information captured on the Window Treatment Proposal and Contract Form to process the Purchase Order:

Step

Action

1

From the POS Desktop -> Main Menu -> Customer Order Entry -> Add Order -> Enter the Vendor Number or use LOOKUP [F3] when prompted for the vendor.

2

Continue entering the required purchase order information until you reach the sales number field -> The Window Treatment sales numbers consists of the Style Number, Width & Height.

3

Use Manufacturer Name when reviewing the lookup procedures to identify a style number.

4

 

 

 

 

 

After entering the style number for the window treatment, an Options Bar will launch with the following fields:

  • Width
  • Height
  • Color Name
  • Number
  • Mount (Horizontal or Vertical)

5

Complete the fields if you have the applicable information. If not, leave the field blank.

6

Select appropriate window treatment from the Orientation Menu:

If

Then

Horizontal Window Treatment

Choose the wand position and press Enter.

Vertical Window Treatment

Choose the stacking method and press Enter.

7

Select the cord position, then press Enter -> Press ACCEPT [F1] once finished with the Options Bar.

8

Select Yes if there is a valance and No if there is no valance.

9

If the valance is for a vertical window treatment, enter the Style and Cost of the valance

Note: If there is no additional cost for the valance, enter the cost as zero.

Ordering Only a Window Treatment Valance

1

If

Then

An Employee places an order for only a valance

Enter all of the above information with a cost of zero -> Enter the valance style and cost

A Horizontal Window Treatment was selected

Nothing will show

2

After pressing ACCEPT [F1], the following message will return to the screen: The unit cost entered for the valance will automatically add to the cost you enter for the window treatment.

3

After the message disappears, the system will prompt you to enter the quantity for the order and the unit cost for the blind.

Note: Remember, if you entered a unit cost for a valance, it will automatically add to whatever unit cost you entered for the blind.

4

Press the COMMENTS [F9] key to capture any specific window treatment information that you were unable to include in the Options area.

5

Complete the remainder of the purchase order and call the vendor to place the order. Once you have called the vendor, make sure you release the purchase order.

Note: For more information on releasing purchase orders, see External Purchase Order Procedure.

6

After entering the above information, if you need to change the unit cost for the blind, valance associated costs will not be automatically added back in.

  • Add the new unit cost for the blind with the unit cost for the valance and enter the total unit cost.
  • If you attempt to make a change to the unit cost, you will see the following message: You must add the unit cost of the valance to the new unit cost you are entering for the blind. This will not be done automatically.

For information on DIY Cash on Completion transactions, see DIY Cash on Completion Policy or DIY Cash on Completion Transaction Procedure.
