Aerosol Vendor Can Fill Request Procedure

Follow steps to generate a vendor purchase order for outside vendors to fill aerosol cans with S-W paint.  While processing an External Purchase Order, be aware of the following special information to be used at the Item Detail Screen.  For more information on creating External Purchase Orders, see 07-01B External Purchase Order.

Use 457-6831.

Sales Number

Press COMMENTS [F9] and enter the following information for the paint being used to fill the cans:

  • Sales Number
  • REX Number
  • Product Description
  • Quantity

Note that in addition to requesting this vendor service, you will also need to adjust inventory to account for both the cans and the product being used to fill the cans.  While processing the Perpetual Adjustments, be aware of the following special information to be used.  For more information on processing Perpetual Adjustments,  see Make a Perpetual Adjustment Procedure.
