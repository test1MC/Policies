Add a Memo Demand Procedure

The Memo Demand feature enables store employees to enter customer requests into the store computer system at any time. These reminder messages can be incorporated into weekly STAR Orders to ensure the correct amount of product is ordered. Memo Demands can be attached to a product for ordering purposes. Enter a Memo Demand anytime a customer requests merchandise that will be ordered on a future STAR Order or while processing a Standard STAR Order.

Step

Action

1

From the POS Desktop, select Main Menu -> Merchandise Management -> Memo Demand.

2

Select Add.

Note: If there is already a Memo Demand for the product for the same date required, select Change.

3

Add the new quantity and additional comment to the existing Memo Demand.

Note: You are allowed to have one Memo Demand per product per date required.

4

Include customer specific information in the Memo area. Add additional customer information for multiple orders.
