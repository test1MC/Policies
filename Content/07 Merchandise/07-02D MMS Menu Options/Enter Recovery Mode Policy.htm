Enter Recovery Mode Policy

  1. In the event of a serious system failure, re-entry of all lost Purchase Order and inventory information will be entered at your store.
  2. This recovery process should only be used if instructed by the Store Computer Help Desk.
  3. Enter only transactions that have previously been processed when instructed by the Help Desk.
  4. Any Purchase Order originally entered on the store computer system must be entered in Recovery.
  5. The transactions will be flagged as having been created in Recovery Mode.
  6. Any transaction entered in Recovery must mirror the original entry.

Note:  Call Store Computer Operations at 216-566-2883.
