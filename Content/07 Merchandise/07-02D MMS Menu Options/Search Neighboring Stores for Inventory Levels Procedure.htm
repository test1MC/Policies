Search Neighboring Stores for Inventory Levels Procedure

The Search Store table has been automatically set-up with up to 40 stores in each District. Store employees will be able to access inventory levels for each store in their districts by selecting Inquiry and then Products and Prices.

Step

Action

1

From the POS Desktop, select Main Menu -> Merchandise Management -> Maintenance -> Search Stores.

2

Press Lookup [F3] to enter a complete or partial store name:

  • To add a store number, go to the end of the list and enter the new store number.
  • To delete a store, press the Delete key with the highlight bar on the correct line.
