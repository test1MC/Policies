Close a Purchase Order Procedure

Step

Action

1

From POS desktop -> Main Menu -> Merchandise Management -> Receive/Verify.

2

Select Order and enter the received order number.

3

After receiving all merchandise, press Accept [F1].

4

Enter the actual receipt date and freight charges along with any special instructions.

5

Accept all in-process receipts.

6

Once all in-process receipts are accepted, the store has approval from Headquarters to pay for the received merchandise.

Note: No adjustments can be made once the Purchase Order is closed. However, if changes need to be made, do so before answering Yes when prompted to close the Purchase Order. If you discover a receiving error after the Purchase Order has been received and closed, the Store Manager must perform an Inventory Adjustment capturing the received Purchase Order number. For more information, see Inventory Adjustment Overview

Note: If possible, the closing process should be performed on the Manager's office system so that the Point-Of Sale terminals are available to process customer transactions.
