Receive by Product

Scan the Universal Product Code (UPC) or enter the sales number. Each Purchase Order containing the sales number entered will return to the screen. Select the proper Purchase Order and continue.

Select ORDER from the Receiving Options List. A window will return to the screen that lists all Released, In-Process, In-Recvng, and Part Rcpt Purchase Orders. Select the proper Purchase Order and continue.

Using the hand-held scanner to receive merchandise. Select the appropriate ISR Purchase Order.

4

Scan each item and enter the quantity received. You will be asked if the receiving is completed for each line. Press Accept [F1] when complete.

Receive ISR Variances
1 An employee with at least level-two security, must review and accept the order on the store computer system. Press View [F7] while reviewing the received Purchase Order for accuracy. You will be taken quickly to each line that is not complete, noted by an N in the complete column.
2 To identify the proper Purchase Order number, look at the first page of the BOL.
3 Select the proper Purchase Order to continue.
4

Review the Previously Received (PRV-Q) and Received (RCV-Q) Columns on the purchase order.

While going through the verification and approval process ask the following questions if there is a receiving variance:

Question Action
Is the product substituted for another product? Reference the instructions outlined for substitutions
Is the product a result of a picking error for another product? Reference the instructions outlined on the front of this Newsletter for picking errors.
Was the merchandise received against the wrong Purchase Order?
  • Verify the Purchase Order number. Check to see if more than one purchase order was made for the product.
