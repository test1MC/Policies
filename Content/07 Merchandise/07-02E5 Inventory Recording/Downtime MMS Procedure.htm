Downtime MMS Procedure

Downtime refers to the period of time that the equipment is inoperable. When the store computer system is not operating properly, process a Manual Purchase Order:

Step

Action

1

Use the Manual Purchase Order for all orders, both Internal and External, and list the following information:

  • Sales Number
  • Quantity
  • Unit Price
  • Calculated Total

Note: For external orders, use the manual Purchase Order number when placing the order with the vendor. For information on downtime STAR Order processing see STAR Order if the Computer is Inoperable. Create a STAR Order on a manual Purchase Order during downtime. Make an estimate based on stock requirements. If possible, when your system comes back up, place an Add to STAR Order for overlooked items.

2

When the computer system is operable, enter all purchasing information. Remember to enter the manual Purchase Order number instead of the system generated Purchase Order number in order to avoid duplicate shipments. Customer Service has been instructed not to automatically fill Manual B Prefix Purchase Orders processed through the computer system.

Note: If the computer system is fixed while writing a manual Purchase Order for the STAR Order and it has already been called in, do not use the manual Purchase Order number when putting the order on the system.

Note: There are no downtime procedures for processing an IBAR or Chargeback. Wait until the computer system is operational to process IBAR's and Chargebacks.
