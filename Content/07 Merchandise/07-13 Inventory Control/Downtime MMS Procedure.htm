Downtime MMS Procedure

Downtime refers to the period of time that the equipment is inoperable. When the store computer system is not operating properly, please follow the downtime procedures listed below:

Step

Action

1

Use the MANUAL PURCHASE ORDER for all orders, Internal and External. List each merchandise item by sales number, quantity, and unit price and calculate the total manually. For external orders use the manual Purchase Order number when placing the order with the vendor.

Note: For information on downtime S.T.A.R. Order processing see S.T.A.R. ORDER IF THE COMPUTER IS INOPERABLE:

Step

Action

1

Create a S.T.A.R. Order on a manual Purchase Order during downtime.

2

Make the best guess based on stock requirements.

3

If possible, when the system comes back up, place an Add to S.T.A.R. Order for overlooked items.

2

When the computer system is operable, enter all purchasing information. Remember to enter the manual Purchase Order number instead of using the system-generated Purchase Order number. Customer Service has been instructed not to automatically fill manual (B prefix) Purchase Orders processed through the computer system.

WARNING: If an employee forgets to enter the manual Purchase Order number for internal order, the store may receive a duplicate shipment.

3

If the computer system is fixed while writing a manual Purchase Order for the S.T.A.R. Order and the order has not yet been called in, DO NOT use the manual Purchase Order number when putting the order on the system. Remember, Customer Service has been instructed not to automatically fill manual (B prefix) Purchase Orders processed through the computer system.

Note: There are no downtime procedures for processing an IBAR or Chargeback. Wait until the computer system is operational to process IBAR's and Chargebacks.
