Create Bin Location & Record Stock Procedure

For locations that use bins for inventory stock, the bin location function within MMS provides a way to record stock by bin location. Bin location information may be used to run store inventory reports by bin location and may also be used to generate Customer Order Pick Lists by bin location. Bin location information cannot be used in the annual inventory count process as a way to establish counting locations.

Step

Action

1

From the POS Desktop, go to Main Menu -> Merchandise Management -> Inventory Control.

2

From the MMS Inventory Control Menu, select Bin Location Maintenance.

3

Enter a sales number or press Lookup [F3] to access Product Lookup. Query for the desired part to add to a bin location. UPC bar codes can be scanned as well.  Employees may also enter the REX Number at the sales number field in Bin Location Maintenance.

4

After selecting the product, existing bin locations for that product will return to the screen if bin locations have previously been set up for the product. Either select to add product to an existing bin location or type in a new bin location. If adding the product to more than one bin location, type in or select as many bin locations as necessary.

5

To designate one bin location as the primary bin location, type or select the name and press TAB. Enter the character P to signify the bin location as the primary and press Enter. If applicable to the product, enter a batch number(s), press Accept [F1]. Once complete, press Accept [F1].

Note: Bin Location Names are alphanumeric and may be up to six positions in length. The names are free form and stores may establish their own Bin Location Name logic.
