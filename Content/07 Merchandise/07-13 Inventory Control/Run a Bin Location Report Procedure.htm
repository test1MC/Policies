Run a Bin Location Report Procedure

Step

Action

1

From the POS Desktop, go to Main Menu -> Merchandise Management -> Reports -> Bin Location Reports.

2

Reports may be ordered by either a specific Sales Number or specific Bin Location. The Lookup [F3] function will work on either field for narrowing criteria. Or, both fields may be left blank in order to review all products in all bins.

Note: Reports may be sorted for viewing by Sales Number, Bin Location, Dept-Sub-Cat, Pattern/Blend ID or Batch Number.

3

Press Print Report to print it, if necessary.

Note: Bin Location reports may also be created in Inquiry.
