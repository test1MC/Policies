Verify Sales Numbers for Class-Controlled Products Procedure

Generate an Inventory Adjustment report to verify that sales numbers have been assigned for all class-controlled products. Check that all class-controlled products have been assigned sales numbers through an Inventory Adjustment:

Step

Action

1

From the POS Desktop, go to Main Menu -> Merchandise Management ->Reports -> Inventory Adjustment Report.

2

Enter the first date of District MMS training. Press Enter to accept.

3

When prompted, Enter Sales/Product Number and Enter Size Code. Press Enter.

4

Select View on the Screen from the Options menu.

Note: Complete Inventory Adjustment information will return to the screen. All Inventory Adjustments made to establish sales numbers for class-controlled merchandise will contain the memo Initial Product Setup.

5

Examine previous class-controlled merchandise to ensure that all products have been relabeled with the new sales number. Failure to relabel products could result in a product sold using the incorrect sales number, which would create a variance with the perpetual inventory. All class number references must be removed when a product is assigned a sales number.

Note: Products should not have price tags containing the class number.

Step

Action

Check that all class-controlled products have been re-labeled with the new sales number:

1

Walk around the sales floor and carefully examine all local buy products and spray equipment parts.

2

Repeat step 5 of the above procedure, if necessary.

3

Confirm that all sales now use the new sales number. To check that this is a correct statement complete the following steps:

Step

Action

1

Press Inquiry [F4]. Select Inquiry Menu.

Note: All products must have assigned sales numbers by day seventeen of MMS Implementation.

2

Enter day seventeen as your start date. Press Enter to accept.

3

Enter one of the old class numbers like 7666, 7575, or 7567 into the sale number field.

Note: For more information, refer to 04-01 Class Number.

Note: No sales transactions should appear, as all sales must be processed using the assigned sales number.
