Reconcile Variances in Cycle Counts Procedure

This procedure will be required going forward and will be reviewed by Field Audit and District Management. For more information, refer to New Inventory Training.

Step

 Reconciling Cycle Counts

1

Once all weekly Cycle Counts have been completed, select Cycle Counts from the MMS Inventory Control Menu.

2

Select Explain –Reconcile Variances in Cycle Counts.


3

The main window will list the line items in which the perpetual quantity did not match the actual quantity counted in the weekly cycle count period of 7 days. Any variance which has not been reconciled will be indicated with a No in the Rec? field.

4

Try to identify the reason for the variances via Unit Movement Report and then process the appropriate POS transaction. (i.e. Correction, Inventory Adjustment, Etc.)

5

Has an inventory adjustment been processed to correct the variance?

If Then
Yes View the Inventory Adjustments for the product by pressing
ACCEPT [F1].  Highlight the Inventory Adjustment that was done
to account for the variances and press ENTER to select it.
No Proceed to step 6.

6

Do you need to complete an Inventory Adjustment for this variance?

If Then
Yes

Press VIEW [F7] to create an Inventory Adjustment within the Explain
–Reconcile Variances in Cycle Counts Report. The system will create the
Inventory Adjustment for the quantity counted during the Cycle Count and will
ask if you would like to change the Rec? field to YES.
No Proceed to step 7.

7

Press TAB to navigate to the Memo field to explain the variance. Note the related POS transaction used to correct the variance. 

8

Once the variance has been explained in the Memo field, press TAB to exit the Memo screen.  Press  ACCEPT [F1] to mark the cycle count as reconciled.
