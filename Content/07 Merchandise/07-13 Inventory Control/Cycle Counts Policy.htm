Cycle Counts Policy

  1. Weekly cycle counts must include all of the following:
    1. All items on the store-specific Cycle Count - Weekly Report found in Store Reports on the SOURCE Intranet. The Cycle Count - Weekly Report is available on Sunday of each week.
    2. Negative balances found on the Negative Inventory Report. 
    3. Sales floor items, either plan-o-gram or not, for which there is no stock upon the sales floor.
    4. Each week the Cycle Count Report will contain five colorant SKUs.  Only include unopened containers in the cycle count.
  2. Locations are required to perform minimum unique cycle counts during each cycle count period as follows:
    1. TAG locations:  fifty (50) unique cycle counts.
    2. Automotive branches:  twenty-five (25) unique cycle counts.
    3. Product Finishes Group: twenty-five (25) unique cycle counts.
    4. Dedicated SpraySOURCE facilities: twenty-five (25) unique cycle counts.
  3. Stores have from Sunday through Saturday to record cycle counts for the week.  The cycle count process may be completed as a single process or throughout the week. 
  4. All inventory variances revealed in the cycle count process must be reconciled through the Explain –Reconcile Variances in Cycle Counts function on the POS by the Tuesday following the calendar week in which the cycle count was performed.
  5. It is critical to verify the accuracy of shelf label pricing while performing cycle counts and correct inaccurate shelf labels using Shelf Labels on Demand immediately.  This is critical to avoid fines assessed by state and local agencies for displaying incorrect pricing.
  6. The District Manager and Auditor will review cycle counts.
  7. Employees are not required to perform cycle counts if the store computer system is not working properly because there is no downtime cycle counting process. As soon as the computer system is operating properly, resume cycle counts.
