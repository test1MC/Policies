Verify Weekly Cycle Counts & Resolve Variances Procedure

If

Then

Verify Cycle Counts

Step

Action

1

From the POS Desktop, go to Main Menu -> Merchandise Management Menu -> Cycle Count Report.

2

Enter the Start Date. Press Accept [F1] and select View on this Screen.

Resolve a Variance

Step

Action

1

Select one item on the previously printed Cycle Count Report that contains a variance.

2

Press Inquiry [F4] -> Unit Movement.

3

Enter the sales number.

4

Enter the date of the cycle count as the From Date and press Enter to accept the To Date.

5

Review the report to see if an expense or correction transaction or an inventory adjustment was performed to correct the variance. 

Note: If the transaction was performed, all variances between the perpetual inventory quantity and the actual on hand quantity have been corrected.
