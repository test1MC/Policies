Automotive Emergency Purchase Orders Procedure

Step

Action

1

From the POS Desktop, go to Main Menu -> Merchandise Management -> Purchase Order-> Add New Purchase Order.

2

Select Emergency Order from the Order Type List. Press Enter.

Note: The "A" Purchase Order Number will automatically generated

If the Purchase Order was created during downtime using the B-Prefix Manual Purchase Order form, override this number with the B Purchase Order Number.

3

Enter the Sales Numbers and Quantities required. Press Accept [F1].

4

Enter the Estimated Receipt Date.

5

Once the order information is complete, release the order.  If the order is successfully released, POS will return to the Purchase Order Menu.  If the order has not been successfully released, there will be an error message. Follow the EDI Downtime Procedure.
