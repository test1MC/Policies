Make-N-Ship Order Procedure

Step Action

1

From the POS Desktop, go to Main Menu - > Merchandise Management -> Purchase Order -> Add New Purchase Order.

2

Select Make-N-Ship Order from the Order Type List.

Note: The "A" Purchase Order Number will be automatically generated. If the Purchase Order was created during downtime using the B-Prefix Manual Purchase Order form, please override this number with the "B" Purchase Order Number.

3

Select a Customer Account Number. Default or edit the ShipTo Address returned for the account.

4

Enter any Deposit Terminal and Transaction information. Press Enter.

Note: Enter Customer Purchase Order Number information, if required.

5

Enter the required Sales Numbers and Quantities. Press Accept [F1].

Note: Make-N-Ship Order requirements must be indicated at the Line Item COMMENTS [F9] level. Indicate requirements such as fill and label information.   Stores must also communication this information to Customer Service via phone. 

6

Enter the Estimated Receipt Date.

7

Instructions for special handling should be included in Special Instructions. This information may include, but, is not limited to, information such as order delivery date/time, phone calls required to schedule delivery, specific LTL carrier, pallet jack required, lift gate required, etc.  Stores must also communication this information to Customer Service via phone.  For more information, see Order Text
