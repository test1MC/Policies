Purchase Order Address Options Overview

When creating ISR, Customer Direct, or Make-n-Ship Purchase Orders, you are able to select ShipTo Address options.  You will be permitted to select from the BillTo Address, Store Address, PRC ShipTo Address, or provided with an option to add another address.

You will also be provided with a preview of the selected address for verification.

While creating an ISR or a Customer Direct Purchase Order, the Address Options Window will pop-up after tabbing off the Customer PO Field or it may be accessed by using Lookup [F3] while the cursor is positioned in any of the ShipTo Address fields.

While creating a Make-n-Ship Purchase Order, the Address Options Window will automatically pop-up when the “Special” has been selected as the Order Option, “Yes” has been selected for the Direct Option, and a Customer Order has been selected from which to create the Make-n-Ship Order or the user presses Cancel [F5] to exit the COE Browser.
