Check the Status of MMS IBAR Procedure

You can view the status of any IBAR relating to your store. IBAR numbers can be searched and sorted by:

  • Date Range
  • IBAR Number
  • Problem Number
  • Status

Step

Action

1

Select the Search Type.

2

Select the Date Range.

Note: The date range is automatically populated with today’s date in the End Date field and twelve days prior in the Start Date field.

3

Click the Search Button to display IBAR activity.

Note: IBARs requiring action will have the word Response displayed on the screen.

IBAR Status Code Description

In Process

Under review by Store Billing.

Closed

IBAR has been processed. Please click on details link in My Paint for more information.

Void

IBAR has been processed. Please click on details link in My Paint for more information.

Error

For more information regarding the error related to this IBAR, please contact Stores Billing at (216) 566-2297 option 3, option 4.
