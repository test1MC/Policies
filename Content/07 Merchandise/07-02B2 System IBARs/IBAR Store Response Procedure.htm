IBAR Store Response Procedure

Step

Action

1

Select the Response Option on the far right of the screen to view IBAR Details.

2

Complete a store response?

If Then
Yes Select Response.
No Select Search Results to return to the IBAR Inquiry screen.

3

Should this IBAR be voided?

If Then
Yes Select the yes button.
No Select the no button.

Note: If you select no, there will be additional questions to answer.

4

Enter comments in the Comment Field.

Note: The more information you provide will assist with processing the IBAR correctly.

5

Select the Submit button to complete the transaction.

6

The system will confirm successful completion and submission of information.
