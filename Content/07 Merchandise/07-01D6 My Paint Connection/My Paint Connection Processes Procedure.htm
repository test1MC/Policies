My Paint Connection Processes Procedure

Step Action
Shipment Inquiry

1

Select Orders -> Shipment Inquiry .

2

Under Search Type, select Date Range.

Note: If looking for a specific order, select Order # or PO #.

3

Enter the search parameters.

4

From the listing, select Details to access specific product information or select Track to follow the shipment of a product.

  • Note: The Track option is only available for CTS shipments. In order to track other carrier shipments, use the direct link to the website.
Order Inquiry

1

Select Orders -> Order Inquiry.

2

Under Search Type, select Date Range.

Note: If looking for a specific order, select Order # or PO #.

3

Enter the search parameters.

4

From the listing, select Details to access specific product information.

5

Print the Bill of Lading or packing list.
Product Search

1

Select Products -> Product Search.

2

Select Advanced Search.

Enter the REX number, including the zeros.

Note:

If Then

Searching for all available sizes,

Enter the percent [%] sign after the REX number.

Searching for a specific size,

Enter a dash [-] followed by the two-digit size code. 

3

Click Find Products.

4

From the listing, select Details to access specific SKU information.

5

From Product Details, select Buying History Report or Print SDS.

6

From Inventory Availability, select Show ALL Inventory to view the batch dates of the inventory at the DSC.
Garage Sale Search

1

Select Products -> Garage Sale.

2

Read and Accept the Old Age Product Disclaimer.

3

Enter specific product or click Find Products to search all available products.
Buying History Search

1

Select Order -> Buying History.

2

Enter Product Number and Start-End Ship Dates.

3

Click Fetch Report.
IBAR Inquiry

1

Select IBARs -> IBAR Inquiry.

2

Under Search Type, select Date Range.

Note: If looking for a specific IBAR, enter IBAR # or e-trac Ref #.

3

Enter the search parameters.
