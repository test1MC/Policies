My Paint Connection Overview

My Paint Connection is an online order and inventory tracking system that allows stores to stay connected with the Distribution Centers. It can be accessed from the SOURCE homepage-> Store Operations-> Lookup->My Paint Connection.

Use My Paint Connection to:

  • Track shipments
  • Check inventory levels
  • Look up batch dates, packing slips, or IBARS

Products

Menu Selection

Description

Product Search

This function is used to inquire about general or specific product information. It provides specific details about the product, as well as,  inventory availability. Also, it allows access to Material Safety Data Sheets (SDS) and Environmental Data Sheets (EDS).

Garage Sale

This function is used to search inventory and product availability in the old age warehouse.

Orders

Menu Selection

Description

Order Inquiry

This function is used to provide real time access to and retrieve the specific details regarding an order.

Shipment Inquiry

This function is used to access any current shipments. The TMW tracking system shows the arrival and departure times for all CTS shipment stops and allows stores to track delivery times. Stores can use direct links to most websites to track other carrier shipments.

Buying History

This function is used to search for all orders placed on a specific product.

IBARS

IBAR Inquiry is used to search for specific IBARs from the store.

User Admin

User Profile and Change Password are available to help manage user settings.

Help

FAQ and Feedback are available to help users field any questions and pose questions or concerns about My Paint Connection.
