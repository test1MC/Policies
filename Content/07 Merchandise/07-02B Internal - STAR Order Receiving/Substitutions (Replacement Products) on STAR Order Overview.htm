Substitutions (Replacement Products) on STAR Order Overview

Stores often receive unordered products because these are replacement products for the sales number that you ordered. When the order gets to the DSC, the order quantity is automatically switched to the replacement product. Substitutions due to warehouse holds are on the Order Advisory. Other items that do not arrive because of warehouse stock outages are identified through MyPaintConnection. When scanning the ASN License Plate and receiving the order at the store, the purchase order will reflect the following for the old and replacement products.

A message will notify a sales number change whenever you try to enter a sales number that has been replaced by a new sales number. After the message is displayed, the sales number that was keyed in is replaced with the new sales number in the input field. If this message is received, update the Min/Max Levels for both numbers.  Adjusting the Min/Max Levels will ensure that you will order the most current product.

This message will only appear in the following locations:

  • Inventory Adjustment
  • Conversion Adjustment
  • Chargebacks
  • Purchase Order Entry
