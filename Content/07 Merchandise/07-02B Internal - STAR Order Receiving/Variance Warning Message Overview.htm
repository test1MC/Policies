Variance Warning Message Overview

While closing Internal STAR Purchase Orders, if there are variances between the SHP-Q and RCV-Q quantities and there have been zero receipts for the shipment, or if none of the products for a given shipment have been received, the following message will appear:

The following products have variances between the SHP-Q and the RCV-Q quantities. The system will generate an IBAR for these products. These products might have been shipped from a different DSC than the other products on this order or a mistake could have occurred in Shipping/Receiving

Only close this PO if you are certain receiving is completely finished.
