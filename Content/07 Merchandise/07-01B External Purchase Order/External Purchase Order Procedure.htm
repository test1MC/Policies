External Purchase Order Procedure

Step

Action

1

Prepare the purchase order for placement and calculate the total value. If the total value of the purchase order, or the aggregate value of two or more purchase orders to the same vendor, is more than $3,000, contact the District Manager for approval.

Note: The District Financial Manager may also be contacted in the South Western Division.

2

Update the vendor phone and fax number the next time you place an external purchase order to the vendor.

3

Determine whether or not the vendor is EDI-capable:

If

Then

Yes

Proceed to Step 4.

No

Give the vendor the complete Purchase Order number as it appears on the screen. Provide the store number or the account number while placing the order.

4

Obtain the inventory status of the order from the vendor.

If

Then

Back-ordered and is for an individual customer

Place the order and contact the customer to inform them of the back-order. If the back-order date is not acceptable to the customer, call the vendor to see if the order has been shipped. If it has not been shipped, cancel the order immediately by selecting Cancel Unnecessary Purchase Order from the Purchase Order Menu.

Note: Remember to call the vendor to cancel the order prior to canceling it on the system. Don't cancel the order if already shipped.
For more information, see Cancel a Purchase Order Policy.

In-Stock

Proceed to Step 5.

5

Record a reference number and the order taker's name, as well as, all pertinent information provided by the vendor in Special Instructions area. Press Accept [F1] when finished.

Note: Purchase Order comments can be added at any time. Press Comments [F9] or select Yes when prompted for Special Instructions.
