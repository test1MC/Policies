Product Not Available Internally Error Message Procedure

If the error message: “This product is not available internally” appears, but it is available at the DSC, then Lookup the Product:

Step

Action

1

Press Lookup [F3] when prompted to enter the sales number.

2

Select Sales Number and press Enter.

Note: The sales number in the database will return to the screen.

3

Press Lookup [F3] to search the Headquarters Database for the sales number.

Note: After locating the sales number on the Headquarters Database, press Enter with the highlight bar on the sales number to update the information in your store.

4

Enter the sales number again when prompted:

If

Then

An error message appears

Complete a New Vendor Request Form available through SWeep.

Note: While filling out the form, add a comment identifying the problem and complete a Manual [B] Prefix Purchase Order to immediately order the product.

There is no error

The procedure is complete.

Note: Overwrite the system generated purchase order number with the "B" prefix number.
