External Merchandise Vendor Lookup Procedure

Step

Action

1

Select the vendor from the tailored vendor file by pressing Lookup [F3] when prompted to enter the Vendor Number.

2

Enter a partial vendor name, press Enter. For more information, see Vendor Short Name Lookup.

If

Then

Vendor is found

Press Enter to return all information to the Purchase Order.

Vendor is not found

Press the Lookup [F3] key to connect with Headquarters and search the Accounts Payable vendor file. If you see that the vendor is not available, complete a new vendor request. For more information, see New Vendor Request Procedure.

Note: It is critical that you receive written District/Division approval before placing an order with an unapproved vendor. Purchases over $3,000 per Purchase Order must also have written District Manager approval. Provide accurate vendor information on the New Vendor Request, otherwise this could affect the vendor payment process.

3

If you need to order merchandise prior to receiving vendor information from Accounts Payable, get District Manager approval and complete a manual "B" Prefix Purchase Order to order the merchandise. Process the manual Purchase Order through the system after Accounts Payable gets back to you with the necessary vendor information.
