View Category Codes Procedure

A Category Code Listing is available in POS.  This listing is updated nightly with changes made by Merchandise Control.  You may view Category Codes two ways in the system:

Step Action
Store Management
1 From the POS desktop, select Main Menu -> Manager Functions -> File Listings -> Category File -> Category Code Listing. A report named Category Master File List will be prepared for the Reports File,
2
  • View the report by selecting the Report [Shift + F4] function. 
    		•
    Employees
    1 From the POS desktop, select INQUIRY [F4] -> Products and Prices.
    2 Use Lookup [F3] at the Sales Number Field and select the Product Lookup Option for Dept, Sub, Cat, and Descr to display the Department List.
    3
  • Select the Department you wish to view and, then, select a Subdepartment to display the available Category Codes for the selected Department and Subdepartment.
    		•
    4
  • Access the report from the Report function.
