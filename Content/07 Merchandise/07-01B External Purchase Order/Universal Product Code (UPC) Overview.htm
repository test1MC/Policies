Universal Product Code (UPC) Overview

The Universal Product Code (UPC) is a 12-digit all numeric code that identifies the company/product combination.

  • The code uses a six-digit number to uniquely identify each company coupled with a five-digit number to identify each of the company's products.
  • The combination of these eleven digits plus a check character from the 12-digit UPC number uniquely identifies one and only one item.
  • If you choose to manually enter the UPC number, be careful that you include all 12-digits.
