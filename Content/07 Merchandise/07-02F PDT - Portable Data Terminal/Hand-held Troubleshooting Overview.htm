Hand-held Troubleshooting Procedure

Issue

Description

Order is In Receiving or In Update

The hand-held scanner may still be logged in even if the power is turned off.

Make sure to press CANCEL [F5] to return to the Go to Logon prompt. Otherwise the order could be in In Receiving or In Update status.

When at Go to Logon prompt -> Store Main Menu -> Help Desk -> Clear MMS/COE Status -> Enter Employee Number and Password -> Reset any order in In Update status. Once order is In Process status, select Update/Review Order for customer orders or select Receive/Verify for purchase orders.

Boot Process Fails

Terminal is out of the access point's range. If you move the terminal during the boot process, terminal may be out of the access point's range and unable to complete the process. Move back in range and repeat the boot process.

Contact the Store Computer Operations Help Desk at (216)566-2740 if you see the any of the following:

  • Boot server doesn't exist
  • Boot server not configured for this terminal

File Transfer Process Failed

Terminal is out of the access point's range. If you move the terminal during the download process, terminal may be out of the access point's range and unable to complete the process. Move back in range and repeat the download process.

Contact the Store Computer Operations Help Desk at (216)566-2740 if you see the any of the following:

  • File transfer host is not set up with the boot file information
  • Host is not available
  • Segments missing from file transfer directory

Low Battery Message

Perform any of the following options:

  • Place the terminal in a cradle and recharge battery
  • Connect terminal to charger and recharge battery
  • Power terminal off and replace battery

Battery is dead

Battery not replaced after receiving low battery message or terminal left on for more than 24 hours. Replace the battery.

Note: Terminal loses software when power is lost. Call the Store Computer Operations Help Desk at (216)566-2740. For instructions on how to change the battery, see Change Hand-Held Battery Procedure.

Terminal disassociated from Access Point

Application does not respond to interactive operations.

Applications using internal batch mode continue to function until required to transmit via radio, then fail to work.

Note: No message displays. Call the Store Computer Operations Help Desk at (216)566-2740.
