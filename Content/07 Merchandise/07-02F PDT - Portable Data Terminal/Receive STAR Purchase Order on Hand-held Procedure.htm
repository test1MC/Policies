Receive STAR Purchase Order on Hand-held Procedure

Step

Action

1

From the POS Desktop, go to the Main Menu -> Merchandise Management -> Receive -> Select Internal/STAR from the Receive Order Type Menu.

1. INTERNAL / S.T.A.R.

2. I.S.R

3. EXTERNAL

2

Select ASN from the Receive By Menu. Employees can now scan or manually enter the pallet label number for each pallet to receive all of the merchandise on each pallet. If there is a valid ASN, there will be a list of scanned ASN numbers at the bottom of your screen after scanning the pallet label.

3

Employees also have the choice of receiving part of the order by ASN and part of the order manually by scanning each product. After finishing the ASN portion, press ACCEPT [F1] to return to the menu.

4

Select Purchase Order from the Receive By Menu to receive by individual product number and quantity.

1. A.S.N.

2. Purchase Order

5

Specific information displays at the bottom of the screen that identifies the Purchase Order type and status. Select the correct Purchase Order to receive against.

To receive against multiple Purchase Orders, answer yes to the question: Select another P.O.?

7

Add to the previously received PRV REC amount by entering a new ADD QTY. After finishing the Purchase Order portion, press CANCEL [F5] to return to the MMS Main Menu.
