Hand-held Scanner & Cycle Count Overview

The Portable Data Terminal (PDT) is a hand-held data collection device with an integrated radio used to communicate with the store computer system. The MMS Main Menu on the hand-held scanner provides selections for each hand-held function.

A Cycle Count is a periodic counting of a small selection of products to verify actual on hand to perpetual quantity. The Cycle Count option from the MMS Main Menu enables you to verify current stock levels for all products. For more information on performing cycle counts, please see 27-08 Inventory Control.
