Kit Mistint Procedure

Currently, a kit that is processed as a mistint associates all components in the kit as part of the mistint. The automatic inventory adjustment for mistints will apply to all components in the kit instead of only the mistinted component. This, in turn, requires a manual inventory adjustment to correct the automatic adjustments. In the near future, a kit mistint will only adjust the inventory quantity of the tinted component. In the interim, the instructions below provide a solution to maintain accurate inventory values for a mistinted kit.

DO NOT:

  • Process a mistinted component with SHER-COLOR – 3. Tag as MisTint
  • Process a mistinted component with ORDER ENTRY – 8. Order Disposition
  • Process a mistinted component with ORDER ENTRY – By deleting the kit order line and entering a mistint quantity

When a component is mistinted, a second order must be entered to correct the mistinted component.  The first order will serve as the customer's order, while the second order is reserved for the mistint. The initial order will be sold and the second order will be expensed in POS.
