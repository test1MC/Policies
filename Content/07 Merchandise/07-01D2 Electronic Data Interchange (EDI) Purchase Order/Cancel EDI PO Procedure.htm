Cancel an EDI PO Procedure

Step

Action

Cancel an EDI Order

1

Call the vendor to verify that the merchandise has not shipped. Do not cancel the purchase order before calling the vendor.

If

Then

Cancel an order prior to confirmation with the vendor

  • Issue a manual "B" prefix purchase order
  • Receive against the Manual [B] Prefix PO and contact Accounts Payable to advise them that the Manual [B] PO replaces the canceled purchase order.  Then, call the vendor to tell them to bill you using the Manual [B] PO number.

Note: Call the vendor to tell them to bill the store using the manual [B] purchase order number. After receiving the merchandise, arrange a return with the vendor, if necessary.

The vendor confirms the merchandise has not been shipped

Proceed to Step 2.

2

Main Menu -> Merchandise Management -> Purchase Order -> Cancel Unnecessary Purchase Order.

3

Select the correct order and answer Yes to the cancel order prompt.

Note: Pending Cancel means that the purchase order is waiting for the system to update the purchase order file to reflect a canceled purchase order status.

Canceled PO Received Merchandise

1

Enter a Comment on the purchase order noting the original purchase order number, date and a note stating that the original purchase order was canceled but the merchandise was received.

2

Call the vendor and explain that they will need to use the new "B” prefix purchase order to bill you for the merchandise that you just received. The manual B Prefix purchase order will only be a confirmation order, but make sure that the vendor does not ship the merchandise again.

3

Receive the merchandise against the B Prefix purchase order.

4

Process a Chargeback for the merchandise returned to the vendor.

Note: For more information, see Cancel a Purchase Order Policy.
