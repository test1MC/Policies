EDI B Prefix Purchase Order Procedure

Step

Action

1

From the POS Desktop, go to Main Menu -> Merchandise Management -> Purchase Order -> Add New Purchase Order.

2

Select External Order from the Order Type List.

Note: The cursor stops at the [P.O.#:] field. The prompt in the lower left corner says: Enter PURCHASE ORDER NUMBER. When the employee starts typing the complete B Prefix Purchase Order number, the system generated number is deleted from the screen.

3

Enter all of the information into the system using the manual B prefix purchase order as a guide.

Note: Enter the special cost for the merchandise.

4

When the B prefix Purchase Order is released on the store computer system, it becomes the EDI Confirmation of the order given to the Vendor Sales Representative at the store. Immediately receive against the purchase order because the merchandise from the Vendor Sales Representative was received.

The following reminder message will appear if there is a difference in freight:

VENDOR'S EDI AKNOWLEDGEMENT FREIGHT VALUE
FREIGHT IS $20.00
USE THIS AMOUNT FOR FREIGHT?  Y or  N

If

Then

Yes

The purchase order freight will be adjusted to match vendor freight charge.

No

Any freight difference will reflect in the Freight Variance Report.
