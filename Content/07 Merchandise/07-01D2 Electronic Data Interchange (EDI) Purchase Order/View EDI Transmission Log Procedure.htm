View EDI Transmission Log Procedure

Through the EDI Status Logging Update, any employee at a Sherwin-Williams store will now be able to check the status of all EDI purchase orders that have been released from the store.  This will enable the employee to see if EDI purchase orders were sent successfully in the transmission log. This enhancement logs the success and failure of EDI Purchase Order transmissions. A new menu option in the Print/View Order section was added to allow users to view the current EDI transmission queue (i.e. those orders still waiting to be sent to Cleveland) as well as view entries in the EDI PO Transmission log (by date range or by control number).

Step Action

1

 From the POS Desktop, go to Main Menu - > Merchandise Management - > Purchase Order - > View EDI Transmission Log.

2

Select one of the following options:

Part Description

Current EDI Queue

Shows pending purchase orders that have not yet been successfully sent via EDI.

Control Number

Gives the ability to search by the control number of the EDI purchase order.

Time Range

Gives the ability to search by specific dates and times for all EDI purchase orders.

Most Recent

Shows all EDI purchase order transmission attempts. The most recent attempt will be at the top of the list.

3

Determine what option to view:

If Then

Control EDI Queue
  • Select the appropriate purchase order and press Enter.
  • Select Cancel [F5] to view another pending purchase order and repeat the process or to go back to the purchase order screen.

Note: The EDI Transmission Log will display the list of attempts of the purchase order being sent.  EDI attempts every fifteen minutes to send a purchase order.

Control Number
  • View the attempts made by the EDI system.
  • Select Cancel [F5] when finished.

Time Range
  • Press Tab to select Yes [Y] or No [N] to Show Errors Only.
  • Enter the required date and time range.
  • View the range of purchase orders for that time period.
  • Select Enter to view the EDI Transmission Log.
  • Select Cancel when finished.

Most Recent
  • Press Enter to show all recent EDI orders.
  • Press Enter to select the desired purchase order to view the EDI Transmission Log.
  • Select Cancel when finished.
