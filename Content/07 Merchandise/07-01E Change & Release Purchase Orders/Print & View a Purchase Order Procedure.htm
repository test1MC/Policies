Print & View a Purchase Order Procedure

Any purchase order which is transmitted through EDI will have the option to print a hard copy of the order prior to its release.  Any purchase order that is not transmitted through EDI will automatically print.

Step

Action

1

From the POS Desktop, go to Main Menu -> Merchandise Management -> Purchase Orders -> Print/View Purchase Order.

Note: Receiving Status, which contains only those Purchase Orders that have been released, partially received or Purchase Orders that are In-Process or In-Recvng, can also be viewed by selecting  Main Menu -> Merchandise Management-> Purchase Orders -> Print/View Receiving Status.

2

Enter the search parameters  or the Control Nbr in the PO Lookup Menu.

Note: The Purchase Order Control Number is a sequential number assigned to every purchase order, both system generated and those downloaded from Headquarters. Press Cancel [F5] to send a journal to the Reports file.

3

Press Enter or Accept [F1]  to select the Purchase Order and view the Options Menu.

Option Type

Description

Display line for this PO

Review information for each line of the PO

View on this Screen

View the complete PO before printing.

Print on Mgr’s Printer

An additional copy of the PO can be printed.

View Special Instructions

Review any comments that may have been added to the PO

View Another PO

Return to the previous screen to select another PO.

Note: All employees are able to print/view a Purchase Order, which will print on the Manager’s printer.
