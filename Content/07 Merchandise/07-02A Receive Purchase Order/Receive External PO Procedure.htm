Receive External PO Procedure

Step Action

1

When merchandise arrives at your store, you can receive by either selecting Receive/Verify from the Merchandise Management Menu on the Store Computer System or using a Hand-Held Scanner.

2

 

 

 

 

Regardless of method, choose to receive by the following options:

Option Action

Receive by Product

Scan the Universal Product Code or enter the Sales Number.

Receive by Vendor

Select Vendor from the Receiving Type List -> Press LOOKUP [F3] -> Enter Vendor Name and press Enter

Receive by Order

Select Order from the Receiving Type List

3

Select the proper Purchase Order when the following options return to the screen:

  • Released
  • In-Process
  • In-Recvng
  • In-Part RCPT

4

If you wish to receive against only Direct Shipment Purchase Orders, select the Direct Shipment selection.

5

Have an employee with at least Level 2 security clearance approve and verify the order.

6

The Store Manager closes all external Purchase Orders.
To close a Purchase Order on the Store Computer System

7

Select Receive/Verify from the Merchandise Management Menu -> Press Enter when the highlight bar is on Order -> Select a Purchase Order with a Received status.
