MMS Report Types Overview

There are an array of reports available to Store Management that provide more information about inventory. To review these reports from the POS Desktop, select Main Menu -> Merchandise Management -> Reports.

Type Description

Receipts Report

Enter the date range and the sales number to review receipt history for a specific sales number.

Note: Press ACCEPT [F1] when prompted to enter Sales/Product Number to view all products for the date range.

Memo Demand Report

Enter the date range to review the Memo Demands for the selected period.

  • Review the Memo Demands for products before placing a Star Order.
  • The Memo Demand Quantities will automatically be considered for Suggested Star Orders.

Inventory Adjustment Report

The Inventory Adjustment Report allows quick review of Inventory Adjustment history for each product.

  • Store employees can review the beginning on hand quantity and value and the adjusted quantity and value. The dollar impact and employee number also displays on the screen.
  • See 07-02d MMS Menu options for more information.

Manufacturing Adjustment Report

 

 

This only applies to Chemical Coatings Facilities and Automotive Branches

The Manufacturing Adjustment Report shows all inventory movement specific to the manufacturing of blends.  The Inventory Adjustment Reportshows Perpetual Adjustments that were entered by a user to adjust for cycle counts.

Stage Description

1

A store enters a blend on a customer order.

2

Phoenix queries the open orders on POS for blend requests and determines a formula for the blend.

3

Once a blend is made and completed, phoenix returns the raw materials list and quantities to the POS and the inventory is adjusted for the new product.

4

The new blend is added to inventory and the raw materials used in the blend are deducted from inventory.

5

Each of the transactions in and out of inventory are tagged on the inventory details with an X transaction type.

Cycle Count Report

The Cycle Count Report provides a list of items counted during periodic cycle counts. You will see the perpetual quantity, actual quantity, unit variance, and dollar variance.

This report should be used by Store Management to review cycle count inventory results and analyze quantity variances.

If Then

A Cycle Count was not performed on the date entered

The system will look first to see if there is a more current Cycle Count than the date entered

There is not a more current Cycle Count than the date entered

The system will return the most recent previous Cycle Count prior to the date entered.

If an employee needs to look at a Cycle Count performed earlier than the Cycle Count returned on the screen

Enter an earlier date

Note: In the future, employees will be able to enter a date range and review reports containing all Cycle Count Information.

Negative Inventory Report

The Negative Inventory Report lists products with negative perpetual inventory balances.

Note: The items on this report should be included in your cycle count each week.

Special Order WC/WT Report

 

 

This does not apply to Automotive Branches or General Polymer

The Special Order WC Report shows the unsold received book order wallcovering in your store.

Note: This report should be reviewed on a weekly basis. Follow-ups should be documented on the individual wallcovering orders.

Unit Movement Report

The Unit Movement Report identifies all activities for a specific sales numbered product.

Min Max Report
  • Generate a min-max report based on the Dept-Sub-Cat, product number, or tab number and specify a last changed date range to include only the products that were changed between the date ranges.
  • Sorting options help you sort the lines on the report by Dept-Sub-Cat, Product number, Tab number, or Change Date.

Product List
  • The Product List will assist you in the counting and recording process. 
  • Once printed, the report provides a worksheet used to note product locations, quantities, and obsolete status if the product has been recorded on the store computer system.

Shipment Variance Report

This report shows any variances on any Star Orders that have been received.

Order Advisory
  • The Order Advisory Report identifies products that will not be received from the DSC because of  product request errors during transit.
  • An Order Advisory Report is sent to stores every two to three hours throughout the day. Information in each new report will replace information in the prior report.

Bin Location Reports

This report shows what products are in specific bin locations.

Spray Equipment Log
  • The Spray Equipment Log displays all serialized spray equipment at your location. 
  • It also shows the purchase order and date the equipment was received on, serial number, landed cost, and if it is a rental or stock piece of equipment.

Inventory Usage By All Parts

This shows the inventory history of any SKU. 

This report can be narrowed down by a specific sales number, department number, or sub-department number.
