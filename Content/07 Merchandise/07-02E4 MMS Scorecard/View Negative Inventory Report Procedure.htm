View Negative Inventory Report Procedure

Check that Negative Inventory is Cycle Counted Weekly, follow these steps:

Step

Action

1

From the POS Desktop -> Main Menu -> Merchandise Management Menu -> Reports -> Negative Inventory Report -> View on this screen.

Note: This report will show the current products that have a negative inventory value. All items appearing on this report should be included on a weekly Cycle Count. If the Negative Inventory Report is of significant length (greater than two pages) check with the Store Manager to ensure that all merchandise delivered to the store has been received against the appropriate Purchase Orders.
