Review Cycle Counts & Inventory Variances Procedure

Step

Action
Ensure a Cycle Counts was performed within the last week

1

From the POS Desktop -> Main Menu -> Merchandise Management-> Reports -> Cycle Count Report -> Enter the Start and End Dates -> Press ACCEPT [F1] .

2

From the Options Menu, select View to quickly review all Cycle Counts performed during the date range.

3

Determine that all variances between the perpetual inventory and the actual quantity have been corrected with the appropriate transaction.

Check that all variances between perpetual inventory and actual quantity have been corrected

5

From the POS Desktop, press INQUIRY [F4] -> Select Unit Movement -> Enter the Sales Number containing a variance -> Enter Date of the Cycle Count as the From Date -> Press Enter twice.

6

Review the report to see if a transaction was performed to correctly by viewing the Cycle Count variance.

7

If no transaction can be identified, perform a Cycle Count. Press CANCEL [F5] to exit the Inquiry screen -> Select Inventory Control from the Merchandise Management Menu -> Cycle Count from the Inventory Control Menu.

8

Enter the Selected Sales Number -> Count the total amount of the product -> At the Actual Field, enter the counted quantity.

9

If a variance still exists, contact the Store Manager.
