Enter Intermix as Price Exception Procedure

All Intermixes will be entered as Price Exceptions by Intermix Sales Number or by Department Sub Category Code by Size and Quantity.

Step

Action

1

From the POS Desktop, go to Main Menu -> Manager Functions -> Price Record Cards -> Add New Price Record Card.

2

Complete the Price Record Card Header.  Enter the following:

  • Customer account number
  • Price Record Card number
  • Job number
  • Review/Expiration date
  • Select Y/N: Pro Buy, DIY, Promo, District Exceptions, Store Specials
  • Select Y/N: Packing Slip, Additional Copy, Laser, PO Required
  • Tax Exempt number
  • Ship-To Address (if different than Bill-To)
  • Phone Number
  • Main Contact

3

Enter Price Schedule levels. Press Accept [F1].

4

Select Yes to Do you want to enter Price Exceptions?. 

Note: All Intermix pricing will be entered as exceptions.

5

Create Intermix Price Exceptions by Intermix Number:

Step

Action

1

Tab over to Sales/Product# field.

2

Press Lookup [F3] and select Intermix Blend.

3

Enter Intermix number, color name, and customer part number.

Note: If the Intermix exists in the database, proceed to the next step. If not, a message will appear, select Y when prompted, and press Enter.

4

Select Done.

5

Enter the Min. Qty, if applicable. Press Enter.

6

Enter the selling price.

6

Create Intermix Price Exceptions by Dept / Sub / Cat Code:

Step

Action

1

Press Lookup [F3] and select Dept 00 for Automotive/Sub/Category.

2

Press Accept [F1] and the chosen Dept Sub Cat code will appear.

3

Select size code and enter a min. qty., if applicable.

4

Press Enter again to bypass including a level.

5

Enter a selling price.

7

Include comments and authorize buyer information, when prompted.
