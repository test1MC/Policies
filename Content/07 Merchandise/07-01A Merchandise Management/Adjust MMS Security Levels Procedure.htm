Adjust MMS Security Levels Procedure

Step

Action

1

From the POS Desktop, select Manager Functions. Enter the Manager Password.

2

Select File Maintenance from the Manager Functions Menu.

3

Select Employees from the File Maintenance Menu.

4

Select Change from the Employee Maintenance Menu.

5

Enter the employee number.

6

Press ENTER until prompted with Enter Level of MMS Security (0-4).

7

Adjust the security level.

8

Press ACCEPT [F1] when complete.
