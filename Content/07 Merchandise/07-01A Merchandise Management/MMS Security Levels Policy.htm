MMS Security Levels Policy

Merchandise Management is divided into four levels based on your position or the level of responsibility you have at your location. Use the following Security Levels policy as a guide for MMS security levels.

  1. New employees are automatically set at Level 1 MMS Security. There is no additional set-up required for this security level.
  2. The Manager is able to give Level 2 MMS Security to select individuals that will be able to perform additional functions.
  3. The Assistant Manager and/or Operations Manager can be given Level 3 MMS Security that will enable them to perform additional Merchandise Management System functions.  
  4. MTP employees should be set at Level 3 MMS Security to ensure required training scope and store experience.
  5. The Store Manager is permitted to assign an employee a security level having less responsibility.  The Store Manager, for example, might not want a new Assistant Manager maintaining the Min/Max levels until the Assistant totally understands Merchandise Management. Also, if an employee does not understand the receiving process, the Manager may choose to leave the employee at LEVEL ONE security.
  6. The Store Manager may choose for an employee to prevent access to any portion of MMS by setting the security to Level 0 MMS Security.
  7. Written District Manager approval must be on file if the Manager delegates Level 3 MMS Security to someone other than the Assistant Manager and/or Operations Manager.
  8. The Manager is not permitted to delegate Level 4 MMS Security to someone else in store management without written District Manager approval.
