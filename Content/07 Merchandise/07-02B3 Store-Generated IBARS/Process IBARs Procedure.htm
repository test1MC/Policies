Process IBARs Procedure

Step

Action

1

From the POS Desktop, go to Main Menu -> Merchandise Management -> IBAR -> Add New IBAR -> Returns -> Approved Store Operations.

2

Select the IBAR.

3

Add approved products and quantities.

If

Then

Entering the SMIS to go directly to the line

Change the quantity to the amount to be returned. Press Accept [F1] to complete.

Using Lookup [F3] to view all the products from the Approved Return IBAR

Select the products from the list and change the quantity amount to be returned. Press Accept [F1] to complete.

Note: To save the IBAR, press Accept [F1] but do not release the order.

4

Release the IBAR.

Note: Please refer to the Return Goods Pick-Up Procedure to determine the carrier.

5

Place the store copy of the IBAR in the Awaiting SPR folder and retain the driver-signed copy of the Bill-of-Lading

Note: Products must be returned in original packaging and standard cartons. Empty cartons can be ordered from the DSC. Also, products must be returned on SW pallets (30”x42”) with the total height under 48” from the ground (including pallet height). All pallets must be properly stacked and stretch wrapped for shipment. The DSC Return Label form can be found in the Forms Repository.
