Return Goods Pick-Up Procedure

Step

Action

1

Determine which carrier will pick up the return goods:

If

Then

Con-way picks up the return goods

Con-way will contact the store within 48 hours of the IBAR submission and will provide pallet labels and Bill-of-Lading.

Note: Pallet labels must be attached to each pallet or they will not be picked up.

CTS picks up the return goods

CTS will contact the store via email regarding the date and time for shipment pick-up. Proceed to Step 2.

A carrier other than Con-way or CTS picks up the return goods

Proceed to Step 2.

2

Create a Bill-of-Lading using BOLDOC.

Note: Questions regarding the freight carrier should be sent to Stores Customer Service.
