IBAR Corrections Policy

  1. Do not wait for Store Billing to issue a correction.
  2. Correct an IBAR as soon as an error is detected.
  3. The Store Manager must approve the voided IBAR prior to mailing.
  4. The Store Manager must approve the voided Computer Generated Form and file according to batching instructions.

Note: Call 216-566-2328 if you have any questions on IBAR Corrections.
