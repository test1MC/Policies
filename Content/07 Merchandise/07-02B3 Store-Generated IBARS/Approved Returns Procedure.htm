Approved Returns Procedure

After completing the Approved Request Return Form found in the Forms Repository, send it to Stores Operations. They will review the request and respond back to the store within seven to ten business days identifying the proper course of action:

If

Then

Stores Operations suggests a rebalance

Follow these steps to Process a Rebalance:

Step

Action

1

The receiving store must create an ISR Purchase Order for the material being transferred.

Note: For more information on creating an ISR, see Place ISR Purchase Orders Procedure.

2

The transferring store must create an IST for the material and arrange for shipment to the receiving store.

Note: For more information on creating an IST, see Create Inter Store Request (ISR) Procedure.

Stores Operations suggests a return to the DSC

Follow these steps to Process a Return to the DSC:

Step

Action

1

From the POS Desktop, go to Main Menu -> Merchandise Management -> IBAR ->  Add New IBAR -> Returns -> Approved Returns

2

Enter the appropriate parameters.

Note: The default carrier is Con-way.

3

Add approved products and quantities or use Lookup [F3] to view all approved products.

4

Select the product from the list and change the quantity to the amount being returned.

5

Release the IBAR.

Note: Please refer to the Return Goods Pick-Up Procedure to determine the carrier.

6

Place the store copy of the IBAR in the Awaiting SPR folder and retain the driver-signed copy of the Bill-of-Lading.

Stores Operations denies the request

List the product on the Overstock Inventory Reduction Program site.

SOURCE homepage -> Division -> Division Marketing.

Note: If the product cannot be moved on through the Overstock Inventory Reduction Program, take appropriate action to liquidate or dispose of the product.
