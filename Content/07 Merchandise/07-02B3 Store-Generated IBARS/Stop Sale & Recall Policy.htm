Stop Sale & Recall Policy

  1. A Return IBAR must be processed within sixty days of the Product Recall / Stop Sale Bulletin issue date in order to receive credit.
  2. TAG / PFG locations have sixty days to process this IBAR.
  3. Automotive branches must process no later than the return date noted in the return bulletin.
  4. Recalled products should be processed using a Reason Code as indicated on the Product Recall / Stop Sale Bulletin. The Stop Sale will involve a return to the plant or DSC, vendor, or no return.
  5. Defective products not involved in a recall must be returned using the Quality Return – PQR Reason Code.
  6. Defective products not involved in a recall must be recorded on a Product Quality Report (PQR).
  7. Once all Product Recall / Stop Sale criteria is met, the store will receive credit within ten days of receipt at the factory.
  8. If the Reason Code selected indicates a return of goods to the DSC or plant, the product must be returned via the carrier selected during the IBAR submission process.  
  9. If the Reason Code selected indicates a return of goods, product must be packaged according to specific guidelines outlined in the Product Recall / Stop Sale Bulletin.
  10. If the Reason Code selected indicates a return of goods, return all recalled product within the required timeline after the issue date of the bulletin. Product remaining in the store after the deadline cannot be returned and should be store expensed.
  11. All recalled defective merchandise, regardless of quantity, must be returned per the Product Recall Bulletininstructions. Defective merchandise must be isolated and stored away from the sales floor. The Store Manager or Branch Manager is responsible for ensuring that all defective merchandise is returned.
  12. Do not sell, use, or give away defective merchandise.
  13. Do not destroy recalled merchandise locally.
  14. Recalled merchandise not returned must be inventoried as non-current at year-end.
  15. Unsafe defective products must be reported to the Wholesale Marketing Department at Headquarters.
  16. Each carton or container of defective merchandise must be labeled.
  17. TAG and PFG locations:  Use recall sticker (Available via Print on Demand – 2000531PD).
  18. Automotive Branches:  Label each carton, container, or pallet of defective merchandise with the DSC Return Label noting the Recall / Stop Sale Code. The DSC Return Label can be found in the Forms Repository.
  19. Any product not listed on the Product Recall / Stop Sale Bulletin should be returned.
