Automotive Approved Return Overview

This section is for Automotive Branch only.

The Automotive Approved Return allows Automotive Branches to reduce inventory levels based on current sales demand and return products no longer needed to be stocked or sold.  Automotive Branches will receive information regarding scheduled Approved Returns from Automotive HQ. Branch min/max levels must be manually updated prior to the scheduled Approved Return. Authorized Approved Returns will be downloaded to the Branch on an IBAR and should be processed as directed by policy and procedure.
