Emergency Returns Procedure

Step

Action

1

Determine the correct action based on the product type.

If

Then

The product is Architectural, IM, or CC

Submit a request for return using the Approved Return Process.

The product is Automotive, Marine, or GP Product

Contact Customer Service and proceed to Step 2 if instructed to create an Emergency Return.

2

From the POS Desktop, go to Main Menu -> Merchandise Management -> IBAR ->  Add New IBAR -> Returns

3

Select the Emergency Reason Code.

4

Enter the appropriate parameters to complete the IBAR.

5

Add products or use Lookup [F3] to add products from original purchase order.

Note: Quantity, invoice, and a return reason must be noted for each product.

6

Release the IBAR.

Note: Please refer to the Return Goods Pick-Up Procedure to determine the carrier.

7

Place the store copy of the IBAR in the Awaiting SPR folder and retain the driver-signed copy of the Bill-of-Lading.
