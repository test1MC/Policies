Approved Returns Overview

This section is for TAG Stores and PF Facilities only.

The Structured Return Type has been discontinued and replaced by the more flexible Approved Return Type which allows stores to request the return of product at any time. Additionally, it rebalances product within districts to save on the freight costs associated with shipping.  Upon receiving a request for an Approved Return, Stores Operations will first see if the product can be rebalanced by moving it to another store within the district or a neighboring district.  If the product cannot be rebalanced, Stores Operations will assess the product for return to the DSC by reviewing product type (paint only), shelf life remaining, product status, package requirements, and freight cost effectiveness for the company.

Additionally, the Approved Return process is used to initiate Emergency Returns for Architectural, IM, and CC product.
