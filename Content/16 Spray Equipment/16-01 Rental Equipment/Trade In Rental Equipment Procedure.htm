Trade In Rental Equipment Procedure

Step

Action

1

Process a Correction Transaction using class number 7708GP=00. This will put the unit into inventory at the trade-in amount.

2

Continue processing the transaction on the positive side charging the customer for the new unit.

3

Process an Inventory Adjustment to register the serial number using sales number 400-1624.

4

When you sell the Trade-In unit, process the sale using class number 7708GP with the appropriate gross profit.

5

Process an Inventory Adjustment to remove the correct serial numbered item from your inventory.

Note: If necessary, inventory the Trade-In unit the same as other rental unit (repairs completed on Trade-ins must be added to the trade-in price to determine store cost).
