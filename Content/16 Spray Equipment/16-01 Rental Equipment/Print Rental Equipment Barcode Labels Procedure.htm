Print Rental Equipment Barcode Labels Procedure

All units in rental can re-print a barcode label.

Step

Action

1

From POS desktop, select Store -> Merchandise Management -> Inventory Adjustment -> Serialized Equipment Rentals/Sales Menu -> Select Print Barcode Labels.

2

Enter the Sales Number or Press LOOKUP [F3].

4

Do you want to print a barcode label?

If Then
Yes Select Y to print the label.
No Select N to cancel.
