Repossessed Spray Equipment Procedure

This section does not apply to Automotive Branches

Step

Action

1

Process a Charge Return to the customer's account for the balance that remains using the class number 770800.

2

Process an Inventory Adjustment to register the serial number using sales number 400-1624.

3

When you sell the repossessed spray unit, process the sale using class number 7708 with the correct gross profit.

4

Process an Inventory Adjustment to remove the correct serial numbered item from your inventory. If necessary, inventory the repossessed unit the same as other rental unit.
