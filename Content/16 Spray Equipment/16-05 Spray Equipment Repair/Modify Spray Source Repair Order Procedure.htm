Modify Spray Source Repair Order Procedure

Step Action

1

From POS desktop, select Store -> Customer Order Entry -> Add Order -> Select Spray Equipment Repair from Customer Service Options Menu -> Use DIY as the account number.

2

Enter Store Rental Repair as the Shipto Address -> Enter rental details on the Repair Information Menu.

3

At the order detail screen, move to the order line which automatically populates on the repair order. Press DELETE [F8] to remove this line from the order.

4

Process a Store Expense Transaction to expense the cost of parts. For more information on how to process a Store Expense Transaction, please see Store Expense Procedure. 

5

Connect the customer order to the store expense using order disposition. For more information on order disposition, please see 07-03A Customer Order Entry.
