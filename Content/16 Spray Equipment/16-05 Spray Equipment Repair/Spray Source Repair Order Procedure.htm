Spray Source Repair Order Procedure

Stores and Spray Source Repair Centers should follow these procedures to initiate a customer repair through the POS System. Manual forms should be avoided except for situations where the computer system is not available and the order must be added to the system as soon as the system is available again. Repair orders are “works in progress.” A repair order will be created and possibly transferred to another location.  During the actual repair process, the order should be updated continuously to reflect the corrective action(s) performed, time spent working on the equipment, and the parts used in the repair.

Step

Action

1

From POS desktop, select Store -> Select Customer Order Entry from the Main Menu.

2

Select Add Order from the Customer Order Entry Menu.

3

Select Repair from the Customer Service Options Menu.

Note: A Customer Account Number and a ShipTo Address must be captured in the Order Header. If the repaired equipment is to be delivered to the customer once the repair is complete, make sure that the address is formatted correctly.

4

Enter the date in the Date Received Field.

5

Enter the name of the customer in the Received From Field.

6

Enter the promised date.

 

If

Then

Handling the repair at your location?

A Repair Employee number is required.

Transferring the repair order to a Spray Source Repair Center?

Enter the store number of the repair center in the Repair Store Field.

Note: The repair store must be set up in Transfer Store Maintenance to Spray Source Center Procedure.

7

Will your customer pick up the repair?

If Then
Yes Select Customer Pickup.
No

Select Delivery.

Note: If delivery is selected, a Bill-of-Lading will be created.

8

Enter the problem description.

Note: Please include information about the current condition of the unit including cleanliness, age, if the unit currently contains any gas, oil, or any other hazardous fluids, etc.

9

The following information is not required when creating a Repair Order, but it is helpful in documenting the unit and its accessories as they come into our care and in helping to establish and verify warranty claims:

  • Received With (accessories and attachments)
  • Serial Number
  • Make
  • Model
  • Model Number
  • Series Number
  • Unit Description
  • Purchase Date

10

Repair orders default to contain one line item in the order detail for the $8 shop charge assessed to every repair order. No other line items should be added while creating the order. Lines should be added later for labor, repair, and other associated charges.

11

The system will generate two copies of a repair contract, which the customer must sign. Give the customer one copy and retain the signed copy at the store for 5 years plus the current year.

12

Attach the paperwork created by the computer while creating the customer repair order to the equipment as the proper repair identification information for the unit while in Sherwin-Williams’ possession.

13

Communicate with repair center:

Who will do the repair?

If

Then

Spray Source Location

Initiate the repair order. Please proceed to Update Spray Source Repair Details Procedure.

Transfer to Spray Source location

Transfer to a Spray Source Repair Center.  For more information on transferring orders to a repair center, please see Transfer Spray Source Repair Order Procedure.

External Vendor

Place an external purchase order. For more information on placing an external purchase order, please see External Purchase Order Procedure.

Note: BOLDOC cannot be used for creating bills-of-lading for spray equipment; a Straight Bill-of-Lading Form must be manually completed. This form is available in the Forms Repository. Search form name or keyword “lading” in the Form Name Field. All available forms containing your search word will appear below in the Form Description Field.

If your district is participating in the SW Store Delivery Program, schedule the delivery of the equipment as a miscellaneous stop.  For more information on this process, please see Store Deliveries Overview.
