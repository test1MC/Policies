Process Spray Source Transfer Repair Procedure

This section does not apply to Automotive or Chemical Coatings.

The instructions in this section are to be used by Spray Source Repair Centers and other repair locations processing customer repair orders, which have been transferred to their location from another store. 

Step

Action

1

You will see the transferred spray repair order before the rig arrives at your shop. Be sure to note with what equipment the spray rig will arrive. When the rig is delivered, verify that all equipment is present before allowing the driver to leave. A copy of the customer repair order created at the store should be attached to the unit for easy identification.

2

You are able to monitor the transferred orders, which are coming into your system through the Status Reminder Box. You will see that the Transferred In number will increase in the Status Reminder Box as you receive orders. When the order is sold or transferred back to the originating store, the Transferred In number will decrease and the Transferred Out number will increase.

3

From POS desktop, select Store -> Select Customer Order Entry.

4

Select Update/Review Orders from the Order Entry Menu.

5

In the Order Lookup Window, query using the following parameters:

  • Order Status N for NEW
  • Transfer Status I to for TRANSFERRED IN orders.

6

Press ENTER to select the order you wish to review.

7

The Blue Information Box in the lower right-hand corner will identify the location from where the order originated.

8

After reviewing the order, select the Print/Review option to print one copy of the Picking Ticket and to place the order in In Process Status. This option will change the order status so that you may better manage your Transfer Orders.  It is suggested that you change this status once the unit arrives so that it is easy to identify orders that have arrived in the store and orders you are still awaiting. Once this option has been chosen for an order, it will disappear from the Print Options Menu.
