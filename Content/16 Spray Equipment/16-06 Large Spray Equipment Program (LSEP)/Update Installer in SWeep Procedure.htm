Update Installer in SWeep Procedure

This section only applies to the LSEP dedicated POS location cost centers, which process equipment sales totals of $100,000 or more.

Step

Action

1

Obtain the signed Subcontractor Agreement, W-9, and Certificate of Insurance from the Installer.

2

Copy these documents to the Large Equipment shared drive (\\cgconhq.sw.sherwin.com\PPFacilities\Large Equipment\Installers).

Note: Create a new folder, if necessary, for the installer with all the required documents.

3

Access SWeep: From POS desktop, select Browser -> Access Source homepage -> Source Operations -> Processes -> SWeep.

4

Select Request New Vendor/Payee.

5

Fill in all the required fields.

6

Attach the required documents.

7

Select Subcontractor for VendType.

Note: This will determine the payment terms.

8

Ensure all installer paper is complete before approval is requested.

9

Receive verification that the installer has been set-up in SWeep before processing payment.

Note: All New Vendor Requests will be approved by Kevin Cross.
