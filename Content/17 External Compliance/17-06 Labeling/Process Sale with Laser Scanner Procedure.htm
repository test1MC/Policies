Process Sale with Laser Scanner Procedure

If a customer purchases several scannable products, follow these steps to process the POS:

Step

Action
Scan Products

1

Scan the first two products. The quantity will default to (1) for the first product. Continue scanning each product, and manually enter the quantity of the last item before totaling the sale.

Note: Sherwin-Williams Gallons and Quarts will show a quantity of 1, but a 5 gallon container of paint will show a quantity of 5 when scanned. Associated Products that have a size code of 99 will return a quantity of 1 even if the actual container size is larger.

For Example: A 3-gallon container of Elroy Paint Thinner shows a size code of 99 (each). In this case, change the quantity from 1 to 3.

2

Scan the same product if the customer is purchasing more than one. This will update the quantity at the Point of Sale to match the number of times to scan the product.

3

Manually enter the quantity at the Point of Sale.
Input Sales Number in Inquiry Menu
1

The Laser Scanner can also be used when prompted for a sales or product number.

For example: when accessing Products and Prices from the Inquiry Menu, use the Laser Scanner to automatically input a Sales Number.
Enter Bar Code Manually
1

If the Laser Scanner is not working properly, enter the Bar Code number manually. The Universal Product Code (UPC) is a 12-digit number (x-xxxxx-xxxxx-x) shown both numerically and as a computer readable bar code.

As shown in the examples below, UPC’s can appear a couple of different ways, so it’s necessary to look closely for all 12 digits.

 

0

3577720199

2

035777201992

 

Note: The scanner uses a low-power laser. As with any bright light source, DO NOT stare into the beam.
