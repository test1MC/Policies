POS Scanner Downtime Procedure

In the event of downtime, follow Downtime Processing Instructions.

Step

Action

1

Enter the product sales number or the UPC code when completing the Manual P.O.S.T Form.

2

Use the current Price Schedule to look up the selling price or, if convenient, check the selling price on the sales floor.

Note: Reference a Manual Price Schedule if the computer system is not working properly.

If the store does not have a current Price Schedule for each product line, select Manager Functions and then Print Price Schedules.
