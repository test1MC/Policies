Canada - Apply Eco Fees Procedure

Eco Fees are automatically visible at the Point-of-Sale or automatically built into the final price.

Step Action

1
If Then

Automatically inputting the Eco Code

After processing the sale, press [Enter] at the sales number prompt to display the Non-Merchandise Code Menu. Codes 1-7 will display. Press the Down Arrow to display the Eco Fee Non-Merchandise Code. Press [Enter] to select the Eco Fee Code.

Manually inputting the Eco Code

See and, in the sales number prompt, enter the code for Eco Fees. Press [Enter].

2

Manually add up the number of containers by their size and multiply by the corresponding rate.

3

Enter the total Eco Fee as one line item with a quantity of 1.

Note: It is permitted to enter the unit quantity and unit Eco Fee to get a system calculation. However, system calculations should not be used if it would be required to enter more than one line for the Eco Fee.

4

Press [Enter].

Note: Contact Corporate Regulatory for questions regarding whether or not an Eco Fee has been, or should be, included.
