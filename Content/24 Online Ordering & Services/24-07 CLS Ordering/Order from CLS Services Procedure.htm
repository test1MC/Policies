Order from CLS Services Procedure

Step Action

1

Generate an External Purchase Order to CLS:

Step Action

1

From POS desktop, select Store -> Merchandise Management -> Purchase Order -> Add New Purchase Order -> External Order.

2

Enter the required information.

Note: Use the Sherwin-Williams Sales Number (161-2720) for each item and use the Part Number of the as item as Expense Descriptor.

3

Retain the PO Number.

2

Complete the CLS Order Form:

Step Action

1

From POS desktop, select Browser -> Access SOURCE homepage -> Information Library -> Forms Repository.

2

Fill out the top portion of the CLS Order Form.

3

Fill in the quantities (left column) for each item needed.

3

Attach the CLS Order Form to an email and send to swsales@2cls.com for processing.

Note: The subject line should read, “Sherwin Order – PO Number”
