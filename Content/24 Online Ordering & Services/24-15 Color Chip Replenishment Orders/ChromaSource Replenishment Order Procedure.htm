ChromaSource Replenishment Order Procedure

You have two options for creating your color chip replenishment order:

  • Use the PDT to scan chips, build the order, and submit to the ChromaSource website.
  • Manually record the chips which need to be replenished and build the order directly in the ChromaSource website.
Step Action
Build Order via PDT

1

Enter your Employee Number and Password in the PDT.

2

Select ChromaSource from the PDT MMS Main Menu.

3

Select Add Order from the Chroma Source Orders Menu.

4

For each color chip which needs to be replenished:

Step Action

1

Scan card.

2

Enter Quantity.

3

Press Enter.

5

To save the order, press Function and Press 5. 

Proceed to MMS from a POS Terminal to submit the order to ChromaSource’s website.
Release Order via MMS

1

From POS dektop, select Store -> Merchandise Management.

2

Select Purchase Order -> Maintain ChromaSource Orders.

3

Do you need to make changes to the order before you send to ChromaSource?

If Then
Yes Proceed to Step 4.
No Proceed to Step 5.

4

To make edits to the order before sending to ChromaSource:

Step Action

1

Select Change Purchase Order.

2

Select the order you wish to edit.

3

Make edits:

  • Quantity can be changed
  • F7 [View] to add a line.

4

Proceed to transmitting the order to ChromaSource.

5

To transmit the order to ChromaSource:

Step Action

1

Select Transmit Purchase Order.

2

Select the order you wish to send.

3

Select Yes.

4

Proceed to ChromaSource website to complete order and submit for processing.

Process Order in ChromaSource

1

From POS desktop, select Browser -> Logon to the SOURCE Intranet.

2

Access Source Operations -> Order -> Color Feature Order Cart.

3

Are you accessing a POS Order , or creating a new order?

If Then
POS Order

Select on the Cart in the upper right-hand corner to access your order.
New Order on ChromaSource Proceed to Step 5.

4

POS will not enforce any of the ordering rules of ChromaSource or validate that the SKUs scanned are color chips.  If any errors or issues exist which must be resolved, you will see these errors at the top of your order (address each error as required).

5

To add, edit, or remove products from the order you may do so using any of these methods:

Option How To
Left Hand Options

Narrow your search by product type.

  • 2x3 Chips
  • HGTV Strips
  • Look Books
  • Post Cards
  • Color Cards
Search Bar

Use the smart search to search by Color Number or SKU.
Chip Browse

Select from the visual catalog of chips.

6

Select Proceed to Checkout.

7

Select Confirm Info.

8

Select Submit Order.
